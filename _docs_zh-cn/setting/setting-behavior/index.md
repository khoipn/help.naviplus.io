# 设置行为 — 动画与交互

## 动画速度

无 / 非常慢（~500ms）/ 慢（~350ms）/ 中等 ~200ms（默认）/ 快（~120ms）/ 非常快（~60ms）

## 动画样式（仅幻灯片菜单和 Tabbar；Mega Menu 始终使用 Fade）

Slide（默认）= 从边缘滑入。Fade = 淡入。

## 展开效果（仅幻灯片菜单）

Top-down expand = 向下展开。Slide-in from right = 新面板从右侧滑入。

## 触摸反馈（Business / Elite）

Touch Background Color（hex）+ Touch Background Opacity（0–100）。空 = 无效果。

## 箭头样式（Elite）

Color（hex，默认 = Text Color）、Opacity（默认 30）、Size px（默认 20px）、Icon = Remixicon 名称 例：ri-arrow-right-s-line
