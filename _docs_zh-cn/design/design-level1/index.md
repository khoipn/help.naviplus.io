# 设计第 1 级 — 主菜单项

设计 → 第 1 级的设置应用于所有顶级项目。基础层 — 第 2-3 级继承，单项覆盖。

## 文本样式

| 字段 | 描述 | 默认值 |
|---|---|---|
| Text Color | 标签文本颜色 | 继承主题 |
| Font Size (px) | 文本大小 | 继承主题 |
| Font Weight | 100–900 或 Inherit | Inherit |
| Text Transform | None / Capitalize / Uppercase / Lowercase | None |
| Font Family | Google Font 或"Default (Theme)" | Default |
| Description Color | 次要行颜色 | Text Color 的 60% 不透明度 |
| Description Size (px) | 描述字体大小 | Font Size 的 70% |

## 图标 / 图片样式

| 字段 | 描述 | 默认值 |
|---|---|---|
| Icon Color | 矢量图标颜色 | 继承 Text Color |
| Icon Size (px) | 图标尺寸 | 22px |
| Space Text-Icon (px) | 图标与文本之间的间距 | Auto |
| Image Radius (px) | 缩略图圆角 | 0 |

## 间距

| 字段 | 描述 | 格式 |
|---|---|---|
| Item Padding (px) | 内部间距 | 上 右 下 左 或 移动端 \| 桌面端 |
| Item Margin (px) | 外部间距 | 上 右 下 左 或 移动端 \| 桌面端 |

## 背景与边框

| 字段 | 描述 | 默认值 |
|---|---|---|
| Background Color | 项目背景 | 透明 |
| Background Image | URL 或图库 | 无 |
| Opacity | 背景不透明度（0–100） | 100 |
| Hide Background | 隐藏背景 | 关闭 |
| Height (px) | 行高 | 50px |
| Border Radius (px) | 圆角 | 0 |
| Drop Shadow | 阴影 | 关闭 |
| Min Height (px) | 最小高度 | 无限制 |

## 显示

| 字段 | 描述 |
|---|---|
| Display Layout | Auto / Top-Down / Left-Right / Down-Top / Right-Left / Icon Only / Text Only / Empty |
| Align | Auto / Left / Center / Right |
| Hide Text | 隐藏所有文本标签 |
| Show Expand Arrow | 在父项上显示箭头 |

## 徽章

| 字段 | 描述 |
|---|---|
| Badge Color | 徽章/圆点背景颜色 |

## 颜色模板

预设：Light、Dark、Ocean、Sand、Rose、Forest。自动填充颜色值。
