# help.naviplus.io

This is a Jekyll-based static site generated by `create-github-site.sh`.

## Local preview
```bash
bundle exec jekyll serve
```
Then open [http://localhost:4000](http://localhost:4000)

## Cách chạy

Đổi link md trong file getMD.py
```bash
python3 getMD.py
```


Chạy để test
```bash
bundle exec jekyll serve
```
