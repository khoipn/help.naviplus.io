## Cách chạy

Đổi link md trong file getMD.py (hiện tại đang là https://help.shopifas.com/manual/website/help.naviplus.io.md)
```bash
python3 getMD.py

## Thế này mới chạy này
source .venv/bin/activate
python getMD.py


```

## Chạy để test trên local
```bash
bundle exec jekyll serve
```
Then open [http://localhost:4000](http://localhost:4000)
