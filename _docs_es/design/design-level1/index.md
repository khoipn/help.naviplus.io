# Diseño Nivel 1 — Elementos del menú principal

La configuración en **Diseño → Nivel 1** se aplica a todos los elementos del nivel superior. Esta es la capa base — el Nivel 2-3 hereda, por elemento anula.

---

## Estilo de texto

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color de texto** | Color de la etiqueta | Hereda del tema |
| **Tamaño de fuente (px)** | Tamaño del texto | Hereda del tema |
| **Peso de fuente** | 100–900 o Heredar | Heredar |
| **Transformación de texto** | Ninguna / Capitalizar / Mayúsculas / Minúsculas | Ninguna |
| **Familia de fuente** | Google Font o "Predeterminado (Tema)" | Predeterminado |
| **Color de descripción** | Color de la línea de descripción secundaria | 60% de opacidad del Color de texto |
| **Tamaño de descripción (px)** | Tamaño de fuente de la descripción | 70% del Tamaño de fuente |

---

## Estilo de icono / imagen

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color de icono** | Color del icono vectorial | Hereda el Color de texto |
| **Tamaño de icono (px)** | Dimensiones del icono | 22px |
| **Espacio texto-icono (px)** | Separación entre icono y texto | Auto |
| **Radio de imagen (px)** | Redondeo de esquinas para imágenes en miniatura | 0 |

---

## Espaciado

| Campo | Descripción | Formato |
|---|---|---|
| **Relleno del elemento (px)** | Espaciado interior | `arriba derecha abajo izquierda` o `móvil \| escritorio` |
| **Margen del elemento (px)** | Espaciado exterior | `arriba derecha abajo izquierda` o `móvil \| escritorio` |

---

## Fondo y marco

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color de fondo** | Fondo del elemento | Transparente |
| **Imagen de fondo** | URL de imagen de fondo o galería | Ninguna |
| **Opacidad** | Opacidad del fondo (0–100) | 100 |
| **Ocultar fondo** | Ocultar fondo | Desactivado |
| **Altura (px)** | Altura de la fila | 50px |
| **Radio de borde (px)** | Redondeo de esquinas | 0 |
| **Sombra paralela** | Sombra paralela | Desactivada |
| **Altura mínima (px)** | Altura mínima | Sin límite |

---

## Visualización

| Campo | Descripción |
|---|---|
| **Diseño de visualización** | Disposición icono + texto: Auto / Arriba-Abajo / Izquierda-Derecha / Abajo-Arriba / Derecha-Izquierda / Solo icono / Solo texto / Vacío |
| **Alineación** | Alineación del contenido: Auto / Izquierda / Centro / Derecha |
| **Ocultar texto** | Ocultar todas las etiquetas de texto |
| **Mostrar flecha de expansión** | Mostrar flecha de expansión en elementos padre |

---

## Insignia

| Campo | Descripción |
|---|---|
| **Color de insignia** | Color de fondo de la insignia/punto |

---

## Plantillas de color

Paletas preestablecidas: **Claro, Oscuro, Océano, Arena, Rosa, Bosque**. Al seleccionar una se rellenan automáticamente los valores de color.
