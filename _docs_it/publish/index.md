# Pubblica

La sezione **Pubblica** controlla come il tuo menu appare sul tuo sito web live: attivazione/disattivazione del menu, scelta del metodo di incorporamento, configurazione della visibilità per dispositivo e filtraggio per URL. Questo è il passaggio finale dopo la progettazione — senza salvare le impostazioni di Publish, il menu non apparirà.

---

## Argomenti

| # | Argomento | |
|---|---|---|
| 1 | Panoramica dei tipi di menu e dei metodi di distribuzione | [Publish Overview](/docs/publish/publish-overview/) |
| 2 | Sticky / FAB — App Embeds (Shopify) o `<head>` (Global) | [Publish Sticky](/docs/publish/publish-sticky/) |
| 3 | Menu Slide — attivazione basata su trigger | [Publish Slide](/docs/publish/publish-slide/) |
| 4 | Section / Mega / Grid — Inserisci/Sostituisci e App Block | [Publish Section](/docs/publish/publish-section/) |
| 5 | Filtra per dispositivo e parola chiave URL | [Publish Filter](/docs/publish/publish-filter/) |
| 6 | Ottimizza velocità e UX dopo la pubblicazione | [Publish Optimize](/docs/publish/publish-optimize/) |

---

## Riferimento rapido

```
Tipo di menu         Metodo principale
────────────────────────────────────────────────────
Sticky / TABBAR      App Embeds (Shopify) / <head> (Global) + interruttore di attivazione
Sticky / FAB         App Embeds (Shopify) / <head> (Global) + interruttore di attivazione
Slide (CONTEXT)      App Embeds / <head> + interruttore di attivazione + configura trigger
Section (Mega/Grid)  Inserisci/Sostituisci (CSS Selector) o App Block (Shopify)
```

**Regola:** Interruttore attivato = menu live. Interruttore disattivato = completamente nascosto dal sito web, non è necessario rimuovere il codice di incorporamento.
