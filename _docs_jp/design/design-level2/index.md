# デザインレベル 2-3 — サブメニュー項目

デザイン → レベル 2-3 の設定はサブメニューに適用されます。空フィールド = レベル 1 から継承。

ルール：サブメニューをメインメニューと異なる外観にしたい場合のみ入力してください。

## テキストスタイル（空の場合レベル 1 から継承）

Text Color、Font Size、Font Weight、Text Transform、Font Family、Description Color、Description Size

## アイコン / 画像スタイル（空の場合レベル 1 から継承）

Icon Color、Icon Size、Space Text-Icon、Image Radius

## 間隔

| フィールド | 説明 |
|---|---|
| Item Padding (px) | 継承しない — テーマのデフォルトを使用 |
| Item Margin (px) | 外部の間隔 |
| Min Height (px) | 最小高さ |

## 背景とフレーム

Background Color、Border Radius、Hide Background、Drop Shadow

## 表示（空の場合レベル 1 から継承）

Display Layout、Align

## 区切り（空の場合レベル 1 から継承）

Divider Color

## 例

目標：暗いメインメニュー、白いテキスト。白いサブメニュー、暗いテキスト。
レベル 2-3 で 2 つのフィールドのみ入力：Background Color = #FFFFFF、Text Color = #000000
