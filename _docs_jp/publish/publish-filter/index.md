# Publish Filter — デバイスと URL ルール

公開トグルを有効にした後、2 つのフィルターレイヤーを使用してメニューが表示される条件を絞り込むことができます：**デバイス**（モバイル/デスクトップ + スクロール動作）と **URL キーワード**（URL の内容に基づいて表示/非表示）。

---

## デバイスによるフィルタリング

### プラットフォームによる有効/無効

Publish モーダルの上部にある 2 つのトグル：

| 設定 | デフォルト | 注意 |
|---|---|---|
| **モバイルでメニューを表示** | オン | モバイルデバイスでメニューを表示 |
| **デスクトップでメニューを表示** | オフ | デスクトップデバイスでメニューを表示 |

一部のメニュータイプは一方のトグルを無効にします — 例えば、Desktop Mega Menu はモバイルトグルを無効にします。

---

### スクロール動作（Sticky メニューのみ）

モバイルとデスクトップで別々に設定します。

#### モバイルスクロール

| 設定 | 動作 |
|---|---|
| **スクロールダウン時に自動非表示** | ユーザーが**下に**スクロールするとメニューを非表示 |
| **スクロール後に自動表示** | ユーザーが下にスクロールした後のみメニューを表示（ページ読み込み時は非表示） |

#### デスクトップスクロール

| 設定 | 動作 |
|---|---|
| **スクロールダウン時に自動非表示** | デスクトップでスクロールダウン時にメニューを非表示 |
| **スクロール後に自動表示** | デスクトップでスクロールダウン後のみ表示 |

> 自動非表示と自動表示は同時に有効にできます：スクロールダウンでメニューが非表示、スクロールアップで再表示 — 底部タブバーの一般的なパターン。

---

### 画面位置（Sticky / Tabbar のみ）

視覚的なイラスト付きのラジオボタンセレクターから位置を選択します。

**モバイル** — 6 つのプリセット位置（中央下、左下、右下、上部バリアント...）

**デスクトップ** — 12 のプリセット位置（上部バー、下部バー、左/右サイドバー、コーナー位置...）

---

### ページグループ（Shopify のみ、Sticky メニュー）

メニューを表示する Shopify ページタイプを選択します。デフォルト：すべて。

| グループ | URL パターン |
|---|---|
| ホーム | `/` |
| 商品詳細 | `/products/*` |
| 商品カテゴリ | `/collections/*` |
| ページ | `/pages/*` |
| ブログ | `/blogs/*` |
| その他 | 残りすべてのページ |

> グローバル市場（WordPress、Wix など）にはこの機能はありません。代わりに URL キーワードフィルターを使用してください。

---

## URL キーワードによるフィルタリング

**「キーワードでメニューを表示」**カード — Context Slide 以外のすべてのメニュータイプに適用されます。

### 2 種類のフィルター

| 設定 | 動作 |
|---|---|
| **キーワードを含む** | URL が少なくとも 1 つのキーワードを**含む**場合のみメニューを**表示** |
| **キーワードを含まない** | URL がキーワードのいずれかを**含む**場合はメニューを**表示しない** |

複数のキーワードはカンマ `,` で区切ります。

### 例

**スノーボード商品ページのみ表示：**
```
含む: snowboard
```
→ `yourdomain.com/products/the-collection-snowboard-hydrogen` などの URL のみでメニューを表示

**複数ロケーションページで非表示：**
```
含まない: multi-location
```
→ `yourdomain.com/products/the-multi-location-snowboard` ではメニューを**表示しない**

**組み合わせ：**
```
含む: collections
含まない: sale, outlet
```
→ コレクションページのみ表示するが、「sale」と「outlet」コレクションでは非表示。

### 注意事項

- フィルターは正規表現ではなく、**完全な URL 文字列マッチング**を使用します。
- キーワードは**大文字/小文字を区別しません**。
- 空欄のまま = フィルターなし（すべてのページでメニューを表示）。
- 両方のフィルターは AND 条件 — 両方の条件を満たす必要があります。
