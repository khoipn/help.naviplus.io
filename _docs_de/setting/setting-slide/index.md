# Einstellung Slide — Slide-Menü-Optionen

Diese Einstellungen gelten **nur für Slide-Menüs** (Kontextleisten / Hamburger-Menü).

---

## Slide-Richtung

| Option | Beschreibung |
|---|---|
| Von links | Gleitet von links ein (am häufigsten) |
| Von rechts | Gleitet von rechts ein |
| Von oben | Gleitet von oben nach unten |
| Von unten | Gleitet von unten nach oben |
| Fixiert links (Desktop) | Immer sichtbare linke Seitenleiste |
| Fixiert rechts (Desktop) | Immer sichtbare rechte Seitenleiste |

**Auf Mobil fixieren** → Auf Mobilgeräten immer sichtbar ohne Auslöser.
**Auf Desktop fixieren** → Auf Desktop immer sichtbar (dauerhaftes Panel).

---

## Slide-Breite

| Feld | Beschreibung | Standard |
|---|---|---|
| **Mobile Breite** | Breite auf Mobilgeräten (px oder %) | Auto |
| **Desktop Max-Breite (px)** | Maximale Breite auf Desktop | Auto |
| **Desktop Unter-Breite (px)** | Stufe 2 Panel-Breite auf Desktop | Auto |

---

## Schließen-Symbol

| Feld | Beschreibung | Standard |
|---|---|---|
| **Schließen-Symbol ausblenden** | × Schaltfläche ausblenden | Sichtbar |
| **Schließen-Symbol Ausrichtung** | Links / Rechts | Rechts |
| **Schließen-Symbol Oben (px)** | Abstand vom oberen Menürand | 12px |

---

## Desktop Untermenü-Richtung

| Wert | Öffnet in Richtung |
|---|---|
| 1 | Rechts (Standard) |
| 2 | Links |
| 3 | Unten |
| 4 | Oben |
