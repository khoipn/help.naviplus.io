# Setting

Controla como seu menu se comporta e é exibido — animação, comportamento de expansão, posicionamento, regras de visibilidade, opções de slide.

## Tópicos

| # | Tópico | |
|---|---|---|
| 1 | Animação, efeitos de expansão, feedback tátil | [Setting Behavior](/docs/setting/setting-behavior/) |
| 2 | Posição, espaçamento, z-index, largura de submenu | [Setting Layout](/docs/setting/setting-layout/) |
| 3 | Mostrar/ocultar por URL, dispositivo, tipo de página Shopify | [Setting Display Rules](/docs/setting/setting-display-rules/) |
| 4 | Direção de slide, largura, botão de fechar | [Setting Slide](/docs/setting/setting-slide/) |

## Referência Rápida

- Behavior — velocidade de animação, estilo de expansão, feedback tátil
- Layout — padding/margem, z-index, largura de submenu
- Display Rules — palavra-chave URL, dispositivo, tipo de página Shopify
- Slide — direção, largura, botão de fechar (apenas menu slide)
