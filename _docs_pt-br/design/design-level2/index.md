# Design Nível 2-3 — Itens de Submenu

As configurações em Design → Nível 2-3 se aplicam aos submenus. Campo vazio = herda do Nível 1.

Regra: Preencha apenas quando quiser que o submenu tenha aparência diferente do menu principal.

## Estilo de Texto (herda do Nível 1 quando vazio)

Text Color, Font Size, Font Weight, Text Transform, Font Family, Description Color, Description Size

## Estilo de Ícone / Imagem (herda do Nível 1 quando vazio)

Icon Color, Icon Size, Space Text-Icon, Image Radius

## Espaçamento

| Campo | Descrição |
|---|---|
| Item Padding (px) | Não herda — usa padrão do tema |
| Item Margin (px) | Espaçamento externo |
| Min Height (px) | Altura mínima |

## Fundo e Moldura

Background Color, Border Radius, Hide Background, Drop Shadow

## Exibição (herda do Nível 1 quando vazio)

Display Layout, Align

## Divisor (herda do Nível 1 quando vazio)

Divider Color

## Exemplo

Objetivo: Menu principal escuro, texto branco. Submenu branco, texto escuro.
Preencha apenas 2 campos no Nível 2-3: Background Color = #FFFFFF, Text Color = #000000
