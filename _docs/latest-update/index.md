# Changelog / Release Notes

### April 2026

Multi-language menus now work automatically — Navi+ detects the visitor's language and switches menu labels instantly, no extra setup needed. Tab Slide is a new layout option that groups menu items into horizontal tabs instead of a flat list, great for organizing sections like "Men / Women / Sale" inside a slide menu. The color picker was redesigned with preset palettes so you can apply your brand color across the entire menu in seconds. All advanced settings — animations, touch feedback, icons, reusable CSS classes — are now in one unified panel. WordPress support is improved with a faster embed and fewer conflicts with Elementor, Divi, and WPBakery.

---

### March 2026

Section headers can now both link to a page and expand children at the same time — tap to expand, click the arrow to navigate. Grid Menu layout edge cases on small screens have been fixed for consistent rendering across all phone sizes. Separator lines and touch-highlight colors inside menus can now be set to any custom color.

---

### February 2026

Z-index conflicts are now resolved automatically — Navi+ detects when your store's native panels (cart drawer, search overlay, cookie banner) appear and steps behind them, so nothing gets covered. A new Slide Horizontal direction makes level-2 panels slide in from the right like a native iOS app, which feels more natural for deep product category trees. Menu configurations can now be exported and imported as JSON via a popup, making it easy to back up or move a menu across stores. Images in menus now lazy-load for faster initial page load. Emoji can be added directly to any menu item label.

---

### January 2026

The new Slide Horizontal menu type is built for deep navigation trees on mobile — each level slides in from the right, with the back button and header updating automatically. Item templates let you build a menu structure once, save it, and reuse it across any menu with one click. Import/Export now supports AI: paste a link list or sitemap and Navi+ generates the full menu tree, or export as JSON, edit in an AI tool, and import back. A new Debug Mode overlays z-index values, menu boundaries, and selector highlights on your live site directly from the dashboard. CDN infrastructure was upgraded to Cloudflare R2 edge with a second failover CDN added.

---

### December 2025

Navi+ now runs on every platform — WordPress, WooCommerce, Wix, Webflow, Squarespace, and any site that accepts a `<script>` tag. Same dashboard, same menu types, same one-line embed as on Shopify. The dashboard UI is now available in multiple languages.

---

### November 2025

Fixed a bug where the auto-hide/auto-show behavior would flicker at the very top and bottom of the page on some devices. A safe-edge threshold now prevents this, so the menu hides and shows smoothly.

---

### October 2025

All Navi+ assets migrated to Cloudflare's global edge network — average menu load time dropped to ~52ms. Elite plan users also get a secondary BunnyCDN failover, keeping menus online if one CDN goes down. SLA upgraded to 99.99%.
