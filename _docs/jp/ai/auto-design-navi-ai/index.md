# Navi+ AIによる自動デザイン

AI の作業はすべて Navi+ に任せましょう — エディタ内で欲しいメニューを説明するだけです。外部ツールも、JSON のコピー＆ペーストも不要です。*(Business / Elite)*

開き方: **AI** ボタン → **Navi+ AI による自動デザイン** カード。

---

## 「あなたの AI アプリで自動デザイン」との違いは？

| | あなたの AI アプリ | Navi+ AI |
|---|---|---|
| AI ツール | ChatGPT / Claude / Gemini を自分で開く | Navi+ が代わりに AI を呼び出す |
| JSON のコピー＆ペースト | はい、手動で | 不要 |
| ステップ | 4 ステップ | 1 ステップ（チャット） |
| プラン | 無料 | Business / Elite |

---

## 始める前に — メニューをバックアップ

> **⋮ もっと見る → バックアップ / 復元** — AI の結果を適用する前に、現在のメニューをバックアップします。

---

## 使い方

1. **Navi+ AI による自動デザイン** カードを開きます。
2. チャットボックスに説明を入力します。例:「ファッションストア向けの 5 項目のタブバーを、白黒で作成して。」
3. （任意）**アップロード** ボタンで参照画像を添付すると、AI がレイアウト／カラーに従います。
4. **送信** をクリックします。AI はデザインを返し、プレビュー用にエディタへ適用します。
5. チャットを続けて調整したり（「Contact のアイコンを変えて」「最後の項目を削除して」）、**新しいチャット** をクリックして最初からやり直すこともできます。
6. 満足ですか？ **保存** をクリックします。

> チャット履歴は保存されます — **履歴** をクリックすると、以前のデザインセッションを再び開けます。

---

## プラン要件

この機能は **Business / Elite** プランで提供されます。下位プランでは、このカードを開いたときにアップグレードの通知が表示されます。

---

## プロンプトの例

- 化粧品ストア向けの 5 項目のタブバーを、パステルピンクのトーンで作成。
- 各項目にアイコンを追加し、最後の項目を目立つ Contact ボタンに変える。
- 今送った画像のようなメガメニューをデザインして。*(アップロードボタンで画像を添付)*
- Blog 項目を削除し、Promotions と Sale を 1 つにまとめる。

---

## ケーススタディ — 3 回のチャットで化粧品タブバーを作る

**背景:** まだメニューのない新しい化粧品ストアが、パステルピンクのブランドに合うタブバーを欲しがっています。

1. 念のためまず**バックアップ**します（⋮ もっと見る → バックアップ / 復元）。
2. **Navi+ AI による自動デザイン** カードを開きます。1 回目:

   ```
   化粧品ストア向けの 5 項目のタブバーを作成して: Home, Products,
   Offers, Beauty Guide, Account。パステルピンクのトーンで。
   ```

   AI はタブバーを構築し、すぐにプレビューに表示します。
3. 2 回目 — **アップロード** ボタンでサンプル UI 画像を添付し、こう入力します:

   ```
   この画像のスタイルに合わせて、角の丸みとアイコンを調整して。
   ```

4. 3 回目 — 小さな調整:

   ```
   Account 項目を、電話アイコン付きの目立つ Contact ボタンに変えて。
   ```

5. 満足したら **保存** をクリックします。次回変更が必要なときは、**履歴** をクリックしてこのセッションを再び開きます。

**結果:** ブランドに合ったタブバーが、Navi+ から離れることなく 3 回のチャットで完成しました。
