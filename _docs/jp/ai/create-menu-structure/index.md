# メニュー構造を作成

必要なメニューを説明して、AI に出来上がった骨組みを構築させましょう — 各項目を手作業で追加する代わりに、素早く出発点を手に入れられます。この機能は**無料**です。

開き方: **AI** ボタン → **メニュー構造を作成** カード。

---

## 始める前に — メニューをバックアップ

> **⋮ もっと見る → バックアップ / 復元** — 結果は既存のメニューを上書きするため、新しい構造を作成する前に現在のメニューをバックアップします。

---

## 使い方

1. **メニュー構造を作成** カードを開きます。
2. （任意）AI が**アイコンを自動追加**するレベルを選びます:

   | オプション | 適用対象 |
   |---|---|
   | レベル 1 | メイン項目（レベル 1） |
   | レベル 2 | レベル 2 のサブ項目 |
   | レベル 3 | レベル 3 のサブ項目 |

3. **リクエストを説明** ボックスに説明を入力します。例:「化粧品ストア向けのメニュー: Home, Products（Lipstick, Moisturizer, Mask）, Promotions, Contact。」
4. （任意）**テンプレートを使う** をクリックすると、メニュータイプに合ったサンプルプロンプトが挿入されるので、それを編集します。
5. **メニューを作成** をクリックします。進行状況は右側の **アクティビティログ** に表示されます。
6. AI が構造を構築し、エディタに適用します。確認・調整して、**保存** をクリックします。

> **保存**（ディスクアイコン）をクリックすると、次回再利用するためにプロンプトを保存できます。

---

## 作成後

AI は**構造の骨組み**（項目、階層、アイコン）のみを構築します。次の項目は自分で確認・追加してください:

- 各項目のリンク（URL）
- 必要に応じて画像とバッジ
- **デザイン** と **設定** パネルによるカラーとレイアウト

---

## 説明の例

- メンズファッションストア: Home, Tops（T-shirts, Shirts, Jackets）, Pants, Accessories, Sale, Contact。
- フードデリバリーアプリ向けの 5 項目のタブバー: Home, Search, Cart, Orders, Account。
- ブログ向けのスライドメニュー: Latest posts, Categories（Tech, Travel, Food）, About, Contact。
- 電化製品ストア向けの 4 グループのメガメニュー: Phones, Laptops, Accessories, Promotions — 各グループにサブ項目 4〜5 個。

---

## ケーススタディ — 電化製品ストア向けのメガメニューを作る

**背景:** ある電化製品ストアは、各グループに複数のサブ項目があり、ざっと見渡しやすいようアイコンが付いた、マルチグループのデスクトップ用メガメニューを必要としています。

1. **バックアップ:** ⋮ もっと見る → バックアップ / 復元。
2. **メニュー構造を作成** カードを開きます。**レベル 1** と **レベル 2** にチェックを入れ、AI がグループとサブ項目の両方にアイコンを追加するようにします。
3. 説明を入力します:

   ```
   電化製品ストア向けのメガメニュー、4 グループ:
   - Phones（iPhone, Samsung, Xiaomi, スマホアクセサリー）
   - Laptops（オフィス, ゲーミング, MacBook, ノートPCアクセサリー）
   - 家電（浄水器, ロボット掃除機, ノンフライヤー）
   - Promotions
   ```

4. **メニューを作成** をクリックします。右側の **アクティビティログ** を見守ります。AI はサブ項目とアイコンを備えた 4 グループのメガメニューを構築します。
5. 作成後に追加: 各項目に **URL** を割り当て、Promotions グループにバナー画像を追加してから、**保存** します。

**結果:** アイコン付きのマルチレベルなメガメニューの骨組みがすぐに完成 — 何十もの項目を手作業で作る代わりに、リンクを付けて微調整するだけで済みます。
