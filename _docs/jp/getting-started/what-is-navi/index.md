# Navi+とは？（仕組みの説明）

Navi+は**ノーコードのナビゲーションビルダー**です — テーマに触れることなく、コードを1行も書かずに、あらゆるウェブサイト向けに美しく高コンバージョンなメニューを作成できます。

サイトの訪問者が探しているものを見つけにくい場合、メニューが足を引っ張っているかもしれません。Navi+はそれを数分で解決します。

## 5つのメニュータイプ

ユースケースに合ったメニューを選択してください：

| メニュータイプ | 最適な用途 |
| --- | --- |
| **Tab Bar** | モバイルファーストのナビゲーション、ネイティブアプリのような感触 |
| **Mega Menu** | 画像・列・リンク付きのリッチなドロップダウン — デスクトップとモバイルで別バージョン |
| **Slide Menu** | マルチレベル対応のハンバーガーメニュー |
| **Grid Menu** | ビジュアルカテゴリナビゲーション — ECサイトに最適 |
| **FAB** | いつでも手の届くクイックアクセスフローティングボタン |

同じサイトで複数のメニュータイプを組み合わせて使用できます。

## 仕組み

Navi+は**ウィジェット**です — 既存のテーマを置き換えたり変更したりしません。代わりに、軽量スクリプトをページに注入し、既存のレイアウトの上にメニューをレンダリングします。

すべてを**Navi+ダッシュボード**から構築・管理します。変更はすぐに反映されます — コードを編集したり、テンプレートを変更したり、テーマを再公開する必要はありません。

フロー：

1. ダッシュボードでメニューを作成
2. Navi+が小さなスクリプトを生成
3. そのスクリプトをサイトに埋め込む（ShopifyのApp Block、または他の場所でコードスニペット）
4. すべてのページ読み込み時に、既存レイアウトの上にメニューがレンダリングされる

## 対応プラットフォーム

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · `<script>` タグを受け入れるあらゆるウェブサイト。

## Navi+ではないもの

- **テーマではありません** — 既存のテーマの代わりではなく、並行して動作します
- **手動インストールが必要なコードではありません** — App Block（Shopify）または埋め込みスニペット1つで十分です
- **サイトを壊す可能性のあるプラグインではありません** — Navi+が無効になると、元のテーマメニューが自動的に戻ります

## スピードのために設計

Navi+のアセットはCloudflare CDN経由で~52ms平均読み込み時間で配信されます — 通常、プラットフォーム独自のスクリプトより高速です。
