# Navi+とは？

Navi+は**ノーコードナビゲーションビルダー**です — テーマに手を加えたり、1行のコードも書かずに、どんなウェブサイトでも美しく、高いコンバージョン率のメニューを作成できます。

サイトの訪問者が探しているものを見つけるのに苦労している場合、メニューが彼らを失望させているかもしれません。Navi+はそれを数分で修正する手助けをします。

## 5つのメニュータイプ

使用ケースに最適なメニューを選択してください：

| メニュータイプ | 最適な用途 |
| --- | --- |
| **Tab Bar** | モバイルファーストのナビゲーション、ネイティブアプリのように感じる |
| **Mega Menu** | 画像、カラム、リンクを含むリッチなドロップダウン — デスクトップとモバイルのバージョンがあります |
| **Slide Menu** | マルチレベルサポートのハンバーガーメニュー |
| **Grid Menu** | ビジュアルカテゴリブラウジング — eコマースに最適 |
| **FAB** | いつでも手の届くところにあるクイックアクセスの浮遊ボタン |

同じサイトで複数のメニュータイプを一緒に使用できます。

## 使い方

Navi+は**ウィジェット**です — 既存のテーマを置き換えたり、変更したりすることはありません。代わりに、ページに軽量スクリプトを注入し、既存のレイアウトの上にメニューをレンダリングします。

すべては**Navi+ダッシュボード**から構築および管理します。変更は即座に反映されます — コードを編集したり、テンプレートを変更したり、テーマを再公開する必要はありません。

フローは次のとおりです：

1. ダッシュボードでメニューを作成します
2. Navi+が小さなスクリプトを生成します
3. そのスクリプトがサイトに埋め込まれます（ShopifyのApp Block経由、または他の場所でのコードスニペット）
4. メニューは、ページが読み込まれるたびに既存のレイアウトの上にレンダリングされます

## あなたのプラットフォームで動作

Navi+は、すでに使用しているツールにフィットします：

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · そして`<script>`タグを受け入れる任意のウェブサイト。

## Navi+が何でないか

- **テーマではない** — 既存のテーマと並行して機能し、代わりにはなりません
- **手動でインストールするコードではない** — 1つのApp Block（Shopify）または1つの埋め込みスニペットがあれば十分です
- **サイトを壊す可能性のあるプラグインではない** — Navi+が無効にされると、元のテーマメニューが自動的に戻ります

## スピードのために構築

Navi+のアセットはCloudflare CDNを介して配信され、平均ロード時間は約52msです — 通常、プラットフォームの独自のスクリプトよりも速いです。最初のロード後、メニューはキャッシュされ、瞬時にロードされます。
