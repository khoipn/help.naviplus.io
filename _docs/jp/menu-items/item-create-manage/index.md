# アイテムの作成と管理

## 新しいアイテムを追加

1. メニューツリーで、**+ アイテムを追加**（またはリストの下部にあるプラスボタン）をクリックします。
2. 希望するアイテムの種類を選択します（アイコン/テキスト、大きな画像、グループタイトル、空白スペース、タブマーカー…）。
3. 新しいアイテムはデフォルトの内容でリストの最後に表示されます。
4. アイテムをクリックしてアイテム編集パネルを開き、詳細を入力します。
5. **更新** → **保存**を押します。

> 新しいアイテムは常に下部に追加されます。追加後に希望の位置までドラッグしてください。

---

## アイテムの順序変更

### ドラッグアンドドロップによる

- アイテムの左側にあるドラッグハンドル（≡）にカーソルを合わせます。
- 同じレベル内で順序を変更するために**上/下**にドラッグします。
- **右**にドラッグするとインデントされ、アイテムは上のアイテムの子になります。
- **左**にドラッグするとアウトデントされ、アイテムは1つ上のレベルに戻ります。

### キーボードショートカットによる（アイテムが選択されている間）

| キー | アクション |
|---|---|
| `↑` | 上に移動（Mac: `Cmd+↑` / Win: `Alt+↑`） |
| `↓` | 下に移動（Mac: `Cmd+↓` / Win: `Alt+↓`） |
| `→` | インデント — アイテムは上のアイテムの子になります |
| `←` | アウトデント — アイテムは親レベルに戻ります |

> ショートカットは、カーソルがメニューツリーの上にあり、アイテムが選択されている（青いアウトライン）ときのみ機能します。

---

## アイテムを編集

1. アイテムをクリック → アイテム編集パネルが開きます。
2. 必要なフィールドを変更します。
3. **更新**を押して、そのアイテムへの変更を保存します。

> **キャンセル**を押すと、保存せずにパネルを閉じます。

---

## アイテムを複製

1. アイテムにカーソルを合わせる → クイックアクションメニューが表示されます。
2. **コピー**をクリック（または`C`を押します）。
3. 元のアイテムのすぐ下にコピーが表示され、すべての内容と設定が保持されます。
4. 必要に応じてコピーを編集します。

---

## アイテムを削除

1. アイテムにカーソルを合わせる → クイックアクションメニューが表示されます。
2. **削除**をクリック（またはMacで`Del` / `Backspace`を押します）。
3. 確認ダイアログが表示されます — **確認**をクリックして永久に削除します。

> **注意:** 親アイテムを削除すると、そのすべての子アイテムも削除されます。確認後は元に戻すことはできません。

---

## 元に戻す / やり直す

| ショートカット | アクション |
|---|---|
| `Ctrl+Z` / `Cmd+Z` | 最後のアクションを元に戻す |
| `Ctrl+Y` / `Cmd+Shift+Z` | やり直す |

元に戻す履歴は現在のセッションにのみ存在します — ページをリロードするとクリアされます。

---

## メニューを保存

アイテムを追加、順序変更、または削除した後:

- ツールバーの**保存**をクリックするか、`Ctrl+S` / `Cmd+S`を押します。
- 保存を押す際に`Shift`を押し続けると、保存してすぐにシミュレーターをリロードします。

変更は保存するまで顧客には表示されません。
