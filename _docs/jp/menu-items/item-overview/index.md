# アイテムの概要

## メニューアイテムとは？

メニューリストの各行は**アイテム**です。アイテムはNavi+における最小単位であり、ナビゲーションリンク、グループヘッダー、空白のスペーサー、またはタブ区切りである可能性があります。

複数のアイテムが重なり合ってメニューを構成します。メニューはその後、あなたのウェブサイトの訪問者に表示されます。

---

## アイテムの種類

新しいアイテムを追加する際には、その**種類**を選択します。種類はアイテムの見た目と機能を決定します。

| 種類 | 説明 | 使用するタイミング |
|---|---|---|
| **アイコン / テキスト** | アイコンとラベルを持つ標準アイテム | 通常のナビゲーションリンク |
| **大きな画像 / テキスト** | フルブリード画像とオーバーレイテキストを持つ大きなタイル | カテゴリバナー、特集商品画像 |
| **グループタイトル** | セクション見出し — リンクではない | その下のアイテムを視覚的にグループ化 |
| **空白スペース** | 透明な隙間 | スペーシングとレイアウトの整列 |
| **タブマーカー** | 新しいタブの開始を示す | メニューを水平タブに分割 |
| **カスタムHTML** | 任意のHTMLを埋め込む | カスタムウィジェット、特別なバナー |

## フィールド構造

任意のアイテムをクリックすると、アイテム編集パネルが開きます。これは以下のセクションに整理されています：

```
┌─────────────────────────────────────────┐
│  メディア                               │
│    アイコン  |  画像                    │
├─────────────────────────────────────────┤
│  基本情報                               │
│    テキスト · リンク先 · HTML          │
│    説明 · 区切り · バッジ              │
├─────────────────────────────────────────┤
│  レイアウト                               │
│    表示レイアウト · 幅 · 位置          │
│    高さ固定                             │
├─────────────────────────────────────────┤
│  インターフェース (UI)                 │
│    クラス名 · テキストスタイル         │
│    アイコン/画像スタイル · 背景      │
│    スペーシング · 整列                 │
├─────────────────────────────────────────┤
│  高度な設定                             │
│    アニメーション · CSS · 属性         │
├─────────────────────────────────────────┤
│  公開                                   │
│    公開されているか · ログイン条件    │
└─────────────────────────────────────────┘
```

## 編集ワークフロー

1. メニューツリー内のアイテムをクリック → アイテム編集パネルが開きます。
2. 必要なフィールドを変更します。
3. **更新**を押して、そのアイテムに変更を適用します。
4. **保存**（`Ctrl+S` / `Cmd+S`）を押して、全体のメニューを保存し、シミュレーターを更新します。

> 更新を押したがまだ保存していない場合、変更はライブメニューに書き込まれません。保存を押す際にShiftを押し続けると、保存と同時にシミュレーターを即座に再読み込みします。
