# アイテムの概要

## メニューアイテムとは？
メニューリスト内のすべての行がアイテムです。アイテムはNavi+の最小単位であり、ナビゲーションリンク、グループヘッダー、空白スペーサー、またはタブ区切りとして使用できます。

## アイテムの種類
| 種類 | 説明 | 使用場面 |
|---|---|---|
| **アイコン / テキスト** | アイコンとラベルを持つ標準アイテム | 通常のナビゲーションリンク |
| **大画像 / テキスト** | フルブリード画像を持つ大タイル | カテゴリバナー、注目商品 |
| **グループタイトル** | セクション見出し — リンクなし | 下のアイテムを視覚的にグループ化 |
| **空白スペース** | 透明な隙間 | スペーシングとレイアウト調整 |
| **タブマーカー** | 新しいタブの開始を示す | メニューを横並びタブに分割 |
| **カスタムHTML** | 任意のHTMLを埋め込む | カスタムウィジェット、特別なバナー |

## フィールド構造
任意のアイテムをクリックしてアイテム編集パネルを開きます：

```
メディア: アイコン | 画像
基本情報: テキスト・リンク先・HTML・説明・区切り線・バッジ
レイアウト: 表示レイアウト・幅・位置・高さ固定
インターフェース: クラス名・テキストスタイル・アイコン/画像スタイル・背景・スペーシング・整列
アドバンス: アニメーション・CSS・属性
公開: 公開状態・ログイン条件
```

## 編集ワークフロー
1. アイテムをクリック → アイテム編集パネルが開く。
2. 必要なフィールドを変更する。
3. **更新**を押して変更を適用する。
4. **保存**（`Ctrl+S` / `Cmd+S`）を押して公開する。
