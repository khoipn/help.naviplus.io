# Navi+のインストール方法

Navi+を使用するために開発やデザインについて知っている必要はありません。これは、あなたのウェブサイトに簡単に埋め込むことができる事前構築されたソフトウェアです。すべての設定 — 機能やビジュアルデザインを含む — は、既存のシステムと衝突することなく、Navi+上で直接管理されます。

***

### ステップ1: アプリケーションのインストール

**Shopifyを使用している場合:**\
以下のリンクから公式アプリをインストールしてください:\
<https://apps.shopify.com/pronavi-navigation-design>

**WordPress、WooCommerce、Wix、Webflow、Squarespace、または自己ホスト型ウェブサイトなどの他のプラットフォームを使用している場合:**

1. <https://naviplus.io>にアクセス
2. 無料アカウントを作成
3. 指示に従ってログインし、ウェブサイトを接続します。

***

### ステップ2: メニューを作成し、ウェブサイトに埋め込む

ログインしたら、ウェブサイト用のメニューを作成し始めることができます。\
**Navi+**は、さまざまなニーズに合わせた複数のタイプのメニューを提供します:

* **Tabbar**\
  画面の下部に固定されたナビゲーションバーで、ユーザーが主要なセクションに迅速にアクセスできるようにします。モバイルレイアウトに最適です。
* **Mega Menu**\
  複数の列を持つ大きなドロップダウンメニューで、製品カテゴリや詳細なコンテンツを明確な構造で表示するのに最適です。
* **Slide Menu (ハンバーガーメニュー)**\
  画面の端からスライドインする隠れたサイドメニューです。スペースを節約し、クリーンでモダンなデザインを維持するのに最適です。
* **Floating Button Menu**\
  クイックリンクのリストを開いたり、特別なキャンペーンやプロモーションを強調表示したりできる浮遊アクションボタンです。

各タイプのメニューには独自の設定プロセスがあり、Navi+ダッシュボード内で完全にステップバイステップでガイドされています。

異なるページで同時に複数のタイプのNavi+メニューを使用することもできます — 例えば、モバイル用のTabbar、デスクトップ用のMega Menu、アクションやプロモーション用のFloating Buttonなど。
