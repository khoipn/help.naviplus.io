# 公開

**公開**セクションは、あなたのメニューがライブウェブサイトにどのように表示されるかを制御します：メニューのオン/オフの切り替え、埋め込み方法の選択、デバイスの可視性の設定、URLによるフィルタリング。これはデザインの後の最終ステップです — 公開設定を保存しない限り、メニューは表示されません。

## クイックリファレンス

```
メニュータイプ           プライマリメソッド
────────────────────────────────────────────────────
スティッキー / TABBAR     アプリ埋め込み (Shopify) / <head> (グローバル) + 有効トグル
スティッキー / FAB        アプリ埋め込み (Shopify) / <head> (グローバル) + 有効トグル
スライド (コンテキスト)     アプリ埋め込み / <head> + 有効トグル + トリガーの設定
セクション (メガ/グリッド) 挿入/置換 (CSSセレクタ) またはアプリブロック (Shopify)
```

**ルール：** 有効トグル = メニューがライブ。無効トグル = ウェブサイトから完全に隠され、埋め込みコードを削除する必要はありません。
