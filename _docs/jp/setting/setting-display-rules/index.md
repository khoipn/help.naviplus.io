# 表示ルールの設定 — 条件付き表示

デバイスタイプ、URL、または Shopify ページタイプに基づいて、メニューが表示される場所とタイミングを制御します。

---

## デバイスによる表示

メニューの **Publish** タブにあります（アイテムごとではなく）：

| トグル | 効果 |
|---|---|
| **Mobile** | モバイル画面でメニューを表示/非表示 |
| **Desktop** | デスクトップ画面でメニューを表示/非表示 |

デフォルト：Mobile のみが有効です。大きな画面にメニューを表示したい場合は、手動で Desktop を有効にする必要があります。

> デバイスごとに個別のアイテムを表示/非表示にするには → [アイテムの可視性](/docs/menu-items/item-visibility/)を参照してください。

---

## URL キーワードによる表示

**Setting → Display by Keywords** の 2 つのフィールドで、メニューが表示される URL を制御できます。

### URL にキーワードが含まれている場合のみ表示

**フィールド:** `Publish — Contain Keyword`

```
collections, products/sale
```

メニューは、URL に `collections` **または** `products/sale` が含まれているページにのみ表示されます。

**例:** 商品ページ用の "Shop Navigation" メニュー → `collections/` と `products/` のみで表示。

---

### URL にキーワードが含まれている場合は非表示

**フィールド:** `Publish — Don't Contain Keyword`

```
checkout, account/login
```

メニューはチェックアウトおよびログインページでは非表示になり、他のすべての場所で表示されます。

**例:** チェックアウト中に Tabbar を非表示にして、顧客の気を散らさないようにします。

---

## Shopify ページタイプによる表示

**Publish** タブには、6 つの Shopify ページグループも含まれています（Shopify ストアのみ）：

| グループ | ページ |
|---|---|
| **Home** | ホームページ |
| **Product Detail** | 商品詳細ページ |
| **Product Categories** | コレクション / カテゴリページ |
| **Pages** | 静的ページ（About、Contact など） |
| **Blogs** | ブログリストおよび記事ページ |
| **Others** | 上記 5 グループに含まれないページ |

グループのチェックを外す → メニューはそのページタイプに表示されません。

---

## ルールのスタック方法

すべての条件は、この順序で一緒に評価されます：

```
1. デバイスは有効ですか？              →  いいえの場合：非表示
2. ページタイプはチェックされていますか？           →  いいえの場合：非表示
3. URL は「表示しない」と一致しますか？    →  はいの場合：非表示
4. URL は「のみ表示」と一致しますか？     →  いいえの場合：非表示
5. メニューを表示
```
