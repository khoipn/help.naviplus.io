# 設定動作 — アニメーションとインタラクション

## アニメーション速度

なし / 非常に遅い（〜500ms）/ 遅い（〜350ms）/ 中くらい 〜200ms（デフォルト）/ 速い（〜120ms）/ 非常に速い（〜60ms）

## アニメーションスタイル（スライドメニューとタブバーのみ、メガメニューは常に Fade）

Slide（デフォルト）= エッジからスライド。Fade = フェードイン。

## 展開エフェクト（スライドメニューのみ）

Top-down expand = 下方向に展開。Slide-in from right = 新しいパネルが右からスライドイン。

## タッチフィードバック（Business / Elite）

Touch Background Color（hex）+ Touch Background Opacity（0–100）。空 = エフェクトなし。

## 矢印スタイル（Elite）

Color（hex、デフォルト = Text Color）、Opacity（デフォルト 30）、Size px（デフォルト 20px）、Icon = Remixicon 名 例：ri-arrow-right-s-line
