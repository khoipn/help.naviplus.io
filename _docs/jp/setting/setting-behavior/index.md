# 設定の動作 — アニメーションとインタラクション

**設定 → アニメーション** と **設定 → 展開効果** で、メニューの速度、動きのスタイル、インタラクションフィードバックを制御します。

---

## アニメーション速度

メニュー内のすべての動きに適用されます（サブメニューのオープン、アイテムの遷移など）。

| オプション | 期間 |
|---|---|
| なし | アニメーションなし |
| 非常に遅い | ~500ms |
| 遅い | ~350ms |
| **中** *(デフォルト)* | ~200ms |
| 速い | ~120ms |
| 非常に速い | ~60ms |

---

## アニメーションスタイル

メニューの表示方法を制御します。**Slide menu** と **Tabbar** のみ適用されます — Mega Menu は常にフェードを使用します。

| オプション | 効果 |
|---|---|
| **スライド** *(デフォルト)* | 画面の端からスライドイン |
| **フェード** | 徐々にフェードイン |

---

## 展開効果

**Slide menu** のみ適用されます — サブメニューの開き方を制御します。

| オプション | 効果 |
|---|---|
| **上から下に展開** | サブメニューがその場で下に展開 |
| **右からスライドイン** | サブメニューが新しいパネルとして右からスライドイン |

---

## タッチフィードバック *(Business / Elite)*

モバイルでユーザーがアイテムをタップしたときの背景フラッシュカラー。

| フィールド | 説明 |
|---|---|
| **タッチ背景色** | フラッシュカラー（16進数） |
| **タッチ背景不透明度** | フラッシュ不透明度（0–100） |

空白のまま = タッチ効果なし。

---

## 矢印スタイル *(Elite)*

子を持つ親アイテムに表示される展開矢印をカスタマイズします。

| フィールド | 説明 | デフォルト |
|---|---|---|
| **色** | 矢印の色（16進数） | テキストカラーを継承 |
| **不透明度** | 矢印の不透明度（0–100） | 30 |
| **サイズ (px)** | 矢印のサイズ | 20px |
| **アイコン** | Remixicon名（例: `ri-arrow-right-s-line`） | システムデフォルト |
