# スライドの設定 — スライドメニューオプション

これらの設定は**スライドメニュー**（コンテキストスライド / ハンバーガーメニュー）にのみ適用されます。

---

## スライド方向

**設定 → スライド方向** — メニューがどこからスライドインするか、そしてどのように固定されるか。

| オプション | 説明 |
|---|---|
| 左から | 左端からスライドイン（最も一般的） |
| 右から | 右端からスライドイン |
| 上から | 上からスライドダウン |
| 下から | 下からスライドアップ |
| 左固定（デスクトップ） | 常に表示される左側のサイドバー — トリガーは不要 |
| 右固定（デスクトップ） | 常に表示される右側のサイドバー |

**モバイルで固定 / デスクトップで固定:**
- **モバイルで固定** → メニューはモバイルで常に表示され、開く必要はありません。
- **デスクトップで固定** → メニューはデスクトップで常に表示されます（持続的なサイドバー）。

---

## スライド幅

| フィールド | 説明 | デフォルト |
|---|---|---|
| **モバイル幅** | モバイルでの幅（pxまたは%） | 自動 |
| **デスクトップ最大幅（px）** | デスクトップでの最大幅 | 自動 |
| **デスクトップサブ幅（px）** | デスクトップでのレベル2サブメニューパネルの幅 | 自動 |

**例:** 左からスライド、モバイル画面の80%、デスクトップで最大400px:
- モバイル幅 = `80%`
- デスクトップ最大幅 = `400`

---

## 閉じるアイコン

| フィールド | 説明 | デフォルト |
|---|---|---|
| **閉じるアイコンを隠す** | ×閉じるボタンを隠す | 表示 |
| **閉じるアイコンの位置** | 閉じるボタンの位置: 左 / 右 | 右 |
| **閉じるアイコンの上部（px）** | メニューの上部から閉じるボタンまでの距離 | 12px |

> 閉じるボタンを隠すと、ユーザーはメニューの外側のオーバーレイエリアをタップして閉じる必要があります。メニューが画面の大部分を覆う場合は、閉じるボタンを表示しておいてください。

---

## デスクトップサブメニューの方向

**設定 → デスクトップ → サブメニューの方向**

スライドメニューがデスクトップで固定サイドバーとして表示されるとき、サブメニューは4つの方向に開くことができます:

| 値 | サブメニューが開く方向 |
|---|---|
| 1 | 右（デフォルト） |
| 2 | 左 |
| 3 | 下 |
| 4 | 上 |
