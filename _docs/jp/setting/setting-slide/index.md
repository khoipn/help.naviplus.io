# 設定スライド — スライドメニューオプション

スライドメニュー（Context Slide / ハンバーガーメニュー）のみに適用されます。

## スライド方向

From Left（最も一般的）/ From Right / From Top / From Bottom / Fixed Left desktop / Fixed Right desktop。
Fix on Mobile = モバイルで常に表示。Fix on Desktop = 固定サイドバー。

## スライド幅

Mobile Width（px または %）、Desktop Max Width（px）、Desktop Sub Width（px）。すべてデフォルト Auto。
例：Mobile Width = 80%、Desktop Max Width = 400。

## 閉じるアイコン

Hide Close Icon（デフォルト表示）、Close Icon Align Left／Right（デフォルト Right）、Close Icon Top px（デフォルト 12px）。
非表示の場合：ユーザーはオーバーレイをタップして閉じる必要があります。

## デスクトップサブメニューの方向

1=Right（デフォルト）、2=Left、3=Down、4=Up
