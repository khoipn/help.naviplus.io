# マルチ言語 (Shopify Markets)

Navi+ は、**インライン翻訳構文** と **言語スイッチャー** の2つの方法でマルチ言語をサポートしています。

## 1. インライン翻訳構文

この構文を使用して、メニュー項目のラベルに直接翻訳を追加します：

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

**例** — 英語、フランス語、ベトナム語で表示されるタブバー：

```
Home <fr: Maison><vi: Trang chủ>
```

メニューがレンダリングされると、Navi+ は現在のページの言語をチェックし、一致する翻訳を表示します。一致するものが見つからない場合は、デフォルトにフォールバックします。

この構文は、**ラベル**、**説明**、および **リンク先** フィールドで機能します。

## 2. 言語スイッチャー

任意のNavi+ メニュー内に言語スイッチャーを構築できます。メニュー項目に `open:ChangeLanguage()` アクションを使用します：

```
open:ChangeLanguage(fr)
```

訪問者がその項目をタップすると、サイトはフランス語に切り替わります。スイッチャーをより視覚的にするために、[Flaticon](https://www.flaticon.com) のようなサイトからフラグアイコンを追加できます。
