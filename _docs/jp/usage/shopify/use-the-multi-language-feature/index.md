# 多言語機能を使用する。

構文：

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

例：

```
Home <fr: Maison><zh: 家>
```

ここにNavi+がサポートするすべての言語コードのデフォルトの完全リストがあります。

時々、SEOシステムを使用して、これらの2文字の言語コードを異なる形式に変換することができます。たとえば、ENをen-ENまたはenglishに変更することです。その場合でも、言語コードを通常通り簡単に使用できます。
