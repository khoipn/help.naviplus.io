# 公開例

以下は、ウェブサイトでスライドメニューを公開する方法のいくつかの例です。

#### 方法1: このメニューを別のNavi+メニュー項目からopen:NaviMenu(SF-123456789)という構文を使用して開く

これは、複数のNavi+メニューを使用する際にスライドメニューを公開する簡単な方法です。たとえば、タブバー項目を使用してスライドメニューを開き、メニューのスペースを拡張できます。
