# Navi+ メニューの公開に関するいくつかの FAQ

**なぜ私のサイトに Navi+ が表示されないのですか？**

すべてのステップを完了したことを確認してください。Tabbar/FAB の場合、次のことが必要です: 1) テーマに Navi+ アプリを埋め込む, 2) 公開メニューのトグルを有効にする。
