# FAQ

#### Question 1: Why doesn’t Navi+ automatically display the desktop Mega Menu on mobile devices?

デスクトップとモバイルでのブラウジング体験は根本的に異なります — 特にユーザーがウェブサイトをナビゲートし、インタラクトする方法において。

デスクトップでは、ユーザーはより多くの画面スペースを持ち、ヘッダーの下に表示される大きく複雑なメガメニューを簡単に見ることができます。

しかし、モバイルでは、画面が小さく、ユーザーは主に親指でナビゲートするため、同じメガメニューを表示するとインターフェースが混雑し、タップしにくく、読み込みが遅くなります。

だからこそ、Navi+はデスクトップのメガメニューを自動的にモバイル版に変換しません。

代わりに、より良いユーザー体験を提供する専用のモバイルメニューを作成する必要があります：

* メインメニュー — 画面の下部にあるTabBarに配置し、親指で簡単にアクセスできるようにします。
* 追加メニュー — スライド（ハンバーガー）メニュー（スライドアウトパネル）またはモバイル専用のメガメニューに移動します。

👉 Navi+の便利なコピー＆ペーストメニュー機能を使用すれば、デスクトップのメガメニューを複製し、モバイル用に適応させるのにわずか数分で済みます — 完全なコントロールを提供し、両方のバージョンがそれぞれのプラットフォームに最適化されていることを保証します。
