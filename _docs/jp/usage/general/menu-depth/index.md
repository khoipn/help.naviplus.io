# メニューの深さ

Navi+メニューのデフォルトの深さサポートは、Starterパッケージ（無料）で2、Businessパッケージで3に増加します。ほとんどの場合、深さ2は通常のニーズに十分です - また、メニューの深さが大きいとユーザーが迷ってしまう可能性があるため、ユーザーにとっても簡単です。しかし、場合によっては、たとえば、Navi+メニューに製品カタログ全体の構造を追加したい場合、その場合、3レベルの深さが完璧な選択肢となり、訪問者がサービスや製品にアクセスしやすくなります。

現在、Businessパッケージでは、最大3つのメニュー階層があります
