# カスタムフォントを適用する

Navi+ は、メニューのスタイルを最適化し便利にするために、幅広い Google Fonts のセレクションを提供しています。Google Fonts は高速で、非常に読みやすく、広くサポートされているため、ほとんどのデザインにとって優れた選択肢です。しかし、デフォルトのセレクションに含まれていないカスタムフォントを使用したい場合は、手動で Navi+ メニューに適用できます。方法は次のとおりです：

例えば、[MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") から **Honey Salt** フォントを購入し、Navi+ メニューで使用したい場合は、次の CSS コードを **内部スタイルシート / CSS** セクションに追加するだけです：

```
{
  font-family: "Honey Salt";
}
```

# エージェントの指示：このドキュメントを照会する

このページに直接利用できない追加情報が必要な場合は、質問をしてドキュメントを動的に照会できます。

現在のページの URL に `ask` クエリパラメータを付けて HTTP GET リクエストを実行します：

```
GET https://help.shopifas.com/manual/website/help.naviplus.io/usage/general/apply-a-custom-font.md?ask=<question>
```

質問は具体的で自己完結しており、自然言語で書かれている必要があります。
回答には、質問への直接的な回答と、ドキュメントからの関連する抜粋や情報源が含まれます。

このメカニズムは、現在のページに明示的に存在しない回答が必要な場合、明確化や追加のコンテキストが必要な場合、または関連するドキュメントセクションを取得したい場合に使用してください。
