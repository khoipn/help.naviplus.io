# なぜ私のウェブサイトにNavi+メニューが表示されないのですか？

Navi+メニューをウェブサイトに表示するには、2つの明確なオプションがあります：

1. **Embed 
[Navi+ On all page]をテーマに埋め込み、Navi+アプリケーションで表示を調整する**：これは最も一般的で簡単な方法です。ただし、Navi+メニューは画面に固定することができます（下部、ヘッダー、または浮遊）。調整は非常に多様で、ニーズに応じてナビゲートできるように3つの異なるレベルがあります。
   * Level 1: Navi+メニューを表示/非表示にし、特定のページグループで表示/非表示にします。Shopifyでは、ホームページ、カタログ、製品、ページなどの一般的なページグループがあります。
   * Level 2: モバイルまたはデスクトップデバイスで表示/非表示にします。モバイルとデスクトップの間には全く異なる体験があるため、多くのユーザーはモバイルとデスクトップ用に別々のメニューを作成します。
   * Level 3: キーワードによって：ウェブページのURLに1つまたは複数のキーワードが含まれている場合/含まれていない場合に表示/非表示にできます。このオプションは非常に便利で柔軟性があり、各ページや各製品のために特定のメニューを作成できます。
2. **Session 
[Navi+ On Session]は、Shopifyテーマの各ページに直接埋め込むために使用されます**。このオプションでは、メニューを固定せず（下部、ヘッダー、または浮遊）、ページのコンテンツと共に流れるようにする追加の選択肢があります。この効果は、スクロールする際に流れるグリッドタイプのメニューやバータイプのメニューを作成したい場合に便利です。
