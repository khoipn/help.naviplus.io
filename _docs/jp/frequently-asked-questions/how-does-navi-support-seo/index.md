# Navi+はSEOをどのようにサポートしますか？

リンクは、検索エンジンでのSEOを最適化するための重要な要素の1つです。リンクを使用したSEOの最適化は、バックリンクと内部リンクの両方に適用されます。

1. **バックリンク** (External Links)\
   バックリンクは、他のウェブサイトからあなたのサイトに向けられたリンクです。これらはSEOにおいて重要な要素です。
   * **質が量より重要**: 評判の良いウェブサイトからのリンクは、不明瞭または低品質のソースからの多くのリンクよりもはるかに価値があります。
   * **自然なリンク**: 自然で、購入されていない、または交換されていないリンクは、SEOにとってより有益です。
   * **アンカーテキスト**: リンクとして使用するテキストは、リンク先のページのコンテンツに関連するキーワードを含むべきです。
2. **内部リンク**\
   同じウェブサイト内のページ間の内部リンクは、ユーザーを案内し、ページのランキングを改善するのに役立ちます。
   * **内部リンクを賢く使用する**: ウェブサイトのすべての重要なページが他のページからリンクされていることを確認してください。
   * **関連するアンカーテキスト**: 内部リンクのアンカーテキストには、関連するキーワードを使用して検索性を向上させます。

Navi+は、EliteプランからリンクベースのSEOを自動的にサポートし始めます。そのレベルでは、Navi+メニュー内のすべてのリンクがフルURL（検索エンジンによる適切なインデックス作成を保証）で最適化され、アンカーテキストと関連タグが付与されます。Navi+は、画像検索を最適化するためにタイトルとaltタグを追加することで、画像SEOもサポートしています。

Navi+メニューに製品カタログ全体を追加することは、検索エンジンにとって非常に有益であり、ページのリスト、サイトマップ、およびキーワード関連のコンテンツを理解するのに役立ちます。Navi+の速度もSEOにとっての大きな利点です。すべてのライブラリとリソース（画像、JS、CSS）は、応答時間が100ms未満で安定したキャッシングを持つトップティアのCDNシステムにホストされており、これがあなたのSEOスコアを向上させます。

このページに直接利用可能でない追加情報が必要な場合は、質問をしてドキュメントを動的に照会できます。


このメカニズムは、現在のページに明示的に存在しない回答が必要な場合、明確化や追加のコンテキストが必要な場合、または関連するドキュメントセクションを取得したい場合に使用します。
