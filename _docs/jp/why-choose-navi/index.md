# Navi+を選ぶ理由は？

ほとんどの商人は、店舗のすべての部分に慎重に考えを巡らせます。

すべての製品。すべての画像。すべての言葉。

しかし、ナビゲーションに関しては、テーマが提供するものを受け入れます。顧客に会ったことのない開発者が作った一般的なメニュー、彼らの店舗ではないためのものです。

誰も尋ねませんでした：

* 訪問者がモバイルで着地したとき — 彼らが最初に見るものは何ですか？
* カタログに200製品がある場合 — 誰かが10秒で正しいものを見つけることができますか？
* フラッシュセールが実行中 — メニューは実際に人々をそこに導きますか？

**それがNavi+が埋めるために作られたギャップです。**

***

### ナビゲーションは戦略的な決定であり、技術的な詳細ではありません

あなたのテーマはレイアウトとデザインを処理します。しかし、ナビゲーションはまったく異なる質問に答えます：

> *あなたの顧客にどこに行ってほしいですか — そしてどの順序で？*

その決定はあなたに属し、テーマ開発者ではありません。そして、それは訪問者が探しているものを見つけ、さらに探索し、購入するかどうかに直接影響します。

Navi+は、Shopify、WordPress、Wix、Webflowなど、あらゆるプラットフォームの商人に、自分の戦略、ブランド、顧客を反映したナビゲーションを構築する方法を提供します。コードは不要。妥協もありません。

***

### すべてのナビゲーションタイプ。1つのアプリ。

異なるコンテキストには異なるナビゲーションが必要です。モバイルショッパーはデスクトップ訪問者とは異なる行動をします。ファッションストアはテクニカルカタログとは異なる道案内が必要です。

Navi+には、独立してまたは一緒に使用できる5つのナビゲーション形式が含まれています：

| タイプ           | 最適な用途                                                        |
| -------------- | --------------------------------------------------------------- |
| **Tab Bar**    | モバイル用の固定ボトムナビ — 常に表示され、常にアクセス可能です |
| **Mega Menu**  | 列、画像、特集リンクを含むリッチデスクトップメニュー     |
| **Slide Menu** | 深いカタログや複雑な階層のためのマルチレベルパネル    |
| **Grid Menu**  | カテゴリが多いまたは画像優先の店舗のための視覚的ブラウジング        |
| **FAB**        | 浮遊するクイックアクセス — チャット、WhatsApp、サポート、カスタムアクション |

Not "pick one and live with it." 一緒に使用してください — 正しいコンテキスト、正しいデバイス、正しい顧客に合わせて。

ライブで見る： [showcase.naviplus.io](https://showcase.naviplus.io/ "showcase.naviplus.io")

***

### どのテーマでも動作します。数分でライブ。

Navi+は、既存のテーマの上に独立したレイヤーとして動作します — 無料または有料。テーマを切り替えたり、開発者を雇ったり、コードの1行にも触れる必要はありません。

UX専門家によって設計された既製のテンプレートで、2分以内にライブになります。サポートツール — WhatsApp、チャットボタン、ソーシャルリンク、コールアクション — はNavi+内に直接統合されているため、店舗はクリーンで高速のまま、余分なアプリを積み重ねることはありません。

1つのプラットフォーム。すべてのナビゲーションタイプ。完全なコントロール。

***

### 考えることなく十分に速い

ナビゲーションは訪問者が最初に対話するものです。遅延は全体の体験に影響します。

Navi+はこの制約の周りに構築されています。すべてのフロントエンドアセットは、[Cloudflare CDN](https://cloudflare.com "Cloudflare CDN")を通じてグローバルに配信され、各訪問者に最も近いエッジでキャッシュされます — そしてバックエンドから独立して提供されるため、他の何かがひっかかってもナビゲーションは表示されたままです。

Shopifyでは、Navi+は公式に**Built for Shopify**です — Shopifyの厳しい速度、デザイン品質、信頼性の基準を満たしています。
