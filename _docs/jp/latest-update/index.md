# 最新の更新

**2025 - 12月9日**

すべてのプラットフォームで動作するNavi+バージョンを発表します。Shopifyを超えて、WordPress、WooCommerce、Wix、その他のプラットフォームで利用可能です。[dash.naviplus.app](https://dash.naviplus.app/ "dash.naviplus.app")

**2025 - 11月7日**\
自動非表示、まだ表示される自動表示、デバイスの端に達したときにメニューを隠すために安全なエッジを作成しました。

**原因:** ウェブサイト上でiphone + androidのようなモバイルデバイスのUI/UXを扱うことは美しいですが、androidではエラーを引き起こすギャップを作成します。
