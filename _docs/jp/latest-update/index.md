# 最新の更新

### 2026年4月

多言語メニューが自動で動作するようになりました — Navi+ が訪問者の言語を検出し、追加設定なしでメニューラベルを即座に切り替えます。Tab Slide は新しいレイアウトオプションで、メニュー項目をフラットリストではなく横並びのタブにグループ化できます（例：スライドメニュー内に「メンズ / レディース / セール」タブ）。カラーピッカーがプリセットパレット付きで刷新され、ブランドカラーをメニュー全体に数秒で適用できます。すべての詳細設定 — アニメーション、タッチフィードバック、アイコン、CSSクラス — が一つのパネルにまとめられました。WordPress サポートが改善され、Elementor・Divi・WPBakery との競合が減りました。

---

### 2026年3月

セクションヘッダーがページへのリンクと子要素の展開を同時に行えるようになりました — タップで展開、矢印クリックでページへ移動。Grid Menu のレイアウトが小さい画面でも一貫して表示されるよう修正されました。区切り線やタッチハイライトの色を自由に設定できるようになりました。

---

### 2026年2月

Z-index の競合が自動で解決されるようになりました — カートドロワー・検索オーバーレイ・クッキーバナーが表示されると Navi+ が自動で後ろに回ります。Slide Horizontal が追加され、第2階層のパネルが右からスライドインして iOS アプリのような操作感になりました。メニュー設定をポップアップから JSON でエクスポート・インポートでき、他のストアへの移行や複製が簡単になりました。メニュー内の画像が遅延読み込みに対応しました。メニューラベルに絵文字を直接追加できるようになりました。

---

### 2026年1月

新しい Slide Horizontal メニュータイプがモバイルの深いナビゲーションツリー向けに追加されました — 各レベルが右からスライドし、戻るボタンとヘッダーが自動更新されます。Item template でメニュー構造を一度保存し、ワンクリックでどのメニューにも再利用できます。Import/Export が AI 対応になり、リンクリストやサイトマップを貼り付けると Navi+ がメニューツリーを自動生成します。Debug モードでダッシュボードからライブサイトに直接 z-index とメニュー境界を表示できます。CDN を Cloudflare R2 エッジネットワークに強化し、セカンダリフェイルオーバー CDN を追加しました。

---

### 2025年12月

Navi+ がすべてのプラットフォームで動作するようになりました — WordPress・WooCommerce・Wix・Webflow・Squarespace、そして `<script>` タグが使えるあらゆるサイトに対応。Shopify と同じダッシュボード、同じメニュータイプ、同じ一行埋め込みで利用できます。ダッシュボードが多言語対応しました。

---

### 2025年11月

一部のデバイスでページ上端・下端での自動表示/非表示がちらつくバグを修正しました。安全なエッジしきい値を追加し、メニューの表示/非表示が滑らかになりました。

---

### 2025年10月

Navi+ の全アセットを Cloudflare のグローバルエッジネットワークに移行しました — メニューの平均読み込み時間が約 52ms に短縮されました。Elite プランユーザーには BunnyCDN によるセカンダリフェイルオーバーが追加され、一方の CDN が停止してもメニューが正常に表示されます。SLA が 99.99% にアップグレードされました。
