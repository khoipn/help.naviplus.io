# Shopify: ハンバーガーメニューまたはカート・検索パネルを開く

Navi+は現在、ハンバーガーメニュー、カート、検索パネルを開くための人気テーマのリストをサポートしています。このリストにないテーマについては、Navi+にサポート追加を依頼できます（対応時間：1営業日）。

**サポートされているテーマ一覧：** Dawn, Spotlight, Sense, Refresh, Craft, Impulse, Tailor, Symmetry, District, Warehouse, Veena, Hongo, Shark, Honey, Focal, Impact, Broadcast, Expanse, ShowTime, Pipeline, Empire, Enterprise, Local, Avenue, Parallax, Palo Alto, Minion, Borders, Xclusive, Prestige...
