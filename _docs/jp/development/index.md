## クイックリファレンス

- **このメニューのカスタムCSS**を使用してメニュー固有のスタイルを設定する
- **Reusable Classes (CSS)**を使用してメニュー間で共有スタイルを設定する
- CSSは3つのレベルを対象とする：メインバー → サブメニュー → ネストされたサブメニュー
- JavaScriptライブラリ（例：TourGuideJS）はメニュー項目の属性で埋め込み可能
