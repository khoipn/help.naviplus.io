# デザイン区切り — セパレーターライン

設定は **デザイン → 区切り** で、メニュー内のすべての区切りのデフォルトの外観を定義します。

これは **メニュー レベル** のスタイルです — 個々のアイテムは独自の区切りを有効/無効にし、Edit Item → Basic Info → Add Divider を通じてスタイルを上書きできます。

---

## フィールド

| フィールド | 説明 | デフォルト |
|---|---|---|
| **区切り色** | ラインの色 (hex) | `#EEEEEE` |
| **区切りサイズ (px)** | ラインの厚さ | 1px |
| **区切りタイプ** | ラインスタイル: ソリッド / ダッシュ / ドット / ダブル | ソリッド |
| **区切り幅** | ラインのスパン: フル / インセット / テキストの下 | フル |

> **区切りタイプ** と **区切り幅** は Business または Elite プランが必要です。

---

## 仕組み

1. ここでの設定は、アイテムに **Add Divider** が有効な場合に適用される **デフォルト** です。
2. アイテムが独自の区切りスタイルを持っている場合 (Edit Item 内)、そのスタイルはここで設定されたものを **上書き** します。
3. レベル 2-3 は、`submenuDividerColor` フィールドが空の場合、このセクションから `Divider Color` を継承します。

---

## 区切り幅オプション

| 値 | 表示 |
|---|---|
| **フル** | ラインはアイテムの全幅に広がります |
| **インセット** | ラインは両側がインデントされています |
| **テキストの下** | ラインはアイテムのテキスト部分の下にのみ表示されます |
