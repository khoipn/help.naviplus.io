# デザインレベル2-3 — サブメニュー項目

**デザイン → レベル2-3**の設定は、親項目が開かれたときに表示されるサブメニュー項目（ドロップダウン）に適用されます。空のフィールドはすべて**自動的に**レベル1から対応する値を継承します。

> **ルール:** サブメニューをメインメニューとは**異なる**外観にしたい場合のみ、ここにフィールドを入力してください。空のままにすると = レベル1と同じです。

---

## テキストスタイル

| フィールド | 空の場合はレベル1から継承 |
|---|---|
| **テキストカラー** | `textColor` |
| **フォントサイズ(px)** | `textSize` |
| **フォントウェイト** | `fontWeight` |
| **テキスト変換** | `textTransform` |
| **フォントファミリー** | `fontFamily` |
| **説明カラー** | `descriptionColor` |
| **説明サイズ(px)** | `descriptionTextSize` |

---

## アイコン / 画像スタイル

| フィールド | 空の場合はレベル1から継承 |
|---|---|
| **アイカラー** | `iconColor` |
| **アイコンサイズ(px)** | `iconSize` |
| **スペーステキスト-アイコン(px)** | `spaceTextIcon` |
| **画像半径(px)** | `imageRadius` |

---

## スペーシング

| フィールド | 説明 |
|---|---|
| **アイテムパディング(px)** | サブメニュー項目の内部スペーシング。継承しない — 空の場合はテーマのデフォルトを使用 |
| **アイテムマージン(px)** | サブメニュー項目の外部スペーシング |
| **最小高さ(px)** | サブメニュー項目の最小高さ |

---

## 背景 & フレーム

| フィールド | 説明 |
|---|---|
| **背景色** | サブメニューエリアの背景色 |
| **ボーダー半径(px)** | サブメニューコンテナの角の丸み |
| **背景を隠す** | サブメニューの背景を隠す |
| **ドロップシャドウ** | サブメニューコンテナのドロップシャドウ |

---

## 表示

| フィールド | 空の場合はレベル1から継承 |
|---|---|
| **表示レイアウト** | `displayLayout` |
| **アライン** | `align` |

---

## ディバイダー

| フィールド | 空の場合はレベル1から継承 |
|---|---|
| **ディバイダー色** | `dividerColor` (デザイン → ディバイダーから) |

---

## 例

**目標:** メインメニューはダーク背景とホワイトテキスト。サブメニューはホワイト背景とダークテキストにする。

レベル2-3でこれらの2つのフィールドのみを入力します:
- 背景色 = `#FFFFFF`
- テキストカラー = `#000000`

他のすべてのフィールド（フォントサイズ、アイコンサイズ、スペーシングなど）は自動的にレベル1から継承されます。
