# デザインレベル1 — メインメニュー項目

**デザイン → レベル1**の設定は、メニュー内のすべてのトップレベル項目に適用されます。これは基本レイヤーです — レベル2-3はこれらの値を継承し、項目ごとの設定がそれを上書きします。

---

## テキストスタイル

| フィールド | 説明 | デフォルト |
|---|---|---|
| **テキストカラー** | ラベルテキストの色 | テーマ継承 |
| **フォントサイズ(px)** | テキストサイズ | テーマ継承 |
| **フォントウェイト** | 厚さ: 100–900 または 継承 | 継承 |
| **テキスト変換** | なし / 大文字化 / 大文字 / 小文字 | なし |
| **フォントファミリー** | Googleフォントまたは"デフォルト (テーマ)" | デフォルト |
| **説明カラー** | 二次説明行の色 | テキストカラーの60%不透明度 |
| **説明サイズ(px)** | 説明行のフォントサイズ | フォントサイズの70% |

---

## アイコン / 画像スタイル

| フィールド | 説明 | デフォルト |
|---|---|---|
| **アイカラー** | ベクターアイコンの色 | テキストカラーを継承 |
| **アイコンサイズ(px)** | アイコンの寸法 | 22px |
| **スペース テキスト-アイコン(px)** | アイコンとテキストの間の隙間 | 自動 |
| **画像半径(px)** | サムネイル画像の角の丸み | 0 |

---

## スペーシング

| フィールド | 説明 | フォーマット |
|---|---|---|
| **アイテムパディング(px)** | 各アイテムの内側の間隔 | `top right bottom left` または `mobile \\| desktop` |
| **アイテムマージン(px)** | 各アイテムの外側の間隔 | `top right bottom left` または `mobile \\| desktop` |

---

## 背景 & フレーム

| フィールド | 説明 | デフォルト |
|---|---|---|
| **背景色** | アイテムエリアの背景色 | 透明 |
| **背景画像** | 背景画像 (URLまたはギャラリー) | なし |
| **不透明度** | 背景の不透明度 (0–100) | 100 |
| **背景を隠す** | 背景を隠してデバイスの背景を表示 | オフ |
| **高さ(px)** | 各アイテムの行の高さ | 50px |
| **ボーダー半径(px)** | 角の丸み | 0 |
| **ドロップシャドウ** | アイテムの下のドロップシャドウ | オフ |
| **最小高さ(px)** | 最小アイテム高さ | 制限なし |

---

## 表示

| フィールド | 説明 |
|---|---|
| **表示レイアウト** | アイコン + テキストの配置: 自動 / 上下 / 左右 / 下上 / 右左 / アイコンのみ / テキストのみ / 空 |
| **整列** | コンテンツの整列: 自動 / 左 / 中央 / 右 |
| **テキストを隠す** | すべてのテキストラベルを隠す — アイコンのみ |
| **展開矢印を表示** | 子を持つ親アイテムに展開矢印を表示 |

---

## バッジ

| フィールド | 説明 |
|---|---|
| **バッジカラー** | アイテムの通知バッジ/ドットの背景色 |

---

## カラーテンプレート

Navi+にはプリセットのカラーパレットが含まれています: **ライト、ダーク、オーシャン、サンド、ローズ、フォレスト**。

テンプレートを選択すると、一般的なカラーバリューが自動的に入力されます。適用後に個々のフィールドを調整できます。
