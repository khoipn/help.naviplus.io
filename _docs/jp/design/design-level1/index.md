# デザインレベル 1 — メインメニュー項目

デザイン → レベル 1 の設定はすべてのトップレベル項目に適用されます。ベースレイヤー — レベル 2-3 が継承し、項目ごとがオーバーライドします。

## テキストスタイル

| フィールド | 説明 | デフォルト |
|---|---|---|
| Text Color | ラベルテキストの色 | テーマ継承 |
| Font Size (px) | テキストサイズ | テーマ継承 |
| Font Weight | 100–900 または Inherit | Inherit |
| Text Transform | None / Capitalize / Uppercase / Lowercase | None |
| Font Family | Google Font または「Default (Theme)」 | Default |
| Description Color | セカンダリライン色 | Text Color の 60% 不透明度 |
| Description Size (px) | 説明フォントサイズ | Font Size の 70% |

## アイコン / 画像スタイル

| フィールド | 説明 | デフォルト |
|---|---|---|
| Icon Color | ベクターアイコンの色 | Text Color を継承 |
| Icon Size (px) | アイコンの寸法 | 22px |
| Space Text-Icon (px) | アイコンとテキストの間隔 | Auto |
| Image Radius (px) | サムネイルのコーナー丸め | 0 |

## 間隔

| フィールド | 説明 | 形式 |
|---|---|---|
| Item Padding (px) | 内部の間隔 | 上 右 下 左 または モバイル \| デスクトップ |
| Item Margin (px) | 外部の間隔 | 上 右 下 左 または モバイル \| デスクトップ |

## 背景とフレーム

| フィールド | 説明 | デフォルト |
|---|---|---|
| Background Color | 項目の背景 | 透明 |
| Background Image | URL またはギャラリー | なし |
| Opacity | 背景の不透明度（0–100） | 100 |
| Hide Background | 背景を非表示 | オフ |
| Height (px) | 行の高さ | 50px |
| Border Radius (px) | コーナー丸め | 0 |
| Drop Shadow | ドロップシャドウ | オフ |
| Min Height (px) | 最小高さ | 制限なし |

## 表示

| フィールド | 説明 |
|---|---|
| Display Layout | Auto / Top-Down / Left-Right / Down-Top / Right-Left / Icon Only / Text Only / Empty |
| Align | Auto / Left / Center / Right |
| Hide Text | すべてのテキストラベルを非表示 |
| Show Expand Arrow | 親項目に矢印を表示 |

## バッジ

| フィールド | 説明 |
|---|---|
| Badge Color | バッジ／ドットの背景色 |

## カラーテンプレート

プリセット：Light、Dark、Ocean、Sand、Rose、Forest。色の値を自動入力します。
