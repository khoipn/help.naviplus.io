# Usa la funzione multilingue.

Sintassi:

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

Esempio:

```
Home <fr: Maison><zh: 家>
```

Ecco l'elenco COMPLETO predefinito di tutti i codici lingua supportati da Navi+.

A volte, puoi utilizzare sistemi SEO per convertire questi codici lingua di due caratteri in un formato diverso, ad esempio, cambiando EN in en-EN o english. In quel caso, puoi comunque utilizzare il codice lingua normalmente e facilmente.
