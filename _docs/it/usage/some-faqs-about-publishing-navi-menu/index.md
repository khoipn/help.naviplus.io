# Alcune FAQ sulla pubblicazione del menu Navi+

**Perché Navi+ non viene visualizzato sul mio sito?**

Verifica di aver completato tutti i passaggi. Per Tabbar/FAB, hai bisogno di: 1) Incorporare l'app Navi+ nel tuo tema, 2) Abilitare l'interruttore di pubblicazione del menu.
