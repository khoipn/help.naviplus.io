# Esempi di pubblicazione

Di seguito sono riportati alcuni esempi di come pubblicare un Slide Menu sul tuo sito web.

#### Metodo 1: Apri questo menu da un altro elemento del menu Navi+ utilizzando la sintassi open:NaviMenu(SF-123456789)

Questo è un modo semplice per pubblicare un Slide Menu quando si utilizzano più menu Navi+. Ad esempio, un elemento della Tabbar può essere utilizzato per aprire il Slide Menu e espandere lo spazio del menu.
