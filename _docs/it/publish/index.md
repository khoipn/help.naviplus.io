# Pubblica

La **sezione Pubblica** controlla come appare il tuo menu sul tuo sito web live: attivare/disattivare il menu, scegliere il metodo di incorporamento, configurare la visibilità del dispositivo e filtrare per URL. Questo è l'ultimo passaggio dopo la progettazione — senza salvare le impostazioni di Pubblica, il menu non apparirà.

## Riferimento veloce

```
Tipo di menu           Metodo primario
────────────────────────────────────────────────────
Sticky / TABBAR     App Embeds (Shopify) / <head> (Globale) + abilita toggle
Sticky / FAB        App Embeds (Shopify) / <head> (Globale) + abilita toggle
Slide (CONTEXT)     App Embeds / <head> + abilita toggle + configura trigger
Sezione (Mega/Grid) Inserisci/Sostituisci (Selettore CSS) o Blocco App (Shopify)
```

**Regola:** Abilita toggle = menu è live. Disabilita toggle = completamente nascosto dal sito web, non è necessario rimuovere il codice di incorporamento.
