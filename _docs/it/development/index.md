## Riferimento Rapido

- Usa **CSS personalizzato per questo menu** per stili specifici di ogni menu
- Usa **Reusable Classes (CSS)** per stili condivisi tra i menu
- Il CSS si rivolge a tre livelli: barra principale → sottomenu → sottomenu annidato
- Le librerie JavaScript (es. TourGuideJS) possono essere integrate tramite attributi degli elementi di menu
