# Come installare Navi+

Non è necessario conoscere lo sviluppo o il design per utilizzare Navi+. È un software preconfezionato che può essere facilmente integrato nel tuo sito web. Tutta la configurazione — comprese le funzionalità e il design visivo — è gestita direttamente su Navi+, senza causare alcun conflitto con il tuo sistema esistente.

***

### Passo 1: Installa l'Applicazione

**Se stai usando Shopify:**\
Si prega di installare l'app ufficiale dal link qui sotto:\
<https://apps.shopify.com/pronavi-navigation-design>

**Se stai usando altre piattaforme** come WordPress, WooCommerce, Wix, Webflow, Squarespace, o un sito web self-hosted:

1. Vai su <https://naviplus.io>
2. Crea un account gratuito
3. Accedi e collega il tuo sito web seguendo le istruzioni.

***

### Passo 2: Crea Menu e Integrali nel Tuo Sito Web

Una volta effettuato l'accesso, puoi iniziare a creare menu per il tuo sito web.\
**Navi+** fornisce diversi tipi di menu per soddisfare diverse esigenze:

* **Tabbar**\
  Una barra di navigazione fissa nella parte inferiore dello schermo, che consente agli utenti di accedere rapidamente alle sezioni chiave. Ideale per layout mobili.
* **Mega Menu**\
  Un ampio menu a discesa con più colonne, perfetto per visualizzare categorie di prodotti o contenuti dettagliati in una struttura chiara.
* **Slide Menu (Hamburger Menu)**\
  Un menu laterale nascosto che scorre dal bordo dello schermo. Ottimo per risparmiare spazio e mantenere un design pulito e moderno.
* **Floating Button Menu**\
  Un pulsante di azione flottante che può aprire un elenco di link rapidi o evidenziare campagne o promozioni speciali.

Ogni tipo di menu ha il proprio processo di configurazione, che è completamente guidato passo dopo passo all'interno del dashboard di Navi+.

Puoi anche utilizzare più tipi di menu di Navi+ contemporaneamente su diverse pagine — ad esempio, un Tabbar per mobile, un Mega Menu per desktop, e un Floating Button per la tua call-to-action o promozioni.
