# Cos'è Navi+? (Come funziona)

Navi+ è un **costruttore di navigazione senza codice** — ti permette di creare menu bellissimi e ad alta conversione per qualsiasi sito web senza toccare il tuo tema né scrivere una singola riga di codice.

Se i visitatori del tuo sito faticano a trovare quello che cercano, il tuo menu potrebbe essere il problema. Navi+ ti aiuta a risolverlo in pochi minuti.

## Cinque tipi di menu

Scegli il menu giusto per il tuo caso d'uso:

| Tipo di menu | Ideale per |
| --- | --- |
| **Tab Bar** | Navigazione mobile-first, si sente come un'app nativa |
| **Mega Menu** | Dropdown ricchi con immagini, colonne e link — versioni separate per Desktop e Mobile |
| **Slide Menu** | Menu hamburger con supporto multi-livello |
| **Grid Menu** | Navigazione visuale delle categorie — ottimo per l'e-commerce |
| **FAB** | Pulsante flottante di accesso rapido, sempre a portata di mano |

Puoi usare più tipi di menu insieme sullo stesso sito.

## Come funziona

Navi+ è un **widget** — non sostituisce né modifica il tuo tema esistente. Invece, inietta uno script leggero nella tua pagina che renderizza il menu sopra il tuo layout esistente.

Costruisci e gestisci tutto dal **pannello Navi+**. Le modifiche vanno in produzione istantaneamente — nessun bisogno di modificare il codice, cambiare template o ripubblicare il tuo tema.

Il flusso:

1. Crei un menu nel pannello
2. Navi+ genera un piccolo script
3. Quello script viene incorporato nel tuo sito (tramite App Block su Shopify, o uno snippet di codice altrove)
4. Il menu si renderizza sopra il tuo layout esistente ad ogni caricamento di pagina

## Funziona con la tua piattaforma

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · e qualsiasi sito che accetti un tag `<script>`.

## Cosa Navi+ NON È

- **Non è un tema** — funziona a fianco del tuo tema esistente, non al suo posto
- **Non è codice che installi manualmente** — un App Block (Shopify) o uno snippet di incorporamento è tutto ciò che serve
- **Non è un plugin che può rompere il tuo sito** — se Navi+ viene disabilitato, il menu originale del tuo tema torna automaticamente

## Progettato per la velocità

Gli asset Navi+ vengono consegnati tramite Cloudflare CDN con ~52ms di tempo di caricamento medio — tipicamente più veloce degli script nativi della tua piattaforma.
