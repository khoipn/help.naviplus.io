# Cos'è Navi+?

Navi+ è un **costruttore di navigazione senza codice** — ti consente di creare menu belli e ad alta conversione per qualsiasi sito web senza toccare il tuo tema o scrivere una sola riga di codice.

Se i visitatori del tuo sito faticano a trovare ciò che cercano, il tuo menu potrebbe deluderli. Navi+ ti aiuta a risolvere questo problema in pochi minuti.

## Cinque tipi di menu

Scegli il menu giusto per il tuo caso d'uso:

| Tipo di Menu | Ideale per |
| --- | --- |
| **Tab Bar** | Navigazione mobile-first, sembra un'app nativa |
| **Mega Menu** | Dropdown ricchi con immagini, colonne e link — ha versioni separate per Desktop e Mobile |
| **Slide Menu** | Menu hamburger con supporto multi-livello |
| **Grid Menu** | Navigazione visiva delle categorie — ottimo per l'e-commerce |
| **FAB** | Pulsante flottante di accesso rapido, sempre a portata di mano |

Puoi utilizzare più tipi di menu insieme sullo stesso sito.

## Come funziona

Navi+ è un **widget** — non sostituisce o modifica il tuo tema esistente. Invece, inietta uno script leggero nella tua pagina che rende il menu sopra il tuo layout esistente.

Costruisci e gestisci tutto dal **dashboard di Navi+**. Le modifiche vengono pubblicate istantaneamente — non è necessario modificare codice, modificare template o ripubblicare il tuo tema.

Ecco il flusso:

1. Crei un menu nel dashboard
2. Navi+ genera un piccolo script
3. Quello script è incorporato nel tuo sito (tramite App Block su Shopify, o un frammento di codice altrove)
4. Il menu viene reso sopra il tuo layout esistente ad ogni caricamento della pagina

## Funziona con la tua piattaforma

Navi+ si integra con gli strumenti che già utilizzi:

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · e qualsiasi sito web che accetta un `<script>` tag.

## Cosa NON è Navi+

- **Non è un tema** — funziona insieme al tuo tema esistente, non al suo posto
- **Non è codice che installi manualmente** — un App Block (Shopify) o un frammento di incorporamento è tutto ciò che serve
- **Non è un plugin che può rompere il tuo sito** — se Navi+ è disabilitato, il menu originale del tuo tema torna automaticamente

## Costruito per la velocità

Le risorse di Navi+ vengono fornite tramite Cloudflare CDN a ~52ms di tempo di caricamento medio — tipicamente più veloce degli script della tua piattaforma. Dopo il primo caricamento, i menu vengono memorizzati nella cache e si caricano istantaneamente.
