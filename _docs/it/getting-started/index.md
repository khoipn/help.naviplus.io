# Iniziare

Navi+  Sito Web Menu Builder è un costruttore di navigazione per siti web senza codice che ti aiuta a guidare i visitatori nel posto giusto — più velocemente.

Crea menu moderni come **Tabbar, Mega Menu, Slide (Hamburger), Grid Menu, e Floating Button** in pochi minuti. Basta trascinare e rilasciare. Nessuna codifica richiesta.

Navi+ funziona senza problemi con [Shopify](https://www.shopify.com/ "Shopify"), [Wix](https://www.wix.com/ "Wix"), [WordPress](https://wordpress.org/ "WordPress"), [WooCommerce](https://woocommerce.com/ "WooCommerce"), [Webflow](https://webflow.com/ "Webflow"), [Squarespace](https://www.squarespace.com/ "Squarespace"), e qualsiasi sito web tramite codice di incorporamento.

È basato su cloud, velocissimo e non ha impatto sulla velocità della pagina o SEO.

Inizia gratuitamente. Aggiorna quando hai bisogno di più controllo e scalabilità.

Progetta una navigazione più intelligente. Migliora l'esperienza utente. Aumenta le conversioni.
