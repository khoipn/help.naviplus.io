# Elementi di menu

Un **elemento di menu** è una singola riga nel tuo elenco di menu. Gli elementi sono i mattoni di ogni menu Navi+ — ognuno può essere un collegamento di navigazione, un'intestazione di gruppo, uno spazio vuoto o un divisore di schede.

## Riferimento rapido

**Dove modifico un elemento?**  
Fai clic su qualsiasi elemento nell'albero del menu → il pannello Modifica elemento si apre a destra. Premi **Update** per applicare le modifiche, quindi **Save** (`Ctrl+S` / `Cmd+S`) per pubblicare.

**Ordine dei campi in Modifica elemento:**  
Media (icona / immagine) → Informazioni di base (testo, link, descrizione, badge, divisore) → Layout → Interfaccia (colori, spaziatura) → Avanzate (animazione, CSS) → Pubblica (visibilità)
