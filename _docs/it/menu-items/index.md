# Elementi del menu

Un **elemento del menu** è una singola riga nella tua lista di menu. Gli elementi sono i mattoni di ogni menu Navi+ — ognuno può essere un collegamento di navigazione, un'intestazione di gruppo, uno spazio vuoto o un divisore di schede.

## Riferimento rapido

**Dove modifico un elemento?**  
Clicca su qualsiasi elemento nell'albero del menu → si apre il pannello Modifica elemento a destra. Premi **Aggiorna** per applicare le modifiche, poi **Salva** (`Ctrl+S` / `Cmd+S`) per pubblicare.

**Ordine dei campi in Modifica elemento:**  
Media (icona / immagine) → Informazioni di base (testo, collegamento, descrizione, distintivo, divisore) → Layout → Interfaccia (colori, spaziatura) → Avanzate (animazione, CSS) → Pubblica (visibilità)
