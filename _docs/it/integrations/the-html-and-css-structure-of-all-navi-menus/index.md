# La struttura HTML e CSS di tutti i menu Navi+

Di seguito è riportata la struttura HTML e CSS per tutti i menu Navi+, che rimarrà stabile per tutto il ciclo di vita del prodotto. Anche se potrebbero esistere variazioni minori per tipi di menu specifici, la struttura di base riportata di seguito non cambierà. Puoi fare riferimento a questa struttura per scrivere codice CSS personalizzato per un design di menu avanzato.
