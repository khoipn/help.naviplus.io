# Design automatique par Navi+ AI

Laissez Navi+ gérer tout le travail de l'IA — il vous suffit de décrire le menu que vous voulez directement dans l'éditeur. Pas d'outils externes, pas de copier-coller de JSON. *(Business / Elite)*

Ouvrir : le bouton **AI** → carte **Design automatique par Navi+ AI**.

---

## En quoi est-ce différent de « Design automatique avec ton app IA » ?

| | Ton app IA | Navi+ AI |
|---|---|---|
| Outil IA | Vous ouvrez ChatGPT / Claude / Gemini | Navi+ appelle l'IA pour vous |
| Copier-coller JSON | Oui, manuellement | Pas nécessaire |
| Étapes | 4 étapes | 1 étape (chat) |
| Forfait | Gratuit | Business / Elite |

---

## Avant de commencer — sauvegardez votre menu

> **⋮ Plus → Backup / Restore** — sauvegardez votre menu actuel avant d'appliquer le résultat de l'IA.

---

## Comment l'utiliser

1. Ouvrez la carte **Design automatique par Navi+ AI**.
2. Tapez une description dans la zone de chat, par exemple « Crée une Tab Bar de 5 éléments pour une boutique de mode, en noir et blanc. »
3. (Facultatif) Joignez une image de référence avec le bouton **Upload** pour que l'IA suive la mise en page / les couleurs.
4. Cliquez sur **Send**. L'IA renvoie un design et l'applique à l'éditeur pour aperçu.
5. Continuez à discuter pour affiner (« change l'icône Contact », « supprime le dernier élément »)... ou cliquez sur **New chat** pour recommencer.
6. Satisfait ? Cliquez sur **Save**.

> Votre historique de chat est enregistré — cliquez sur **History** pour rouvrir les sessions de design précédentes.

---

## Forfait requis

Cette fonctionnalité fait partie du forfait **Business / Elite**. Avec un forfait inférieur, vous verrez un avis de mise à niveau lorsque vous ouvrez cette carte.

---

## Exemples de prompts

- Crée une Tab Bar de 5 éléments pour une boutique de cosmétiques, tons rose pastel.
- Ajoute une icône à chaque élément et transforme le dernier élément en bouton Contact bien visible.
- Conçois un Mega Menu comme l'image que je viens d'envoyer. *(joignez une image avec le bouton Upload)*
- Supprime l'élément Blog, fusionne Promotions et Soldes en un seul.

---

## Étude de cas — construire une tab bar de cosmétiques en 3 tours de chat

**Contexte :** Une nouvelle boutique de cosmétiques sans menu pour le moment souhaite une Tab Bar correspondant à son image de marque rose pastel.

1. **Sauvegardez** d'abord par précaution (⋮ Plus → Backup / Restore).
2. Ouvrez la carte **Design automatique par Navi+ AI**. Tour 1 :

   ```
   Crée une Tab Bar de 5 éléments pour une boutique de cosmétiques :
   Accueil, Produits, Offres, Guide beauté, Compte. Tons rose pastel.
   ```

   L'IA construit la Tab Bar et l'affiche immédiatement dans l'aperçu.
3. Tour 2 — joignez une image d'UI d'exemple avec le bouton **Upload**, puis tapez :

   ```
   Adapte le rayon des coins et les icônes au style de cette image.
   ```

4. Tour 3 — un petit ajustement :

   ```
   Transforme l'élément Compte en bouton Contact bien visible avec une
   icône de téléphone.
   ```

5. Satisfait, cliquez sur **Save**. La prochaine fois que vous aurez besoin de modifications, cliquez sur **History** pour rouvrir cette session.

**Résultat :** Une Tab Bar correspondant à l'image de marque, terminée en 3 tours de chat sans quitter Navi+.
