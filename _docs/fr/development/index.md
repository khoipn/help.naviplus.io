## Référence Rapide

- Utilisez **CSS personnalisé pour ce menu** pour les styles propres à chaque menu
- Utilisez **Reusable Classes (CSS)** pour les styles partagés entre les menus
- Le CSS cible trois niveaux : barre principale → sous-menu → sous-menu imbriqué
- Les bibliothèques JavaScript (ex. TourGuideJS) peuvent être intégrées via les attributs des éléments de menu
