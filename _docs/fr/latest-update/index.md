# Dernière mise à jour

**2025 - 9 déc**

Lancement de la version Navi+ qui fonctionne sur toutes les plateformes au-delà de Shopify : WordPress, WooCommerce, Wix, et d'autres à [dash.naviplus.app](https://dash.naviplus.app/ "dash.naviplus.app")

***

**2025 - 7 novembre**\
Correction de la fonction de masquage automatique, le masquage automatique montre toujours, masque le menu lorsqu'il atteint le bord de l'appareil en créant un bord de sécurité.

**Cause :** Gestion de l'UI/UX sur les appareils mobiles comme iphone + android sur le site web le rend beau, android mais crée un écart qui cause des erreurs.

***

**2025 - 20 octobre**\
Migration de tous les services CDN de BunnyCDN vers Cloudflare en raison d'un dysfonctionnement de purge de cache sur BunnyCDN.
