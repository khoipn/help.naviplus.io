# Éléments de menu

Un **élément de menu** est une seule ligne dans votre liste de menu. Les éléments sont les blocs de construction de chaque menu Navi+ — chacun peut être un lien de navigation, un en-tête de groupe, un séparateur vide ou un onglet diviseur.

## Référence rapide

**Où puis-je modifier un élément ?**  
Cliquez sur n'importe quel élément dans l'arborescence du menu → le panneau "Modifier l'élément" s'ouvre à droite. Appuyez sur **Mettre à jour** pour appliquer les modifications, puis **Enregistrer** (`Ctrl+S` / `Cmd+S`) pour publier.

**Ordre des champs dans "Modifier l'élément" :**  
Médias (icône / image) → Infos de base (texte, lien, description, badge, séparateur) → Mise en page → Interface (couleurs, espacement) → Avancé (animation, CSS) → Publier (visibilité)
