# Shopify : Ouvrir le menu hamburger ou le panier, panneau de recherche

Actuellement, Navi+ prend en charge une liste de thèmes populaires pour ouvrir les menus Hamburger, le Panier et les panneaux de recherche. Pour les thèmes qui ne figurent pas sur cette liste, vous pouvez demander à Navi+ d'ajouter le support (délai de réponse : 1 jour ouvrable).

**Liste des thèmes pris en charge :** Dawn, Spotlight, Sense, Refresh, Craft, Impulse, Tailor, Symmetry, District, Warehouse, Veena, Hongo, Shark, Honey, Focal, Impact, Broadcast, Expanse, ShowTime, Pipeline, Empire, Enterprise, Local, Avenue, Parallax, Palo Alto, Minion, Borders, Xclusive, Prestige...
