# La structure HTML et CSS de tous les menus Navi+

Ci-dessous se trouve la structure HTML et CSS pour tous les menus Navi+, qui restera stable tout au long du cycle de vie du produit. Bien que des variations mineures puissent exister pour des types de menus spécifiques, la structure de base ci-dessous ne changera pas. Vous pouvez vous référer à cette structure pour écrire du code CSS personnalisé pour un design de menu avancé.
