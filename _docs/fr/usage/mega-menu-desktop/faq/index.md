# FAQ

#### Question 1: Pourquoi Navi+ ne montre-t-il pas automatiquement le Mega Menu desktop sur les appareils mobiles ?

L'expérience de navigation sur desktop et mobile est fondamentalement différente — surtout en ce qui concerne la façon dont les utilisateurs naviguent et interagissent avec un site web.

Sur desktop, les utilisateurs ont plus d'espace à l'écran et peuvent facilement voir de grands et complexes mega menus affichés sous l'en-tête.

Cependant, sur mobile, l'écran est plus petit et les utilisateurs naviguent principalement avec leurs pouces, donc afficher le même mega menu rendrait l'interface encombrée, difficile à toucher et plus lente à charger.

C'est pourquoi Navi+ ne convertit pas automatiquement un mega menu desktop en une version mobile.

Au lieu de cela, vous devriez créer un menu mobile dédié qui offre une meilleure expérience utilisateur :

* Menus principaux — placez-les dans une TabBar en bas de l'écran, où ils sont faciles à atteindre avec un pouce.
* Menus supplémentaires — déplacez-les dans un Slide (Hamburger) menu (panneau coulissant) ou un Mega Menu spécifique aux mobiles.

👉 Avec la fonction pratique de menu Copier & Coller de Navi+, vous pouvez dupliquer votre mega menu desktop et l'adapter pour mobile en quelques minutes seulement — vous donnant un contrôle total et garantissant que les deux versions sont parfaitement optimisées pour leurs plateformes.
