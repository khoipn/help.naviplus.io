# Exemples de publication

Voici quelques exemples de la façon de publier un Slide Menu sur votre site web.

#### Méthode 1 : Ouvrir ce menu à partir d'un autre élément de menu Navi+ en utilisant la syntaxe open:NaviMenu(SF-123456789)

C'est un moyen simple de publier un Slide Menu lors de l'utilisation de plusieurs menus Navi+. Par exemple, un élément de Tabbar peut être utilisé pour ouvrir le Slide Menu et étendre l'espace du menu.
