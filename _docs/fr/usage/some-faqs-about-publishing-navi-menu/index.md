# Quelques FAQ sur la publication du menu Navi+

**Pourquoi Navi+ ne s'affiche-t-il pas sur mon site ?**

Vérifiez que vous avez complété toutes les étapes. Pour Tabbar/FAB, vous devez : 1) Intégrer l'application Navi+ dans votre thème, 2) Activer le commutateur de publication du menu.
