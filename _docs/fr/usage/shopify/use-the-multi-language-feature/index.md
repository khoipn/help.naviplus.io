# Utilisez la fonction multilingue.

Syntaxe :

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

Exemple :

```
Home <fr: Maison><zh: 家>
```

Voici la liste complète par défaut de tous les codes de langue pris en charge par Navi+.

Parfois, vous pouvez utiliser des systèmes SEO pour convertir ces codes de langue à deux caractères dans un format différent, par exemple, changer EN en en-EN ou english. Dans ce cas, vous pouvez toujours utiliser le code de langue normalement et facilement.
