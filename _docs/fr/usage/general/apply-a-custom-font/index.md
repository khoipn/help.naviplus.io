# Appliquer une police personnalisée

Navi+ propose une large sélection de Google Fonts, offrant un moyen optimisé et pratique de styliser votre menu. Google Fonts sont rapides, très lisibles et largement supportés, ce qui en fait un excellent choix pour la plupart des designs. Cependant, si vous souhaitez utiliser une police personnalisée qui n'est pas incluse dans la sélection par défaut, vous pouvez l'appliquer manuellement à votre menu Navi+. Voici comment :

Par exemple, si vous avez acheté la police **Honey Salt** chez [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") et que vous souhaitez l'utiliser dans votre menu Navi+, ajoutez simplement le code CSS suivant à la section **Feuille de style interne / CSS** :

```
{
  font-family: "Honey Salt";
}
```
