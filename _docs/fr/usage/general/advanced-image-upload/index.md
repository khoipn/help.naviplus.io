# Téléchargement avancé d'images

#### I. Pourquoi les tailles d'image plus petites sont-elles meilleures pour le SEO et l'expérience utilisateur ?

Utiliser des tailles d'image plus petites aide votre site Web à se charger plus rapidement. Les moteurs de recherche comme Google considèrent la vitesse de la page comme un facteur de classement important, donc des images optimisées peuvent améliorer votre SEO.

Un chargement plus rapide rend également le site Web plus fluide et plus agréable pour les visiteurs, en particulier sur les appareils mobiles ou les connexions plus lentes. Cela signifie que les utilisateurs sont plus susceptibles de rester sur votre site et de continuer à explorer vos produits.

#### II. Comment Navi+ traite-t-il vos images ?

Navi+ télécharge d'abord votre image dans sa taille d'origine. Ensuite, il compresse l'image avec un algorithme optimisé et réduit la taille d'affichage. Cette taille est soigneusement choisie par Navi+ pour être idéale à la fois pour l'expérience utilisateur et le SEO, puisque les images dans les menus sont généralement des vignettes ou des icônes qui doivent être petites et se charger rapidement.

**Remarque :** Pour les petites images comme les logos ou les icônes, Navi+ ne les compresse pas pour préserver la netteté.

Après cela, Navi+ stocke votre image sur BunnyCDN — un puissant réseau de distribution de contenu (CDN) pour les images, JavaScript, CSS, et plus encore. BunnyCDN fournit des vitesses de chargement extrêmement rapides grâce à son réseau de serveurs mondial. Cela garantit que votre site Web peut gérer des millions de visites sans ralentir ni provoquer de retards.

#### III. Que dois-je faire si je veux garder mes images à leur qualité d'origine ?

Dans certains cas particuliers, vous pouvez vouloir que Navi+ ne modifie pas la qualité de votre image—sinon, la compression pourrait réduire la qualité que vous attendez, par exemple, certaines images de produits importantes. Dans ce cas, vous devez télécharger l'image directement sur Shopify et fournir le lien à Navi+ pour conserver la qualité d'origine.

**Étape 1 : Préparez votre image (processus manuel)**\
Préparez votre image comme souhaité et renommez-la pour l'optimisation SEO. (Navi+ ne renomme les images pour le SEO que si elles sont téléchargées via l'outil Navi+.)\
Vous devez également compresser l'image à une taille raisonnable—idéalement quelques centaines de Ko. Vous pouvez utiliser le redimensionneur d'images de Shopify ici : <https://www.shopify.com/tools/image-resizer>.

**Étape 2 : Téléchargez sur Shopify**\
Cliquez avec le bouton droit sur le menu **Content**, sélectionnez **Open link in new Tab**, puis allez dans **Files**. Faites glisser et déposez l'image préparée à l'étape 1. Une fois téléchargée, trouvez votre fichier dans la liste et cliquez sur l'icône de lien pour copier son URL. Cette URL sera utilisée dans Navi+ pour télécharger l'image.
