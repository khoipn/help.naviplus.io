# Profondeur du menu

La profondeur de support par défaut pour le menu Navi+ est de 2 pour le package Starter (gratuit) et augmentée à 3 pour le package Business. Dans la plupart des cas, une profondeur de 2 est suffisante pour des besoins ordinaires - Et aussi facile pour les utilisateurs car une grande profondeur du menu peut faire perdre leur chemin aux utilisateurs. Cependant, dans certains cas, par exemple, vous pouvez vouloir ajouter l'ensemble de la structure de votre catalogue de produits au menu Navi+, dans ce cas, une profondeur de 3 niveaux pourrait être un choix parfait, facilitant l'accès de vos visiteurs à vos services et produits.

Actuellement, avec le package Business, vous avez jusqu'à 3 niveaux de menu
