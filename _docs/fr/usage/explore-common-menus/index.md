# Explorer les types de menus

👀 **Vous voulez les voir en action ?** [Voir la démo en direct → showcase.naviplus.io](http://showcase.naviplus.io/ "Voir la démo en direct → showcase.naviplus.io")

La plupart des boutiques n'utilisent qu'un seul type de menu — mais chaque type a un objectif différent. Combiner plusieurs types de menus offre une bien meilleure expérience à vos clients : une Tab Bar pour un accès instantané, un Slide Menu pour un catalogue complet, un Mega Menu pour la découverte sur bureau, un FAB pour le support — chacun faisant exactement une chose, parfaitement.

***

## 1. Tab Bar *(aussi appelé : Navigation inférieure, Barre de navigation)*

Une barre de navigation fixe — placeable en bas, en haut, à gauche ou à droite. Contient 4 à 5 sections avec icône + étiquette. Donne à votre boutique l'aspect d'une application native. Fonctionne sur mobile et bureau. Idéale comme navigation principale — toujours visible, toujours à portée de pouce. **Utilisée par :** Shein, ASOS, Zara, Etsy, H&M, Nike, Uniqlo.

**Plateforme :** Bureau et Mobile *(Extrêmement populaire)*

***

## 2. Slide Menu *(aussi appelé : Menu tiroir, Menu hamburger)*

Glisse depuis la gauche ou la droite — éléments illimités, aucune limite de taille, s'adapte à toute la structure du site. Le modèle de navigation le plus populaire et adapté au mobile. Peut être déclenché par une icône hamburger, un élément de Tab Bar, ou n'importe quel bouton. Utilisez-le pour votre catalogue complet, les liens de compte, les politiques, et tout ce qui n'a pas besoin d'être toujours visible.

**Plateforme :** Mobile *(Extrêmement populaire)* · Bureau *(moins courant)*

***

## 3. Desktop Mega Menu *(aussi appelé : Menu déroulant, Menu multi-colonnes)*

Un menu déroulant multi-colonnes qui se déploie sous l'en-tête — conçu pour couvrir toute la navigation bureau en un seul endroit. Tire pleinement parti du grand écran avec images, bannières et mises en page riches. Remplace entièrement le menu d'en-tête par défaut. La référence pour toute boutique avec un catalogue conséquent sur bureau.

**Plateforme :** Bureau uniquement *(Extrêmement populaire)*

***

## 4. Mobile Mega Menu *(aussi appelé : Menu d'en-tête, Menu horizontal défilant)*

Une barre de menu compacte juste sous l'en-tête — toujours visible, aucun appui nécessaire. Défile horizontalement ou se divise en onglets pour afficher instantanément vos principales catégories. L'un des modèles de navigation les plus reconnaissables sur les sites mobiles. Idéal pour les boutiques avec 4 à 8 catégories prioritaires accessibles en un appui.

**Plateforme :** Mobile uniquement *(Populaire)*

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2Froax2ECBLjO03CIm2ITb%2FSlide-0-1.webp?alt=media&#x26;token=3c6aae5c-e7fb-4399-bf86-ce46bbc2d21d" alt="Exemple de Mobile Mega Menu"><figcaption><p>Mobile Mega Menu — barre horizontale toujours visible juste sous l'en-tête</p></figcaption></figure>

***

## 5. FAB & Support *(aussi appelé : Bouton flottant, Speed Dial)*

Un bouton flottant fixé dans un coin — toujours visible quel que soit le défilement. Se développe en plusieurs actions : WhatsApp, Messenger, widgets de chat, hotline, ou tout lien personnalisé. Le premier endroit où les clients cherchent de l'aide. Peut se masquer automatiquement pour économiser l'espace écran.

**Plateforme :** Bureau *(populaire)* · Mobile *(Extrêmement populaire)*

***

## 6. Grid Menu *(aussi appelé : Menu icônes, Grille de catégories)*

Une grille de tuiles — chacune avec une icône ou image et une étiquette. S'intègre directement dans n'importe quelle section de page. Donne à votre boutique un aspect d'application. Idéal pour présenter 6 à 12 catégories en un coup d'œil. Fonctionne particulièrement bien sur les pages d'accueil, les pages de destination et à l'intérieur des Slide Menus.

**Plateforme :** Mobile *(Populaire)*

## Comment les combiner

Vous n'avez pas à choisir un seul type. Les boutiques les plus performantes utilisent plusieurs types de menus — chacun faisant un travail spécifique :

| Objectif | Recommandé |
|---|---|
| Navigation principale sur mobile | Tab Bar + Slide Menu |
| Navigation desktop complète | Desktop Mega Menu |
| Afficher les catégories principales sur mobile | Mobile Mega Menu |
| Support & contact permanent | FAB & Support |
| Navigation visuelle des catégories en page d'accueil | Grid Menu |

***

## Questions fréquentes

<details><summary>Je veux que mon menu fonctionne bien sur mobile. Quel type choisir ?</summary>
<p>Commencez par une <strong>Tab Bar</strong> ou un <strong>Slide Menu</strong> — les deux sont conçus pour le mobile. La Tab Bar donne un accès instantané à 4-5 sections principales ; le Slide Menu contient toute la structure du site sans limite d'éléments.</p>
</details>

<details><summary>Peut-on utiliser plusieurs types de menus simultanément ?</summary>
<p>Oui — et vous devriez. Une combinaison populaire : <strong>Tab Bar</strong> (navigation principale) + <strong>Slide Menu</strong> (structure complète du site) + <strong>FAB</strong> (WhatsApp / support). Chaque type a un objectif différent sans chevauchement.</p>
</details>
