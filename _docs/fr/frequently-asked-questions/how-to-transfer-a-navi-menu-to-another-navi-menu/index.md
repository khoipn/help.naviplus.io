# Comment transférer un menu Navi+ vers un autre menu Navi+

Avez-vous déjà créé un menu que vous aimiez vraiment… puis réalisé que vous en aviez besoin ailleurs ? 😅

Bonne nouvelle : vous n'avez pas à le reconstruire depuis le début.

### Quand auriez-vous besoin de transférer un menu ?

Ceci est couramment nécessaire dans les situations suivantes :

* Vous avez complété un menu sur mobile et souhaitez l'utiliser sur desktop (ou vice versa)
* Vous avez créé un menu pour un site web et souhaitez le réutiliser sur un autre site web, avec quelques ajustements pour mieux s'adapter au nouveau design

Actuellement, il n'existe aucun outil automatique qui peut transférer des menus entre les systèmes Navi+. Donc, la méthode la plus efficace pour l'instant est la classique mais fiable : copier & coller.

### Guide étape par étape

#### Étape 1 : Préparer le menu de destination

Tout d'abord, configurez la structure du menu de destination.

Créez le cadre de base avec des éléments de menu de Niveau 1 et de Niveau 2 afin qu'il soit prêt à recevoir le contenu transféré.

#### Étape 2 : Ouvrir les deux menus côte à côte

Ouvrez les deux menus dans le même navigateur (par exemple, Chrome) :

* Un onglet : Menu source (le menu qui a déjà du contenu)
* Un onglet : Menu de destination (le menu dans lequel vous transférez)

Les garder ouverts côte à côte rendra le processus plus rapide et aidera à éviter les erreurs.

#### Étape 3 : Copier depuis le menu source

Dans l'onglet du menu source, sélectionnez les éléments de menu que vous souhaitez déplacer et cliquez sur le bouton COPIER.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2Fz8sYKzjcUT1ouCcroiPu%2F1.2.webp?alt=media&#x26;token=d69221b6-a952-4c27-833c-3df74d9d4044" alt=""><figcaption></figcaption></figure>

#### Étape 4 : Coller dans le menu de destination

Passez à l'onglet du menu de destination, naviguez jusqu'à l'emplacement correct et collez les éléments de menu copiés.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2FLX3Be5eqfItKUMpXSCCj%2F1.3.webp?alt=media&#x26;token=74143e7e-efb7-485a-9eba-6a28d2ec9229" alt=""><figcaption></figcaption></figure>

🔁 Répétez les étapes 3 et 4 jusqu'à ce que l'ensemble de la structure du menu ait été entièrement transféré. Conseils : Déplacez le menu par sections (par niveau ou par branche) pour vous assurer que rien n'est oublié et que la structure reste organisée.
