# Qu'est-ce que Navi+ ? (Comment ça fonctionne)

Navi+ est un **constructeur de navigation sans code** — il vous permet de créer de beaux menus à forte conversion pour n'importe quel site Web sans toucher à votre thème ou écrire une seule ligne de code.

Si les visiteurs de votre site ont du mal à trouver ce qu'ils cherchent, votre menu pourrait les décevoir. Navi+ vous aide à corriger cela en quelques minutes.

## Cinq types de menus

Choisissez le bon menu pour votre cas d'utilisation :

| Type de menu | Meilleur pour |
| --- | --- |
| **Tab Bar** | Navigation mobile-first, ressemble à une application native |
| **Mega Menu** | Dropdowns riches avec images, colonnes et liens — a des versions Desktop et Mobile séparées |
| **Slide Menu** | Menus hamburger avec support multi-niveaux |
| **Grid Menu** | Navigation visuelle par catégorie — idéal pour le commerce électronique |
| **FAB** | Bouton flottant d'accès rapide, toujours à portée de main |

Vous pouvez utiliser plusieurs types de menus ensemble sur le même site.

## Comment ça fonctionne

Navi+ est un **widget** — il ne remplace ni ne modifie votre thème existant. Au lieu de cela, il injecte un script léger dans votre page qui rend le menu par-dessus votre mise en page existante.

Vous construisez et gérez tout depuis le **tableau de bord Navi+**. Les changements sont mis en ligne instantanément — pas besoin d'éditer du code, de modifier des modèles ou de republier votre thème.

Voici le flux :

1. Vous créez un menu dans le tableau de bord
2. Navi+ génère un petit script
3. Ce script est intégré dans votre site (via App Block sur Shopify, ou un extrait de code ailleurs)
4. Le menu se rend par-dessus votre mise en page existante à chaque chargement de page

## Fonctionne avec votre plateforme

Navi+ s'intègre dans les outils que vous utilisez déjà :

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · et tout site Web qui accepte une `<script>` balise.

## Ce que Navi+ n'est PAS

- **Pas un thème** — il fonctionne aux côtés de votre thème existant, pas à la place de celui-ci
- **Pas un code que vous installez manuellement** — un App Block (Shopify) ou un extrait intégré suffit
- **Pas un plugin qui peut casser votre site** — si Navi+ est désactivé, le menu de votre thème d'origine revient automatiquement

## Conçu pour la vitesse

Les ressources de Navi+ sont livrées via Cloudflare CDN à ~52ms de temps de chargement moyen — généralement plus rapide que les propres scripts de votre plateforme. Après le premier chargement, les menus sont mis en cache et se chargent instantanément.
