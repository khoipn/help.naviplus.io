# Qu'est-ce que Navi+ ? (Comment ça fonctionne)

Navi+ est un **constructeur de navigation sans code** — il vous permet de créer de beaux menus à fort taux de conversion pour n'importe quel site web sans toucher à votre thème ni écrire une seule ligne de code.

Si les visiteurs de votre site ont du mal à trouver ce qu'ils cherchent, votre menu les laisse peut-être tomber. Navi+ vous aide à résoudre ça en quelques minutes.

## Cinq types de menus

Choisissez le bon menu pour votre cas d'usage :

| Type de menu | Idéal pour |
| --- | --- |
| **Tab Bar** | Navigation mobile-first, ressemble à une appli native |
| **Mega Menu** | Menus déroulants riches avec images, colonnes & liens — versions Desktop et Mobile séparées |
| **Slide Menu** | Menus hamburger avec support multi-niveaux |
| **Grid Menu** | Navigation visuelle des catégories — idéal pour l'e-commerce |
| **FAB** | Bouton flottant d'accès rapide, toujours à portée |

Vous pouvez utiliser plusieurs types de menus ensemble sur le même site.

## Comment ça fonctionne

Navi+ est un **widget** — il ne remplace ni ne modifie votre thème existant. Il injecte plutôt un script léger dans votre page qui affiche le menu par-dessus votre mise en page existante.

Vous construisez et gérez tout depuis le **tableau de bord Navi+**. Les modifications sont publiées instantanément — pas besoin de modifier du code, de changer des modèles ou de republier votre thème.

Voici le processus :

1. Vous créez un menu dans le tableau de bord
2. Navi+ génère un petit script
3. Ce script est intégré à votre site (via App Block sur Shopify, ou un extrait de code ailleurs)
4. Le menu s'affiche par-dessus votre mise en page existante à chaque chargement de page

## Plateformes compatibles

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · et tout site qui accepte une balise `<script>`.

## Ce que Navi+ N'EST PAS

- **Pas un thème** — il fonctionne à côté de votre thème existant, pas à sa place
- **Pas du code à installer manuellement** — un App Block (Shopify) ou un snippet suffit
- **Pas un plugin qui peut casser votre site** — si Navi+ est désactivé, le menu de votre thème d'origine revient automatiquement

## Conçu pour la vitesse

Les assets Navi+ sont livrés via le CDN Cloudflare avec ~52ms de temps de chargement moyen — généralement plus rapide que les scripts propres à votre plateforme.
