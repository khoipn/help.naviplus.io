# What is Navi+? (How it works)

Navi+ is a **no-code navigation builder** — it lets you create beautiful, high-converting menus for any website without touching your theme or writing a single line of code.

If visitors to your site are struggling to find what they're looking for, your menu might be letting them down. Navi+ helps you fix that in minutes.

## Five menu types

Choose the right menu for your use case:

| Menu Type | Best For |
| --- | --- |
| **Tab Bar** | Mobile-first navigation, feels like a native app |
| **Mega Menu** | Rich dropdowns with images, columns & links — has separate Desktop and Mobile versions |
| **Slide Menu** | Hamburger menus with multi-level support |
| **Grid Menu** | Visual category browsing — great for e-commerce |
| **FAB** | Quick-access floating button, always within reach |

You can use multiple menu types together on the same site.

## How it works

Navi+ is a **widget** — it doesn't replace or modify your existing theme. Instead, it injects a lightweight script into your page that renders the menu on top of your existing layout.

You build and manage everything from the **Navi+ dashboard**. Changes go live instantly — no need to edit code, modify templates, or republish your theme.

Here's the flow:

1. You create a menu in the dashboard
2. Navi+ generates a small script
3. That script is embedded in your site (via App Block on Shopify, or a code snippet elsewhere)
4. The menu renders on top of your existing layout on every page load

## Works with your platform

Navi+ fits into the tools you already use:

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · and any website that accepts a `<script>` tag.

## What Navi+ is NOT

- **Not a theme** — it works alongside your existing theme, not instead of it
- **Not code you install manually** — one App Block (Shopify) or one embed snippet is all it takes
- **Not a plugin that can break your site** — if Navi+ is disabled, your original theme menu comes back automatically

## Built for speed

Navi+ assets are delivered via Cloudflare CDN at ~52ms average load time — typically faster than your platform's own scripts. After the first load, menus are cached and load instantly.
