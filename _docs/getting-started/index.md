# Getting Started

Navi+ \| Website Menu Builder is a no-code website navigation builder that helps you guide visitors to the right place — faster.

Create modern menus like **Tabbar, Mega Menu, Slide (Hamburger), Grid Menu, and Floating Button** in just minutes. Simply drag & drop. No coding required.

Navi+ works seamlessly with [Shopify](https://www.shopify.com/ "Shopify"), [Wix](https://www.wix.com/ "Wix"), [WordPress](https://wordpress.org/ "WordPress"), [WooCommerce](https://woocommerce.com/ "WooCommerce"), [Webflow](https://webflow.com/ "Webflow"), [Squarespace](https://www.squarespace.com/ "Squarespace"), and any website via embed code.

It’s cloud-based, lightning-fast, and has no impact on page speed or SEO.

Start free. Upgrade when you need more control and scale.

Design smarter navigation. Improve user experience. Boost conversions.
