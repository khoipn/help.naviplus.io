# Publish

The **Publish** section controls how your menu appears on your live website: toggling the menu on/off, choosing the embed method, configuring device visibility, and filtering by URL. This is the final step after designing — without saving Publish settings, the menu will not appear.

## Quick Reference

```
Menu type           Primary method
────────────────────────────────────────────────────
Sticky / TABBAR     App Embeds (Shopify) / <head> (Global) + enable toggle
Sticky / FAB        App Embeds (Shopify) / <head> (Global) + enable toggle
Slide (CONTEXT)     App Embeds / <head> + enable toggle + configure trigger
Section (Mega/Grid) Insert/Replace (CSS Selector) or App Block (Shopify)
```

**Rule:** Enable toggle = menu is live. Disable toggle = completely hidden from website, no need to remove embed code.
