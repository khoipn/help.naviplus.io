# Shopify: Mở menu hamburger hoặc giỏ hàng, bảng tìm kiếm

Hiện tại, Navi+ hỗ trợ danh sách các theme phổ biến để mở menu Hamburger, giỏ hàng và bảng tìm kiếm. Với các theme không có trong danh sách này, bạn có thể yêu cầu Navi+ thêm hỗ trợ (thời gian xử lý: 1 ngày làm việc).

**Danh sách theme được hỗ trợ:** Dawn, Spotlight, Sense, Refresh, Craft, Impulse, Tailor, Symmetry, District, Warehouse, Veena, Hongo, Shark, Honey, Focal, Impact, Broadcast, Expanse, ShowTime, Pipeline, Empire, Enterprise, Local, Avenue, Parallax, Palo Alto, Minion, Borders, Xclusive, Prestige...
