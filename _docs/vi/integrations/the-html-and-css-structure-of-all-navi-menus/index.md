# Cấu trúc HTML và CSS của tất cả các menu Navi+

Dưới đây là cấu trúc HTML và CSS cho tất cả các menu Navi+, sẽ giữ nguyên trong suốt vòng đời sản phẩm. Mặc dù có thể có những biến thể nhỏ cho các loại menu cụ thể, cấu trúc khung chính dưới đây sẽ không thay đổi. Bạn có thể tham khảo cấu trúc này để viết mã CSS tùy chỉnh cho thiết kế menu nâng cao.
