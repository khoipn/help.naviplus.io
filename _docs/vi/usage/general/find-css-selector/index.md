# Tìm CSS Selector

**CSS Selector** là một khái niệm rất thú vị trong CSS — nó xác định phần nào của trang web của bạn có thể được **thay thế** hoặc **có một menu Navi+ được chèn vào** (có thể *trước* hoặc *sau* nó).

Dưới đây là một số **ví dụ về CSS Selectors** bạn có thể sử dụng:

```css
#header           /* Khu vực tiêu đề */
.navbar           /* Thanh điều hướng chính */
.footer           /* Phần chân trang */
.main-content     /* Container nội dung chính */
.product-gallery  /* Phần hình ảnh sản phẩm */
div#menu-wrapper  /* Một div cụ thể với ID “menu-wrapper” */
section.hero      /* Phần hero ở đầu trang */
```

Theo dõi các hướng dẫn bên dưới để **tìm CSS Selector chính xác** trên trang web của bạn.
