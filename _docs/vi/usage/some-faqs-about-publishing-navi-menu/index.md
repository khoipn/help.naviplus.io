# Một số câu hỏi thường gặp về việc xuất bản menu Navi+

**Tại sao Navi+ không hiển thị trên trang của tôi?**

Xác minh rằng bạn đã hoàn thành tất cả các bước. Đối với Tabbar/FAB, bạn cần: 1) Nhúng ứng dụng Navi+ vào giao diện của bạn, 2) Bật công tắc xuất bản menu.

(Cập nhật 12/30/2025)
