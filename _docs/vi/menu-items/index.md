# Menu Items

Một **menu item** là một dòng trong danh sách menu của bạn. Item là thành phần cơ bản của mọi menu Navi+ — mỗi item có thể là một link điều hướng, một tiêu đề nhóm, một khoảng trắng, hoặc một tab phân vùng.

## Tham chiếu nhanh

**Tôi chỉnh sửa item ở đâu?**  
Click vào bất kỳ item nào trong cây menu → panel Edit Item mở ra bên phải. Nhấn **Update** để áp dụng thay đổi, rồi **Save** (`Ctrl+S` / `Cmd+S`) để xuất bản.

**Thứ tự field trong Edit Item:**  
Media (icon / image) → Basic Info (text, link, description, badge, divider) → Layout → Interface (màu sắc, khoảng cách) → Advance (animation, CSS) → Publish (hiển thị)
