# Bắt đầu

Chào mừng bạn đến với **Navi+** — chúng tôi rất vui vì bạn ở đây! 🎉

Nếu khách truy cập vào trang web của bạn đang gặp khó khăn trong việc tìm kiếm những gì họ đang tìm, menu của bạn có thể đang khiến họ thất vọng. Navi+ giúp bạn khắc phục điều đó — mà không cần viết một dòng mã nào.

## Navi+ là gì?

Navi+ là một **trình tạo điều hướng không mã** giúp bạn dễ dàng tạo ra những menu đẹp mắt, có tỷ lệ chuyển đổi cao cho bất kỳ trang web nào. Dù bạn điều hành một cửa hàng trực tuyến hay một trang web kinh doanh, bạn có thể thiết lập một điều hướng chuyên nghiệp trong **chỉ vài phút**.

Chọn từ năm loại menu mạnh mẽ để phù hợp với nhu cầu của bạn:

| Loại Menu          | Tốt nhất cho                                     |
| ------------------- | ------------------------------------------------ |
| **Tab Bar**         | Điều hướng ưu tiên di động, cảm giác như một ứng dụng gốc |
| **Mega Menu**       | Dropdown phong phú với hình ảnh, cột & liên kết   |
| **Slide Menu**      | Menu hamburger mượt mà với hỗ trợ đa cấp        |
| **Grid Menu**       | Duyệt danh mục trực quan — tuyệt vời cho thương mại điện tử |
| **Floating Button** | Hành động truy cập nhanh luôn trong tầm tay      |
