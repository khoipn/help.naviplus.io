# Was ist Navi+?

Navi+ ist ein **no-code navigation builder** — es ermöglicht Ihnen, schöne, hochkonvertierende Menüs für jede Website zu erstellen, ohne Ihr Thema zu berühren oder eine einzige Zeile Code zu schreiben.

Wenn Besucher Ihrer Seite Schwierigkeiten haben, das zu finden, wonach sie suchen, könnte Ihr Menü sie enttäuschen. Navi+ hilft Ihnen, das in wenigen Minuten zu beheben.

## Fünf Menütypen

Wählen Sie das richtige Menü für Ihren Anwendungsfall:

| Menüt Typ | Am besten geeignet für |
| --- | --- |
| **Tab Bar** | Mobile-first Navigation, fühlt sich wie eine native App an |
| **Mega Menu** | Reichhaltige Dropdowns mit Bildern, Spalten & Links — hat separate Desktop- und Mobile-Versionen |
| **Slide Menu** | Hamburger-Menüs mit Unterstützung für mehrere Ebenen |
| **Grid Menu** | Visuelles Kategoriebrowsing — großartig für E-Commerce |
| **FAB** | Schnellzugriffs-Schwebetaste, immer in Reichweite |

Sie können mehrere Menütypen zusammen auf derselben Seite verwenden.

## Wie es funktioniert

Navi+ ist ein **widget** — es ersetzt oder ändert Ihr bestehendes Thema nicht. Stattdessen injiziert es ein leichtes Skript in Ihre Seite, das das Menü über Ihr bestehendes Layout rendert.

Sie erstellen und verwalten alles über das **Navi+ dashboard**. Änderungen werden sofort live — keine Notwendigkeit, Code zu bearbeiten, Vorlagen zu ändern oder Ihr Thema neu zu veröffentlichen.

Hier ist der Ablauf:

1. Sie erstellen ein Menü im Dashboard
2. Navi+ generiert ein kleines Skript
3. Dieses Skript wird in Ihre Seite eingebettet (über App Block auf Shopify oder einen Code-Schnipsel an anderer Stelle)
4. Das Menü wird bei jedem Seitenaufruf über Ihrem bestehenden Layout gerendert

## Funktioniert mit Ihrer Plattform

Navi+ passt in die Tools, die Sie bereits verwenden:

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · und jede Website, die ein `<script>`-Tag akzeptiert.

## Was Navi+ NICHT ist

- **Kein Thema** — es funktioniert neben Ihrem bestehenden Thema, nicht anstelle davon
- **Kein Code, den Sie manuell installieren** — ein App Block (Shopify) oder ein Embed-Schnipsel ist alles, was es braucht
- **Kein Plugin, das Ihre Seite beschädigen kann** — wenn Navi+ deaktiviert ist, wird Ihr ursprüngliches Themenmenü automatisch wiederhergestellt

## Für Geschwindigkeit gebaut

Navi+ Assets werden über Cloudflare CDN mit einer durchschnittlichen Ladezeit von ~52ms bereitgestellt — typischerweise schneller als die eigenen Skripte Ihrer Plattform. Nach dem ersten Laden werden Menüs zwischengespeichert und laden sofort.
