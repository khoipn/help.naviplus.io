# Was ist Navi+? (Wie es funktioniert)

Navi+ ist ein **No-Code-Navigations-Builder** — er ermöglicht es Ihnen, schöne, hochkonvertierende Menüs für jede Website zu erstellen, ohne Ihr Theme zu berühren oder eine einzige Zeile Code zu schreiben.

Wenn Besucher Ihrer Website Schwierigkeiten haben, das Gesuchte zu finden, könnte Ihr Menü sie im Stich lassen. Navi+ hilft Ihnen, das in Minuten zu beheben.

## Fünf Menütypen

Wählen Sie das richtige Menü für Ihren Anwendungsfall:

| Menütyp | Am besten für |
| --- | --- |
| **Tab Bar** | Mobile-first-Navigation, fühlt sich wie eine native App an |
| **Mega Menu** | Umfangreiche Dropdowns mit Bildern, Spalten & Links — hat separate Desktop- und Mobile-Versionen |
| **Slide Menu** | Hamburger-Menüs mit Unterstützung mehrerer Ebenen |
| **Grid Menu** | Visuelle Kategorienavigation — ideal für E-Commerce |
| **FAB** | Schnellzugriff-Schwebeschaltfläche, immer in Reichweite |

Sie können mehrere Menütypen gleichzeitig auf derselben Website verwenden.

## Wie es funktioniert

Navi+ ist ein **Widget** — es ersetzt oder verändert Ihr bestehendes Theme nicht. Stattdessen injiziert es ein leichtgewichtiges Skript in Ihre Seite, das das Menü auf Ihrem bestehenden Layout rendert.

Sie erstellen und verwalten alles über das **Navi+-Dashboard**. Änderungen werden sofort live — kein Bearbeiten von Code, Ändern von Vorlagen oder erneutes Veröffentlichen Ihres Themes erforderlich.

Ablauf:

1. Sie erstellen ein Menü im Dashboard
2. Navi+ generiert ein kleines Skript
3. Dieses Skript wird in Ihre Website eingebettet (über App Block auf Shopify oder einen Code-Snippet anderswo)
4. Das Menü wird bei jedem Seitenaufruf über Ihrem bestehenden Layout gerendert

## Kompatible Plattformen

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · und jede Website, die einen `<script>`-Tag akzeptiert.

## Was Navi+ NICHT ist

- **Kein Theme** — es arbeitet neben Ihrem bestehenden Theme, nicht anstelle davon
- **Kein Code, den Sie manuell installieren** — ein App Block (Shopify) oder ein Embed-Snippet reicht aus
- **Kein Plugin, das Ihre Website beschädigen kann** — wenn Navi+ deaktiviert wird, kommt Ihr ursprüngliches Theme-Menü automatisch zurück

## Für Geschwindigkeit entwickelt

Navi+-Assets werden über Cloudflare CDN mit ~52ms durchschnittlicher Ladezeit geliefert — typischerweise schneller als die eigenen Skripte Ihrer Plattform.
