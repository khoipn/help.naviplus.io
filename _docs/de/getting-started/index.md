# Loslegen

Willkommen bei **Navi+** — wir freuen uns, dass Sie hier sind! 🎉

Wenn Besucher Ihrer Website Schwierigkeiten haben, das zu finden, wonach sie suchen, könnte Ihr Menü sie im Stich lassen. Navi+ hilft Ihnen, das zu beheben — ohne eine einzige Zeile Code zu schreiben.

## Was ist Navi+?

Navi+ ist ein **no-code Navigation Builder**, der es einfach macht, schöne, hochkonvertierende Menüs für jede Website zu erstellen. Egal, ob Sie einen Online-Shop oder eine Unternehmenswebsite betreiben, Sie können in **nur wenigen Minuten** eine professionelle Navigation einrichten.

Wählen Sie aus fünf leistungsstarken Menütpyen, die Ihren Bedürfnissen entsprechen:

| Menütpy           | Am besten geeignet für                                         |
| ------------------- | ------------------------------------------------ |
| **Tabbar**         | Mobile-first Navigation, fühlt sich an wie eine native App |
| **Mega menu**       | Reichhaltige Dropdowns mit Bildern, Spalten & Links      |
| **Slide menu**      | Sanfte Hamburger-Menüs mit Unterstützung für mehrere Ebenen  |
| **Grid menu**       | Visuelles Kategoriebrowsing — großartig für E-Commerce  |
| **Floating Button** | Schneller Zugriff auf Aktionen, die immer in Reichweite sind         |

## Funktioniert mit Ihrer Plattform

Navi+ passt perfekt in die Tools, die Sie bereits verwenden:

[Shopify](https://www.shopify.com/ "Shopify") · [Wix](https://www.wix.com/ "Wix") · [WordPress](https://wordpress.org/ "WordPress") · [WooCommerce](https://woocommerce.com/ "WooCommerce") · [Webflow](https://webflow.com/ "Webflow") · [Squarespace](https://www.squarespace.com/ "Squarespace") · und **jede Website** über Embed-Code.
