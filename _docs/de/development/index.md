## Kurzreferenz

- Verwenden Sie **Benutzerdefiniertes CSS für dieses Menü** für menüspezifische Stile
- Verwenden Sie **Reusable Classes (CSS)** für gemeinsame Stile über Menüs hinweg
- CSS zielt auf drei Ebenen: Hauptleiste → Untermenü → verschachteltes Untermenü
- JavaScript-Bibliotheken (z. B. TourGuideJS) können über Menüelement-Attribute eingebettet werden
