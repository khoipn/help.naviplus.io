# Veröffentlichen

Der Abschnitt **Veröffentlichen** steuert, wie dein Menü auf deiner Live-Website erscheint: Menü ein-/ausschalten, Einbettungsmethode wählen, Gerätesichtbarkeit konfigurieren und nach URL filtern. Dies ist der letzte Schritt nach dem Design — ohne gespeicherte Publish-Einstellungen wird das Menü nicht angezeigt.

## Kurzübersicht

```
Menütyp              Primäre Methode
────────────────────────────────────────────────────
Sticky / TABBAR      App Embeds (Shopify) / <head> (Global) + Aktivierungsschalter
Sticky / FAB         App Embeds (Shopify) / <head> (Global) + Aktivierungsschalter
Slide (CONTEXT)      App Embeds / <head> + Aktivierungsschalter + Trigger konfigurieren
Section (Mega/Grid)  Einfügen/Ersetzen (CSS Selector) oder App Block (Shopify)
```

**Regel:** Schalter aktivieren = Menü ist live. Schalter deaktivieren = vollständig von der Website ausgeblendet, kein Entfernen des Einbettungscodes nötig.
