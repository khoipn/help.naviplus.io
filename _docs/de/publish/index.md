# Veröffentlichen

Der **Veröffentlichen**-Bereich steuert, wie Ihr Menü auf Ihrer Live-Website erscheint: Menü ein/aus schalten, Einbettungsmethode wählen, Geräte-Sichtbarkeit konfigurieren und nach URL filtern. Dies ist der letzte Schritt nach dem Design — ohne Speichern der Veröffentlichen-Einstellungen wird das Menü nicht angezeigt.

## Schnellreferenz

```
Menütyp           Primäre Methode
────────────────────────────────────────────────────
Sticky / TABBAR     App-Einbettungen (Shopify) / <head> (Global) + aktivieren
Sticky / FAB        App-Einbettungen (Shopify) / <head> (Global) + aktivieren
Slide (CONTEXT)     App-Einbettungen / <head> + aktivieren + Trigger konfigurieren
Abschnitt (Mega/Grid) Einfügen/Ersetzen (CSS-Selektor) oder App-Block (Shopify)
```

**Regel:** Aktivieren = Menü ist live. Deaktivieren = vollständig von der Website verborgen, kein Bedarf, den Einbettungscode zu entfernen.
