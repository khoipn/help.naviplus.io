# Die HTML- und CSS-Struktur aller Navi+ Menüs

Unten finden Sie die HTML- und CSS-Struktur für alle Navi+ Menüs, die während des gesamten Produktlebenszyklus stabil bleiben wird. Während geringfügige Variationen für spezifische Menütypen existieren können, wird sich die grundlegende Rahmenstruktur unten nicht ändern. Sie können diese Struktur verwenden, um benutzerdefinierten CSS-Code für ein fortgeschrittenes Menüdesign zu schreiben.
