# Häufige erste Fragen

### Allgemein

#### Was ist Navi+?

Navi+ ist ein Menü-Builder, der auf jeder Website funktioniert – Shopify, WordPress, WooCommerce, Wix, Webflow und mehr. Es ermöglicht Ihnen, Tab Bars, Mega Menüs, Slide Menüs, Grid Menüs und FABs hinzuzufügen, ohne Code zu schreiben.

#### Welche Plattformen unterstützt Navi+?

Jede Website. Für Shopify verwenden Sie die dedizierte App. Für alles andere melden Sie sich unter [dash.naviplus.app](https://dash.naviplus.app/ "dash.naviplus.app") an und betten Sie es mit einem Code-Snippet ein.

#### Wie installiere ich Navi+?

* **Shopify:** Installieren Sie aus dem [Shopify App Store](https://apps.shopify.com/pronavi-navigation-design "Shopify App Store")
* **Andere Plattformen:** Melden Sie sich unter [dash.naviplus.app](https://dash.naviplus.app/ "dash.naviplus.app") an, erklären Sie Ihre Domain und betten Sie dann das Snippet ein.

#### Kann ich Navi+ ausprobieren, bevor ich es einrichte?

Ja. Besuchen Sie [showcase.naviplus.io](https://showcase.naviplus.io/ "showcase.naviplus.io"), um alle Menütypen in einem echten Shop zu sehen – mobil, Tablet und Desktop.

#### Wird Navi+ meine Website verlangsamen?

Nein. Assets werden über Cloudflare CDN mit einer durchschnittlichen Ladezeit von 52ms bereitgestellt – typischerweise schneller als die eigenen Skripte von Shopify. Nach dem ersten Laden werden Menüs zwischengespeichert und laden sofort.

#### Wie viele Menütypen bietet Navi+ an?

Sechs: Tab Bar, Slide Menu, FAB, Grid Menu, Mega Menu (mobil) und Mega Menu (Desktop). Sie können mehrere Typen zusammen auf derselben Seite verwenden.

#### Wie viele Menüs kann ich erstellen?

* **Starter (Kostenlos):** 1 Menü pro Domain
* **Business:** 4 Menüs pro Domain
* **Elite:** Unbegrenzt

#### Wie viel kostet Navi+?

Starten Sie kostenlos mit dem Starter-Plan. Business- und Elite-Pläne schalten erweiterte Funktionen frei. Siehe vollständige Preisgestaltung unter [naviplus.io/pricing](https://naviplus.io/pricing/ "naviplus.io/pricing").

#### Gibt es eine kostenlose Testversion?

Ja – 7 Tage kostenlos, keine Gebühren. Jederzeit kündbar.

### Design & Anpassung

#### Kann ich Farben, Schriftarten und Abstände anpassen?

Ja. Sie haben die volle Kontrolle über Icons (3000+ Optionen), Schriftarten (1500+ Google Fonts oder benutzerdefiniert), Farben, Polsterung und Abstände auf jeder Ebene.

#### Kann ich benutzerdefiniertes CSS hinzufügen?

Ja. Fügen Sie CSS hinzu, das auf das Menü oder die gesamte Website beschränkt ist, direkt in Navi+. Sie können auch benutzerdefinierte Klassen oder IDs für jedes Element zuweisen.

#### Kann ich eine Vorschau anzeigen, bevor ich veröffentliche?

Ja. Verwenden Sie den integrierten Simulator und veröffentlichen Sie dann sicher mit URL-basierten Anzeige-Regeln, um zuerst auf einer bestimmten Seite zu testen.

#### Kann das Menü nur auf bestimmten Seiten angezeigt werden?

Ja. Steuern Sie die Sichtbarkeit nach Seiten-URL, Gerätetyp (mobil/Desktop) oder Anmeldestatus.

***

### Kompatibilität & Integrationen

#### Wird Navi+ mit meinem Theme oder anderen Apps in Konflikt stehen?

Selten. Navi+ ändert Ihr Theme nicht und benötigt keine Berechtigungen. Wenn Menüs mit anderen fixierten Elementen überlappen, passen Sie den z-index an – [siehe Anleitung](https://help.naviplus.io/docs/usage/general/z-index/ "siehe Anleitung").

#### Kann Navi+ andere App-Widgets in einem Menü konsolidieren?

Ja. Chat-Widgets, Treue-Apps, Support-Buttons und andere schwebende Icons können alle in sauberen Navi+ Menüelementen gruppiert werden.

#### Unterstützt Navi+ mehrere Sprachen?

Vollständig unterstützt auf Shopify. Bald verfügbar für andere Plattformen.

#### Kann ich Menüs exportieren und importieren?

Ja. Menüs werden als JSON gespeichert – Sie können die Datei exportieren, manuell oder mit einem KI-Tool bearbeiten und dann wieder importieren. Dies funktioniert auch zum Sichern oder Übertragen von Menüs zwischen Websites.
