# Menüelemente

Ein **Menüelement** ist eine einzelne Zeile in Ihrer Menüliste. Elemente sind die Bausteine jedes Navi+ Menüs — jedes kann ein Navigationslink, eine Gruppenüberschrift, ein leerer Platzhalter oder ein Tab-Trenner sein.

## Schnellreferenz

**Wo bearbeite ich ein Element?**  
Klicken Sie auf ein beliebiges Element im Menübaum → das Bearbeiten-Panel für das Element öffnet sich rechts. Drücken Sie **Aktualisieren**, um Änderungen anzuwenden, und dann **Speichern** (`Ctrl+S` / `Cmd+S`), um zu veröffentlichen.

**Feldreihenfolge im Bearbeiten-Panel:**  
Medien (Symbol / Bild) → Grundinformationen (Text, Link, Beschreibung, Abzeichen, Trenner) → Layout → Schnittstelle (Farben, Abstände) → Erweitert (Animation, CSS) → Veröffentlichen (Sichtbarkeit)
