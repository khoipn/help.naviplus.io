# Menüpunkte

Ein **Menüpunkt** ist eine einzelne Zeile in Ihrer Menüliste. Punkte sind die Bausteine jedes Navi+-Menüs — jeder kann ein Navigationslink, ein Gruppenheader, ein leerer Abstandshalter oder ein Tab-Trenner sein.

## Schnellübersicht

**Wo bearbeite ich einen Punkt?**  
Klicken Sie auf einen beliebigen Punkt im Menübaum → das Panel "Punkt bearbeiten" öffnet sich rechts. Drücken Sie **Aktualisieren**, um Änderungen anzuwenden, dann **Speichern** (`Ctrl+S` / `Cmd+S`) zum Veröffentlichen.

**Feldreihenfolge in "Punkt bearbeiten":**  
Medien (Icon / Bild) → Grundinfo (Text, Link, Beschreibung, Badge, Trennlinie) → Layout → Oberfläche (Farben, Abstände) → Erweitert (Animation, CSS) → Veröffentlichen (Sichtbarkeit)
