# Wie man ein Navi+ Menü in ein anderes Navi+ Menü überträgt

Haben Sie jemals ein Menü erstellt, das Ihnen wirklich gefiel… und dann festgestellt, dass Sie dasselbe woanders benötigten? 😅

Gute Nachrichten: Sie müssen es nicht von Grund auf neu erstellen.

### Wann müssten Sie ein Menü übertragen?

Dies ist häufig in den folgenden Situationen erforderlich:

* Sie haben ein Menü auf dem Handy fertiggestellt und möchten es auf dem Desktop verwenden (oder umgekehrt)
* Sie haben ein Menü für eine Website erstellt und möchten es auf einer anderen Website wiederverwenden, mit einigen Anpassungen, um besser zum neuen Design zu passen

Derzeit gibt es kein automatisches Tool, das Menüs zwischen Navi+ Systemen übertragen kann. Daher ist die effektivste Methode vorerst die klassische, aber zuverlässige: kopieren & einfügen.

### Schritt-für-Schritt-Anleitung

#### Schritt 1: Bereiten Sie das Zielmenü vor

Zuerst richten Sie die Struktur des Zielmenüs ein.

Erstellen Sie das Grundgerüst mit Level 1 und Level 2 Menüelementen, damit es bereit ist, den übertragenen Inhalt zu empfangen.

#### Schritt 2: Öffnen Sie beide Menüs nebeneinander

Öffnen Sie beide Menüs im selben Browser (zum Beispiel Chrome):

* Ein Tab: Quellmenü (das Menü, das bereits Inhalte hat)
* Ein Tab: Zielmenü (das Menü, in das Sie übertragen)

Wenn Sie sie nebeneinander geöffnet lassen, wird der Prozess schneller und hilft, Fehler zu vermeiden.

#### Schritt 3: Kopieren Sie aus dem Quellmenü

Wählen Sie im Tab des Quellmenüs die Menüelemente aus, die Sie verschieben möchten, und klicken Sie auf die Schaltfläche KOPIEREN.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2Fz8sYKzjcUT1ouCcroiPu%2F1.2.webp?alt=media&#x26;token=d69221b6-a952-4c27-833c-3df74d9d4044" alt=""><figcaption></figcaption></figure>

#### Schritt 4: Fügen Sie in das Zielmenü ein

Wechseln Sie zum Tab des Zielmenüs, navigieren Sie zur richtigen Stelle und fügen Sie die kopierten Menüelemente ein.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2FLX3Be5eqfItKUMpXSCCj%2F1.3.webp?alt=media&#x26;token=74143e7e-efb7-485a-9eba-6a28d2ec9229" alt=""><figcaption></figcaption></figure>

🔁 Wiederholen Sie die Schritte 3 und 4, bis die gesamte Menüstruktur vollständig übertragen wurde. Tipps: Bewegen Sie das Menü in Abschnitten (nach Ebene oder nach Zweig), um sicherzustellen, dass nichts übersehen wird und die Struktur organisiert bleibt.
