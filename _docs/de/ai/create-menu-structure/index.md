# Menüstruktur erstellen

Beschreibe das benötigte Menü und lass die KI ein fertiges Grundgerüst aufbauen — erhalte schnell einen Ausgangspunkt, anstatt jedes Element von Hand hinzuzufügen. Diese Funktion ist **kostenlos**.

Öffnen: Schaltfläche **AI** → Karte **Menüstruktur erstellen**.

---

## Bevor du beginnst — sichere dein Menü

> **⋮ Mehr → Backup / Restore** — sichere dein aktuelles Menü, bevor du eine neue Struktur erstellst, da das Ergebnis das bestehende Menü überschreibt.

---

## So verwendest du es

1. Öffne die Karte **Menüstruktur erstellen**.
2. (Optional) Wähle, zu welchen Ebenen die KI **automatisch Icons hinzufügen** soll:

   | Option | Gilt für |
   |---|---|
   | Ebene 1 | Hauptelemente (Ebene 1) |
   | Ebene 2 | Unterelemente der Ebene 2 |
   | Ebene 3 | Unterelemente der Ebene 3 |

3. Tippe deine Beschreibung in das Feld **Beschreibe deine Anfrage**, z. B. „Menü für einen Kosmetikshop: Startseite, Produkte (Lippenstift, Feuchtigkeitscreme, Maske), Aktionen, Kontakt."
4. (Optional) Klicke auf **Vorlage verwenden**, um einen Beispiel-Prompt für deinen Menütyp einzufügen, und bearbeite ihn dann.
5. Klicke auf **Menü erstellen**. Der Fortschritt wird im **Aktivitätsprotokoll** rechts angezeigt.
6. Die KI baut die Struktur und wendet sie auf den Editor an. Überprüfe, verfeinere und klicke dann auf **Speichern**.

> Klicke auf **Speichern** (das Disketten-Icon), um deinen Prompt zur Wiederverwendung beim nächsten Mal zu speichern.

---

## Nach dem Erstellen

Die KI baut nur das **Strukturgerüst** (Elemente, Hierarchie, Icons). Du solltest Folgendes überprüfen und hinzufügen:

- Links (URLs) für jedes Element
- Bilder und Badges, falls nötig
- Farben und Layout über die Bereiche **Design** und **Einstellungen**

---

## Beispielbeschreibungen

- Herrenmode-Shop: Startseite, Oberteile (T-Shirts, Hemden, Jacken), Hosen, Accessoires, Sale, Kontakt.
- Tab Bar mit 5 Elementen für eine Lieferdienst-App: Startseite, Suche, Warenkorb, Bestellungen, Konto.
- Slide-Menü für einen Blog: Neueste Beiträge, Kategorien (Technik, Reisen, Essen), Über uns, Kontakt.
- Mega Menu für einen Elektronikshop mit 4 Gruppen: Telefone, Laptops, Zubehör, Aktionen — jeweils 4–5 Unterelemente.

---

## Fallstudie — ein Mega Menu für einen Elektronikshop bauen

**Kontext:** Ein Elektronikshop benötigt ein Desktop-Mega-Menu mit mehreren Gruppen, jede Gruppe mit mehreren Unterelementen und Icons zum leichten Überblicken.

1. **Backup:** ⋮ Mehr → Backup / Restore.
2. Öffne die Karte **Menüstruktur erstellen**. Aktiviere **Ebene 1** und **Ebene 2**, damit die KI sowohl Gruppen als auch Unterelementen Icons hinzufügt.
3. Tippe die Beschreibung:

   ```
   Mega Menu für einen Elektronikshop, 4 Gruppen:
   - Telefone (iPhone, Samsung, Xiaomi, Telefonzubehör)
   - Laptops (Büro, Gaming, MacBook, Laptopzubehör)
   - Haushaltsgeräte (Wasserfilter, Saugroboter, Heißluftfritteusen)
   - Aktionen
   ```

4. Klicke auf **Menü erstellen**. Beobachte das **Aktivitätsprotokoll** rechts. Die KI baut ein Mega Menu mit 4 Gruppen, Unterelementen und Icons.
5. Nach dem Erstellen hinzufügen: weise jedem Element eine **URL** zu, füge der Gruppe Aktionen ein Banner-Bild hinzu und dann **Speichern**.

**Ergebnis:** Ein mehrstufiges Mega-Menu-Gerüst mit Icons sofort verfügbar — du musst nur Links anhängen und feinabstimmen, anstatt Dutzende Elemente von Hand zu erstellen.
