# Automatisches Design mit Navi+ AI

Lass Navi+ die gesamte KI-Arbeit übernehmen — beschreibe einfach das gewünschte Menü direkt im Editor. Keine externen Tools, kein Kopieren und Einfügen von JSON. *(Business / Elite)*

Öffnen: Schaltfläche **AI** → Karte **Automatisches Design mit Navi+ AI**.

---

## Worin unterscheidet es sich von „Automatisches Design mit deiner KI-App"?

| | Deine KI-App | Navi+ AI |
|---|---|---|
| KI-Tool | Du öffnest ChatGPT / Claude / Gemini | Navi+ ruft die KI für dich auf |
| JSON kopieren/einfügen | Ja, manuell | Nicht nötig |
| Schritte | 4 Schritte | 1 Schritt (Chat) |
| Plan | Free | Business / Elite |

---

## Bevor du beginnst — sichere dein Menü

> **⋮ Mehr → Backup / Restore** — sichere dein aktuelles Menü, bevor du das KI-Ergebnis anwendest.

---

## So verwendest du es

1. Öffne die Karte **Automatisches Design mit Navi+ AI**.
2. Tippe eine Beschreibung in das Chatfeld, z. B. „Erstelle eine Tab Bar mit 5 Elementen für einen Modeshop, in Schwarz und Weiß."
3. (Optional) Hänge mit der Schaltfläche **Hochladen** ein Referenzbild an, damit die KI dem Layout/den Farben folgt.
4. Klicke auf **Senden**. Die KI liefert ein Design und wendet es zur Vorschau auf den Editor an.
5. Chatte weiter, um zu verfeinern („ändere das Kontakt-Icon", „entferne das letzte Element")... oder klicke auf **Neuer Chat**, um von vorne zu beginnen.
6. Zufrieden? Klicke auf **Speichern**.

> Dein Chatverlauf wird gespeichert — klicke auf **Verlauf**, um frühere Design-Sitzungen erneut zu öffnen.

---

## Plan-Voraussetzung

Diese Funktion ist im Plan **Business / Elite** enthalten. In einem niedrigeren Plan siehst du beim Öffnen dieser Karte einen Upgrade-Hinweis.

---

## Beispiel-Prompts

- Erstelle eine Tab Bar mit 5 Elementen für einen Kosmetikshop, in pastellrosa Tönen.
- Füge jedem Element ein Icon hinzu und verwandle das letzte Element in eine auffällige Kontakt-Schaltfläche.
- Gestalte ein Mega Menu wie das Bild, das ich gerade gesendet habe. *(hänge mit der Schaltfläche Hochladen ein Bild an)*
- Entferne das Element Blog, führe Aktionen und Sale zu einem zusammen.

---

## Fallstudie — eine Kosmetik-Tab-Bar in 3 Chat-Runden bauen

**Kontext:** Ein neuer Kosmetikshop ohne Menü möchte eine Tab Bar, die zu seiner pastellrosa Marke passt.

1. **Sichere** zur Sicherheit zuerst (⋮ Mehr → Backup / Restore).
2. Öffne die Karte **Automatisches Design mit Navi+ AI**. Runde 1:

   ```
   Erstelle eine Tab Bar mit 5 Elementen für einen Kosmetikshop: Startseite, Produkte,
   Angebote, Beauty-Guide, Konto. Pastellrosa Töne.
   ```

   Die KI baut die Tab Bar und zeigt sie sofort in der Vorschau an.
3. Runde 2 — hänge mit der Schaltfläche **Hochladen** ein Beispiel-UI-Bild an und tippe dann:

   ```
   Passe den Eckenradius und die Icons an den Stil in diesem Bild an.
   ```

4. Runde 3 — eine kleine Anpassung:

   ```
   Verwandle das Element Konto in eine auffällige Kontakt-Schaltfläche mit einem Telefon-Icon.
   ```

5. Zufrieden, klicke auf **Speichern**. Wenn du das nächste Mal Änderungen brauchst, klicke auf **Verlauf**, um diese Sitzung erneut zu öffnen.

**Ergebnis:** Eine Tab Bar, die zur Marke passt, in 3 Chat-Runden fertiggestellt, ohne Navi+ zu verlassen.
