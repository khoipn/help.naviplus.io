# Eine benutzerdefinierte Schriftart anwenden

Navi+ bietet eine große Auswahl an Google Fonts und ermöglicht eine optimierte und bequeme Möglichkeit, Ihr Menü zu gestalten. Google Fonts sind schnell, gut lesbar und weit verbreitet, was sie zu einer ausgezeichneten Wahl für die meisten Designs macht. Wenn Sie jedoch eine benutzerdefinierte Schriftart verwenden möchten, die nicht in der Standardauswahl enthalten ist, können Sie sie manuell auf Ihr Navi+ Menü anwenden. So geht's:

Wenn Sie beispielsweise die **Honey Salt** Schriftart von [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") gekauft haben und sie in Ihrem Navi+ Menü verwenden möchten, fügen Sie einfach den folgenden CSS-Code in den Abschnitt **Internes Stylesheet / CSS** ein:

```
{
  font-family: "Honey Salt";
}
```
