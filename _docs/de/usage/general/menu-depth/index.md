# Menütiefe

Die standardmäßige Unterstützung für die Menütiefe des Navi+ Menüs beträgt 2 für das Starter-Paket (kostenlos) und wird auf 3 für das Business-Paket erhöht. In den meisten Fällen ist eine Tiefe von 2 ausreichend für gewöhnliche Bedürfnisse - Und auch einfach für die Benutzer, da eine große Tiefe des Menüs die Benutzer verwirren kann. In einigen Fällen, zum Beispiel, möchten Sie möglicherweise die gesamte Produktkatalogstruktur zum Navi+ Menü hinzufügen, in diesem Fall könnte eine Tiefe von 3 Ebenen eine perfekte Wahl sein, um es Ihren Besuchern zu erleichtern, auf Ihre Dienstleistungen und Produkte zuzugreifen.

Derzeit haben Sie mit dem Business-Paket bis zu 3 Menütiefen
