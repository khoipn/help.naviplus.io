# Wie benutzt man?

**Mobile Mega Menüs** sind auf Mobilgeräten nicht so verbreitet wie Tabbar oder Slide Menüs, da der Tippbereich eines Mega Menüs oft zu klein für die Finger der Benutzer ist. Wie auch immer, manchmal möchten Sie möglicherweise eine konsistente Erfahrung zwischen Desktop und Mobilgerät aufrechterhalten, indem Sie ein Mega Menü auf beiden Plattformen verwenden.

Mobile Mega Menüs haben auch Vorteile: Sie sind kompakt, unter dem Header positioniert und nehmen keinen Bildschirmplatz ein, da sie auf dem Bildschirm schweben. Als Abschnittsmenü scrollen sie mit der Seite.

**Hauptvorteile:**

* Funktioniert nahtlos sowohl auf Mobilgeräten als auch auf Desktop für eine konsistente Erfahrung
* Spart Platz und scrollt mit der Seite.
* Bietet mehrere Präsentationsstile mit **Bildern**, **Raster** und **mehrstufigen** Layouts für eine reichhaltige, professionelle Erfahrung

**FAQ:**

<details><summary><strong>1. Ich habe mich noch nicht entschieden, ob ich die kostenlose oder die kostenpflichtige Version verwenden soll. Kann ich die professionellen Menüvorlagen ausprobieren?</strong></summary>
<p>Absolut! Die Menügruppen sollen hauptsächlich Anfängern helfen, sich nicht von den erweiterten Funktionen in Navi+ überwältigt zu fühlen 😄. Der beste Weg ist, die professionellen Menüs auszuprobieren, um alle Funktionen und Layoutoptionen zu erkunden. Später können Sie einfach jedes Menü löschen, das Sie nicht benötigen oder als ungeeignet empfinden.</p>
</details>

<details><summary><strong>2. Wird das Erstellen eines Menüs hier dazu führen, dass es sofort auf meiner Website angezeigt wird?</strong></summary>
<p>Noch nicht. Sie müssen das richtige Menü <strong>veröffentlichen</strong>, damit es auf Ihrer Website erscheint. Keine Sorge – es ist einfach!</p>
</details>

<details><summary><strong>3. Was ist, wenn ich die falsche Vorlage wähle oder einen Fehler mache?</strong></summary>
<p>Kein Problem. Sie können es jederzeit löschen und ein neues erstellen. Wenn Sie ein Menü testen und eine andere Vorlage ausprobieren möchten, können Sie das aktuelle als Sicherungsdatei <strong>speichern</strong> und später wieder öffnen. Sie können sogar Teile zwischen Menüs <strong>kopieren und einfügen</strong> (indem Sie zwei Browser-Tabs öffnen), um Ihre vorherige Arbeit zu behalten. Fühlen Sie sich frei, zu experimentieren!</p>
</details>

<details><summary><strong>4. Soll ich nur ein Menü oder mehrere erstellen?</strong></summary>
<p>Sie sollten <strong>mehrere Menüs</strong> erstellen. Nur das <strong>aktive</strong> Menü wird auf Ihrer Seite angezeigt, aber Sie können viele Versionen vorbereiten – indem Sie verschiedene Vorlagen auswählen oder Menüs klonen. Menüs können so konfiguriert werden, dass sie je nach Seitentyp (Startseite, Produkt, Sammlung...), Gerät (Mobilgerät oder Desktop) oder URL-Schlüsselwörtern angezeigt oder ausgeblendet werden. Also legen Sie los und erstellen Sie Menüs, die zu jeder Situation passen!</p>
</details>

### Schritt 1: Bearbeiten Sie den Menübaum

#### 1.1. Verständnis des Menübaums

Der **Menübaum** ist der wichtigste Teil jedes Menüs – er definiert die Struktur.

Hier ist ein Beispiel für einen Menübaum eines mobilen Mega Menüs:

```
Menü
Startseite  
Katalog 
- All Mountain  
   - Cruise Flex  
   - Balance Ride
- Freestyle  
   - Park Board  
   - Jib Board  
- Freeride  
Blogs
Support  
Mehr 
```

Dies ist ein 3-stufiger Menübaum. Stufe 1 umfasst **Menü, Startseite**, **Katalog**, **Support, Blogs** und **Mehr**.

**All-Mountain** ist ein Menüpunkt der Stufe 2, mit einigen untergeordneten Punkten: **Cruise Flex** und **Balance Ride**.

Sie werden die meiste Zeit damit verbringen, diesen Menübaum für Ihre Website zu erstellen, um den Kunden zu helfen, die Inhalte zu finden, die sie suchen.
