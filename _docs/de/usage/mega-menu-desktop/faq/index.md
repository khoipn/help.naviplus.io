# FAQ

#### Frage 1: Warum zeigt Navi+ das Desktop Mega Menu nicht automatisch auf mobilen Geräten an?

Das Surferlebnis auf Desktop und Mobilgeräten ist grundlegend unterschiedlich — insbesondere darin, wie Benutzer mit einer Website navigieren und interagieren.

Auf dem Desktop haben Benutzer mehr Bildschirmfläche und können große, komplexe Mega Menüs, die unter dem Header angezeigt werden, leicht einsehen.

Auf mobilen Geräten ist der Bildschirm jedoch kleiner und Benutzer navigieren hauptsächlich mit ihren Daumen, sodass das Anzeigen des gleichen Mega Menüs die Benutzeroberfläche überladen, schwer zu tippen und langsamer laden würde.

Deshalb konvertiert Navi+ nicht automatisch ein Desktop Mega Menu in eine mobile Version.

Stattdessen sollten Sie ein spezielles mobiles Menü erstellen, das ein besseres Benutzererlebnis bietet:

* Hauptmenüs — platzieren Sie sie in einer TabBar am unteren Bildschirmrand, wo sie leicht mit dem Daumen zu erreichen sind.
* Zusätzliche Menüs — verschieben Sie sie in ein Slide (Hamburger) Menü (ausziehbares Panel) oder ein mobiles Mega Menu.

👉 Mit der praktischen Copy & Paste Menüfunktion von Navi+ können Sie Ihr Desktop Mega Menu duplizieren und in nur wenigen Minuten für mobile Geräte anpassen — so haben Sie die volle Kontrolle und stellen sicher, dass beide Versionen perfekt für ihre Plattformen optimiert sind.
