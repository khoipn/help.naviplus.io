# Einige FAQs zur Veröffentlichung des Navi+ Menüs

**Warum wird Navi+ auf meiner Seite nicht angezeigt?**

Überprüfen Sie, ob Sie alle Schritte abgeschlossen haben. Für Tabbar/FAB benötigen Sie: 1) Embed Navi+ app in Ihr Theme, 2) Aktivieren Sie den Publish-Menü-Umschalter.
