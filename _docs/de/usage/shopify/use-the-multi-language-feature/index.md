# Verwenden Sie die Mehrsprachigkeitsfunktion.

Syntax:

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

Exmaple:

```
Home <fr: Maison><zh: 家>
```

Hier ist die STANDARD vollständige Liste aller von Navi+ unterstützten Sprachcodes.

Manchmal können Sie SEO-Systeme verwenden, um diese zweibuchstabigen Sprachcodes in ein anderes Format zu konvertieren, zum Beispiel EN in en-EN oder english zu ändern. In diesem Fall können Sie den Sprachcode weiterhin normal und einfach verwenden.
