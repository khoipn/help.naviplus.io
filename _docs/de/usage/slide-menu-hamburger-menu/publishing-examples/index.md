# Veröffentlichungsbeispiele

Unten finden Sie einige Beispiele, wie Sie ein Slide Menu auf Ihrer Website veröffentlichen können.

#### Methode 1: Öffnen Sie dieses Menü von einem anderen Navi+ Menüelement mit der Syntax open:NaviMenu(SF-123456789)

Dies ist eine einfache Möglichkeit, ein Slide Menu zu veröffentlichen, wenn mehrere Navi+ Menüs verwendet werden. Zum Beispiel kann ein Tabbar-Element verwendet werden, um das Slide Menu zu öffnen und den Menüraum zu erweitern.
