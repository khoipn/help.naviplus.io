# Nivel de Diseño 1 — Elementos del Menú Principal

Los ajustes en **Diseño → Nivel 1** se aplican a todos los elementos de nivel superior en tu menú. Esta es la capa base — el Nivel 2-3 hereda estos valores, y los ajustes por elemento los sobrescriben.

---

## Estilo de Texto

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color del Texto** | Color del texto de la etiqueta | Herencia del tema |
| **Tamaño de Fuente (px)** | Tamaño del texto | Herencia del tema |
| **Peso de Fuente** | Grosor: 100–900 o Heredar | Heredar |
| **Transformación de Texto** | Ninguno / Capitalizar / Mayúsculas / Minúsculas | Ninguno |
| **Familia de Fuente** | Google Font o "Default (Theme)" | Predeterminado |
| **Color de Descripción** | Color de la línea de descripción secundaria | 60% de opacidad del Color del Texto |
| **Tamaño de Descripción (px)** | Tamaño de fuente de la línea de descripción | 70% del Tamaño de Fuente |

---

## Estilo de Ícono / Imagen

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color del Ícono** | Color de los íconos vectoriales | Hereda el Color del Texto |
| **Tamaño del Ícono (px)** | Dimensiones del ícono | 22px |
| **Espacio Texto-Ícono (px)** | Espacio entre el ícono y el texto | Automático |
| **Radio de Imagen (px)** | Redondeo de esquinas para imágenes en miniatura | 0 |

---

## Espaciado

| Campo | Descripción | Formato |
|---|---|---|
| **Relleno del Elemento (px)** | Espaciado interno de cada elemento | `top right bottom left` o `mobile  desktop` |
| **Margen del Elemento (px)** | Espaciado externo de cada elemento | `top right bottom left` o `mobile  desktop` |

---

## Fondo y Marco

| Campo | Descripción | Predeterminado |
|---|---|---|
| **Color de Fondo** | Color de fondo del área del elemento | Transparente |
| **Imagen de Fondo** | Imagen de fondo (URL o galería) | Ninguna |
| **Opacidad** | Opacidad de fondo (0–100) | 100 |
| **Ocultar Fondo** | Ocultar fondo para revelar el fondo del dispositivo | Apagar |
| **Altura (px)** | Altura de fila de cada elemento | 50px |
| **Radio de Borde (px)** | Redondeo de esquinas | 0 |
| **Sombra Proyectada** | Sombra proyectada debajo de los elementos | Apagar |
| **Altura Mínima (px)** | Altura mínima del elemento | Sin límite |

---

## Visualización

| Campo | Descripción |
|---|---|
| **Diseño de Visualización** | Disposición de ícono + texto: Automático / De Arriba a Abajo / De Izquierda a Derecha / De Abajo a Arriba / De Derecha a Izquierda / Solo Ícono / Solo Texto / Vacío |
| **Alinear** | Alineación del contenido: Automático / Izquierda / Centro / Derecha |
| **Ocultar Texto** | Ocultar todas las etiquetas de texto — solo ícono |
| **Mostrar Flecha de Expandir** | Mostrar flecha de expandir en elementos padre con hijos |

---

## Insignia

| Campo | Descripción |
|---|---|
| **Color de Insignia** | Color de fondo para la insignia/punto de notificación en los elementos |

---

## Plantillas de Color

Navi+ incluye paletas de colores preestablecidas: **Claro, Oscuro, Océano, Arena, Rosa, Bosque**.

Seleccionar una plantilla completa automáticamente los valores de color comunes. Puedes ajustar campos individuales después de aplicar.
