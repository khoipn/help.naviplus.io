# ¿Por qué no se muestra el menú de Navi+ en mi sitio web?

Para mostrar el menú de Navi+ en su sitio web, hay dos opciones claras:

1. **Embed 
[Navi+ On all page] into your theme and adjust the display in the Navi+ application**: Esta también es la forma más común y fácil de hacerlo. Sin embargo, sus menús de Navi+ solo pueden fijarse en la pantalla (en la parte inferior, encabezado o flotante). Los ajustes son muy diversos, con tres niveles diferentes para asegurarse de que puede navegar según sus necesidades.
   * Nivel 1: Mostrar/ocultar el menú de Navi+ y mostrar/ocultar en grupos específicos de páginas. En Shopify, hay algunos grupos comunes de páginas como la página de inicio, catálogo, productos y página.
   * Nivel 2: Mostrar/ocultar en dispositivos móviles o de escritorio. Debido a las experiencias completamente diferentes entre móvil y escritorio, muchos usuarios crean menús separados para móvil y escritorio.
   * Nivel 3: Por palabra clave: Puede mostrar/ocultar cuando la URL de la página web contiene/no contiene una o varias palabras clave. Esta opción es muy útil y flexible, permitiéndole crear menús específicos para cada página o cada producto.
2. **Session 
[Navi+ On Session] is used to embed directly into each page of the Shopify theme**. Con esta opción, puede tener una opción adicional de no fijar el menú (en la parte inferior, encabezado o flotante) pero permitiendo que fluya con el contenido de la página. Este efecto es útil cuando desea crear menús tipo cuadrícula o menús tipo barra que fluyan a medida que desplaza la pantalla.
