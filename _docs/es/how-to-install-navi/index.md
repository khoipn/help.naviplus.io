# Cómo instalar Navi+

No necesitas saber sobre desarrollo o diseño para usar Navi+. Es un software preconstruido que se puede integrar fácilmente en tu sitio web. Toda la configuración — incluyendo características y diseño visual — se gestiona directamente en Navi+, sin causar ningún conflicto con tu sistema existente.

***

### Paso 1: Instalar la Aplicación

**Si estás usando Shopify:**\
Por favor, instala la aplicación oficial desde el siguiente enlace:\
<https://apps.shopify.com/pronavi-navigation-design>

**Si estás usando otras plataformas** como WordPress, WooCommerce, Wix, Webflow, Squarespace, o un sitio web autoalojado:

1. Ve a <https://naviplus.io>
2. Crea una cuenta gratuita
3. Inicia sesión y conecta tu sitio web siguiendo las instrucciones.

***

### Paso 2: Crear Menús e Incrustarlos en Tu Sitio Web

Una vez que hayas iniciado sesión, puedes comenzar a crear menús para tu sitio web.\
**Navi+** proporciona varios tipos de menús para satisfacer diferentes necesidades:

* **Tabbar**\
  Una barra de navegación fija en la parte inferior de la pantalla, que permite a los usuarios acceder rápidamente a secciones clave. Ideal para diseños móviles.
* **Mega Menu**\
  Un gran menú desplegable con múltiples columnas, perfecto para mostrar categorías de productos o contenido detallado en una estructura clara.
* **Slide Menu (Hamburger Menu)**\
  Un menú lateral oculto que se desliza desde el borde de la pantalla. Ideal para ahorrar espacio y mantener un diseño limpio y moderno.
* **Floating Button Menu**\
  Un botón de acción flotante que puede abrir una lista de enlaces rápidos o resaltar campañas o promociones especiales.

Cada tipo de menú tiene su propio proceso de configuración, que está completamente guiado paso a paso dentro del panel de control de Navi+.

También puedes usar múltiples tipos de menús de Navi+ al mismo tiempo en diferentes páginas — por ejemplo, un Tabbar para móvil, un Mega Menu para escritorio, y un Floating Button para tu llamada a la acción o promociones.
