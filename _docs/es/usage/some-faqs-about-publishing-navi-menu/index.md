# Algunas preguntas frecuentes sobre la publicación del menú Navi+

**¿Por qué no se muestra Navi+ en mi sitio?**

Verifica que hayas completado todos los pasos. Para Tabbar/FAB, necesitas: 1) Incrustar la aplicación Navi+ en tu tema, 2) Habilitar el interruptor de menú Publicar.
