# Ejemplos de publicación

A continuación se presentan algunos ejemplos de cómo publicar un Slide Menu en su sitio web.

#### Método 1: Abra este menú desde otro elemento de menú de Navi+ utilizando la sintaxis open:NaviMenu(SF-123456789)

Esta es una forma sencilla de publicar un Slide Menu al usar múltiples menús de Navi+. Por ejemplo, un elemento de Tabbar se puede usar para abrir el Slide Menu y expandir el espacio del menú.
