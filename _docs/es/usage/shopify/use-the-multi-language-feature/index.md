# Usa la función de múltiples idiomas.

Sintaxis:

```
Contenido predeterminado <código-de-idioma-1: Contenido-1><código-de-idioma-2: Contenido-2>
```

Ejemplo:

```
Inicio <fr: Maison><zh: 家>
```

Aquí está la lista completa de códigos de idioma soportados por Navi+.

A veces, puedes usar sistemas SEO para convertir estos códigos de idioma de dos caracteres en un formato diferente, por ejemplo, cambiando EN a en-EN o english. En ese caso, aún puedes usar el código de idioma normalmente y fácilmente.
