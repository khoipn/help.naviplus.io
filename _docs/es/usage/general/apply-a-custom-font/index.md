# Aplicar una fuente personalizada

Navi+ ofrece una amplia selección de Google Fonts, proporcionando una forma optimizada y conveniente de estilizar su menú. Google Fonts son rápidos, altamente legibles y ampliamente soportados, lo que los convierte en una excelente opción para la mayoría de los diseños. Sin embargo, si desea utilizar una fuente personalizada que no está incluida en la selección predeterminada, puede aplicarla manualmente a su menú de Navi+. Aquí le mostramos cómo:

Por ejemplo, si ha comprado la fuente **Honey Salt** de [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") y desea usarla en su menú de Navi+, simplemente agregue el siguiente código CSS a la sección **Hoja de estilos interna / CSS**:

```
{
  font-family: "Honey Salt";
}
```
