# Aplicar una fuente personalizada

Navi+ ofrece una amplia selección de Google Fonts, proporcionando una forma optimizada y conveniente de estilizar tu menú. Google Fonts son rápidos, altamente legibles y ampliamente soportados, lo que los convierte en una excelente opción para la mayoría de los diseños. Sin embargo, si deseas utilizar una fuente personalizada que no está incluida en la selección predeterminada, puedes aplicarla manualmente a tu menú de Navi+. Aquí te explicamos cómo:

Por ejemplo, si has comprado la fuente **Honey Salt** de [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") y deseas usarla en tu menú de Navi+, simplemente agrega el siguiente código CSS a la sección de **Hoja de Estilos Interna / CSS**:

```
{
  font-family: "Honey Salt";
}
```
