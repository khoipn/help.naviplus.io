# Publicar

La sección **Publicar** controla cómo aparece tu menú en tu sitio web en vivo: alternar el menú encendido/apagado, elegir el método de inserción, configurar la visibilidad en dispositivos y filtrar por URL. Este es el paso final después de diseñar: sin guardar la configuración de Publicar, el menú no aparecerá.

## Referencia Rápida

```
Tipo de menú           Método primario
────────────────────────────────────────────────────
Sticky / TABBAR     Inserciones de App (Shopify) / <head> (Global) + habilitar alternar
Sticky / FAB        Inserciones de App (Shopify) / <head> (Global) + habilitar alternar
Deslizante (CONTEXT)     Inserciones de App / <head> + habilitar alternar + configurar disparador
Sección (Mega/Grid) Insertar/Reemplazar (Selector CSS) o Bloque de App (Shopify)
```

**Regla:** Habilitar alternar = el menú está en vivo. Deshabilitar alternar = completamente oculto del sitio web, no es necesario eliminar el código de inserción.
