# La estructura HTML y CSS de todos los menús de Navi+

A continuación se muestra la estructura HTML y CSS para todos los menús de Navi+, que permanecerá estable a lo largo del ciclo de vida del producto. Si bien pueden existir variaciones menores para tipos de menús específicos, la estructura del marco central a continuación no cambiará. Puede hacer referencia a esta estructura para escribir código CSS personalizado para un diseño de menú avanzado.
