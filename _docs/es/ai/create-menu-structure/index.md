# Crear estructura de menú

Describe el menú que necesitas y deja que la IA construya un esqueleto listo para usar — obtén un punto de partida rápido en lugar de agregar cada elemento a mano. Esta función es **gratis**.

Abrir: el botón **AI** → tarjeta **Crear estructura de menú**.

---

## Antes de empezar — haz una copia de seguridad de tu menú

> **⋮ Más → Copia de seguridad / Restaurar** — haz una copia de seguridad de tu menú actual antes de crear una nueva estructura, ya que el resultado sobrescribe el menú existente.

---

## Cómo usarlo

1. Abre la tarjeta **Crear estructura de menú**.
2. (Opcional) Elige a qué niveles debe la IA **agregar íconos automáticamente**:

   | Opción | Se aplica a |
   |---|---|
   | Nivel 1 | Elementos principales (nivel 1) |
   | Nivel 2 | Subelementos de nivel 2 |
   | Nivel 3 | Subelementos de nivel 3 |

3. Escribe tu descripción en el cuadro **Describe tu solicitud**, p. ej. "Menú para una tienda de cosméticos: Inicio, Productos (Labial, Hidratante, Mascarilla), Promociones, Contacto."
4. (Opcional) Haz clic en **Usar plantilla** para insertar una indicación de ejemplo según tu tipo de menú, luego edítala.
5. Haz clic en **Crear menú**. El progreso se muestra en el **Registro de actividad** a la derecha.
6. La IA construye la estructura y la aplica al editor. Revisa, refina, luego haz clic en **Guardar**.

> Haz clic en **Guardar** (el ícono de disco) para almacenar tu indicación y reutilizarla la próxima vez.

---

## Después de crear

La IA solo construye el **esqueleto de la estructura** (elementos, jerarquía, íconos). Debes revisar y agregar:

- Enlaces (URL) para cada elemento
- Imágenes e insignias si es necesario
- Colores y diseño a través de los paneles **Diseño** y **Configuración**

---

## Descripciones de ejemplo

- Tienda de moda masculina: Inicio, Tops (Camisetas, Camisas, Chaquetas), Pantalones, Accesorios, Rebajas, Contacto.
- Tab bar de 5 elementos para una app de entrega de comida: Inicio, Buscar, Carrito, Pedidos, Cuenta.
- Slide menu para un blog: Últimas publicaciones, Categorías (Tecnología, Viajes, Comida), Acerca de, Contacto.
- Mega menú para una tienda de electrónica con 4 grupos: Teléfonos, Portátiles, Accesorios, Promociones — 4–5 subelementos cada uno.

---

## Caso de estudio — construir un mega menú para una tienda de electrónica

**Contexto:** Una tienda de electrónica necesita un mega menú de escritorio con varios grupos, cada grupo con varios subelementos, con íconos para una fácil lectura.

1. **Copia de seguridad:** ⋮ Más → Copia de seguridad / Restaurar.
2. Abre la tarjeta **Crear estructura de menú**. Marca **Nivel 1** y **Nivel 2** para que la IA agregue íconos tanto a los grupos como a los subelementos.
3. Escribe la descripción:

   ```
   Mega menú para una tienda de electrónica, 4 grupos:
   - Teléfonos (iPhone, Samsung, Xiaomi, Accesorios de teléfono)
   - Portátiles (Oficina, Gaming, MacBook, Accesorios de portátil)
   - Electrodomésticos (Purificadores de agua, Robots aspiradores, Freidoras de aire)
   - Promociones
   ```

4. Haz clic en **Crear menú**. Observa el **Registro de actividad** a la derecha. La IA construye un mega menú de 4 grupos con subelementos e íconos.
5. Agrega después de crear: asigna una **URL** a cada elemento, agrega una imagen de banner al grupo Promociones, luego **Guarda**.

**Resultado:** Un esqueleto de mega menú de varios niveles con íconos de inmediato — solo necesitas adjuntar enlaces y afinar detalles, en lugar de crear decenas de elementos a mano.
