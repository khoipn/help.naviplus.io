# ¿Qué es Navi+? (Cómo funciona)

Navi+ es un **constructor de navegación sin código** — te permite crear menús hermosos y de alta conversión para cualquier sitio web sin tocar tu tema ni escribir una sola línea de código.

Si los visitantes de tu sitio tienen dificultades para encontrar lo que buscan, tu menú podría estar fallando. Navi+ te ayuda a solucionarlo en minutos.

## Cinco tipos de menú

Elige el menú adecuado para tu caso de uso:

| Tipo de menú | Ideal para |
| --- | --- |
| **Tab Bar** | Navegación mobile-first, se siente como una app nativa |
| **Mega Menu** | Menús desplegables ricos con imágenes, columnas y enlaces — versiones separadas para Desktop y Mobile |
| **Slide Menu** | Menús hamburguesa con soporte multinivel |
| **Grid Menu** | Navegación visual de categorías — ideal para e-commerce |
| **FAB** | Botón flotante de acceso rápido, siempre al alcance |

Puedes usar múltiples tipos de menú juntos en el mismo sitio.

## Cómo funciona

Navi+ es un **widget** — no reemplaza ni modifica tu tema existente. En cambio, inyecta un script ligero en tu página que renderiza el menú sobre tu diseño existente.

Construyes y gestionas todo desde el **panel de Navi+**. Los cambios se publican al instante — sin necesidad de editar código, modificar plantillas ni republicar tu tema.

Aquí está el flujo:

1. Creas un menú en el panel
2. Navi+ genera un pequeño script
3. Ese script se embebe en tu sitio (a través de App Block en Shopify, o un fragmento de código en otros lugares)
4. El menú se renderiza sobre tu diseño existente en cada carga de página

## Plataformas compatibles

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · y cualquier sitio que acepte una etiqueta `<script>`.

## Lo que Navi+ NO ES

- **No es un tema** — funciona junto a tu tema existente, no en lugar de él
- **No es código que instalas manualmente** — un App Block (Shopify) o un snippet de incrustación es todo lo que necesitas
- **No es un plugin que puede romper tu sitio** — si Navi+ se desactiva, el menú original de tu tema vuelve automáticamente

## Diseñado para la velocidad

Los assets de Navi+ se entregan a través de Cloudflare CDN con ~52ms de tiempo de carga promedio — generalmente más rápido que los propios scripts de tu plataforma.
