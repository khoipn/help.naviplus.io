# ¿Qué es Navi+? (Cómo funciona)

Navi+ es un **constructor de navegación sin código** — te permite crear menús hermosos y de alta conversión para cualquier sitio web sin tocar tu tema o escribir una sola línea de código.

Si los visitantes de tu sitio tienen dificultades para encontrar lo que buscan, tu menú podría estar decepcionándolos. Navi+ te ayuda a solucionar eso en minutos.

## Cinco tipos de menú

Elige el menú adecuado para tu caso de uso:

| Tipo de Menú | Mejor Para |
| --- | --- |
| **Tab Bar** | Navegación móvil primero, se siente como una aplicación nativa |
| **Mega Menu** | Desplegables ricos con imágenes, columnas y enlaces — tiene versiones separadas para Desktop y Mobile |
| **Slide Menu** | Menús tipo hamburguesa con soporte de múltiples niveles |
| **Grid Menu** | Navegación visual por categorías — excelente para comercio electrónico |
| **FAB** | Botón flotante de acceso rápido, siempre al alcance |

Puedes usar múltiples tipos de menú juntos en el mismo sitio.

## Cómo funciona

Navi+ es un **widget** — no reemplaza ni modifica tu tema existente. En su lugar, inyecta un script ligero en tu página que renderiza el menú sobre tu diseño existente.

Construyes y gestionas todo desde el **tablero de Navi+**. Los cambios se publican al instante — no es necesario editar código, modificar plantillas o republicar tu tema.

Aquí está el flujo:

1. Creas un menú en el tablero
2. Navi+ genera un pequeño script
3. Ese script se incrusta en tu sitio (a través de App Block en Shopify, o un fragmento de código en otro lugar)
4. El menú se renderiza sobre tu diseño existente en cada carga de página

## Funciona con tu plataforma

Navi+ se integra en las herramientas que ya usas:

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · y cualquier sitio web que acepte una `<script>` etiqueta.

## Lo que NO es Navi+

- **No es un tema** — funciona junto a tu tema existente, no en lugar de él
- **No es código que instalas manualmente** — un App Block (Shopify) o un fragmento de incrustación es todo lo que se necesita
- **No es un plugin que puede romper tu sitio** — si Navi+ está desactivado, el menú original de tu tema vuelve automáticamente

## Construido para la velocidad

Los activos de Navi+ se entregan a través de Cloudflare CDN a ~52ms de tiempo de carga promedio — típicamente más rápido que los propios scripts de tu plataforma. Después de la primera carga, los menús se almacenan en caché y se cargan al instante.
