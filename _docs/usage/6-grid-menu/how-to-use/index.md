# How to use?

**Grid Menu** is a popular menu type used on many apps and e-commerce websites. It typically displays catalogs or key features and is often shown on the homepage.

Mobile  Grid menu take up screen space since they float on the screen. As a section menu, they scroll along with the page.

**Key Benefits:**

* Works seamlessly on both mobile and desktop for a consistent experience
* Saves space and scrolls with the page.
* Offers multiple presentation styles with **images**, **grids**, and **multi-level** layouts for a rich, professional experience

### Step 1: Edit the menu tree

#### 1.1. Understanding the Menu Tree

The **menu tree** is the most important part of any menu‚Äîit defines the structure.

Here‚Äôs an example of a menu tree of mobile grid menu:


This is a 1 level menu tree. Grid Menus do not support multiple menu levels to maintain simplicity..
