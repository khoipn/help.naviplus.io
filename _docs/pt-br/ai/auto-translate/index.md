# Traduzir menu automaticamente

Traduza os nomes, descrições e links dos itens do seu menu para vários idiomas em apenas alguns passos, em vez de digitar cada tradução manualmente. *(Business / Elite)*

Abra: o botão **AI** → cartão **Traduzir menu automaticamente**.

---

<div class="install-step" markdown="1">
### 1 — Faça backup do seu menu

**⋮ Mais → Backup / Restaurar** — crie uma cópia do seu menu antes de traduzir.
</div>

<div class="install-step" markdown="1">
### 2 — Selecione os idiomas de destino

Digite na caixa de busca para escolher um ou mais idiomas para os quais traduzir. A IA traduz o **nome**, o **link** e a **descrição** de cada item.

> Ignore o idioma atual do menu — o texto padrão já está nesse idioma.
</div>

<div class="install-step" markdown="1">
### 3 — Confirme as opções

**Escopo da tradução**

- **Adicionar apenas idiomas ausentes** — Para cada item, adiciona traduções apenas para os idiomas ainda não presentes. As traduções existentes são mantidas.
- **Retraduzir tudo do zero** — Substitui todas as traduções existentes e traduz novamente por completo.

**Tratamento de links**

- **Mudar o link automaticamente para /[idioma]/nome-do-produto** — Quando sua loja usa um prefixo de caminho, por exemplo `/vi/collections` — bom para SEO multilíngue.
- **Usar o mecanismo automático do Navi+** — Deixe o Navi+ cuidar dos links com base no idioma exibido.
</div>

<div class="install-step" markdown="1">
### 4 — Traduza e salve

Clique no botão de tradução. O progresso aparece no registro de atividades; a IA traduz em paralelo, então geralmente é rápido. Ao terminar, revise o menu e clique em **Salvar**.

> As traduções são armazenadas dentro de cada item do menu. O menu exibe automaticamente o idioma certo com base no idioma do site que o visitante está visualizando.
</div>

## Requisito de plano

Este recurso está nos planos **Business / Elite**. Um plano inferior verá um aviso de upgrade ao abrir este cartão.

## Estudo de caso — leve um menu em vietnamita para inglês e japonês

**Contexto:** Uma loja tem um menu completo em vietnamita e acabou de abrir para clientes internacionais, precisando de inglês e japonês. A loja usa os prefixos de caminho `/en/` e `/ja/`.

1. **Backup:** ⋮ Mais → Backup / Restaurar (o menu já tem conteúdo, não pule).
2. Abra o cartão **Traduzir menu automaticamente**. Na caixa de busca, selecione **Inglês** e **日本語 (Japonês)**. Ignore o vietnamita, já que é o idioma de origem.
3. **Escopo da tradução:** escolha **Adicionar apenas idiomas ausentes** (primeira execução, então ainda não existem traduções — a IA adiciona EN e JP).
4. **Tratamento de links:** escolha **Mudar o link automaticamente para /[idioma]/...** porque a loja usa prefixos — os links se tornam `/en/...` e `/ja/...`, bom para SEO multilíngue.
5. Clique em traduzir. A IA traduz o nome + a descrição de cada item para ambos os idiomas em segundos.
6. Revise alguns itens principais (especialmente nomes próprios/nomes de marca), depois **Salvar**.

**Da próxima vez que você adicionar novos produtos:** execute novamente com **Adicionar apenas idiomas ausentes** — os itens já traduzidos são mantidos, apenas os novos itens são traduzidos.

**Resultado:** O menu exibe vietnamita / inglês / japonês com base no idioma do site que o visitante está visualizando, com links que correspondem ao prefixo.
