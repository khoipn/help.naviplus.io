# Criar estrutura de menu

Descreva o menu que você precisa e deixe a IA construir um esqueleto pronto — obtenha um ponto de partida rápido em vez de adicionar cada item manualmente. Este recurso é **gratuito**.

Abra: o botão **AI** → cartão **Criar estrutura de menu**.

---

## Antes de começar — faça backup do seu menu

> **⋮ Mais → Backup / Restaurar** — faça backup do seu menu atual antes de criar uma nova estrutura, já que o resultado sobrescreve o menu existente.

---

## Como usar

1. Abra o cartão **Criar estrutura de menu**.
2. (Opcional) Escolha em quais níveis a IA deve **adicionar ícones automaticamente**:

   | Opção | Aplica-se a |
   |---|---|
   | Nível 1 | Itens principais (nível 1) |
   | Nível 2 | Subitens de nível 2 |
   | Nível 3 | Subitens de nível 3 |

3. Digite sua descrição na caixa **Descreva sua solicitação**, por exemplo "Menu para uma loja de cosméticos: Início, Produtos (Batom, Hidratante, Máscara), Promoções, Contato."
4. (Opcional) Clique em **Usar modelo** para inserir um prompt de exemplo para o seu tipo de menu, depois edite-o.
5. Clique em **Criar menu**. O progresso aparece no **Registro de atividades** à direita.
6. A IA constrói a estrutura e a aplica ao editor. Revise, refine e clique em **Salvar**.

> Clique em **Salvar** (o ícone de disco) para armazenar seu prompt e reutilizá-lo da próxima vez.

---

## Após criar

A IA constrói apenas o **esqueleto da estrutura** (itens, hierarquia, ícones). Você deve revisar e adicionar:

- Links (URLs) para cada item
- Imagens e distintivos, se necessário
- Cores e layout pelos painéis **Design** e **Configuração**

---

## Descrições de exemplo

- Loja de moda masculina: Início, Tops (Camisetas, Camisas, Jaquetas), Calças, Acessórios, Liquidação, Contato.
- Tab bar de 5 itens para um app de entrega de comida: Início, Buscar, Carrinho, Pedidos, Conta.
- Slide menu para um blog: Posts recentes, Categorias (Tecnologia, Viagem, Comida), Sobre, Contato.
- Mega menu para uma loja de eletrônicos com 4 grupos: Celulares, Laptops, Acessórios, Promoções — 4–5 subitens cada.

---

## Estudo de caso — construa um mega menu para uma loja de eletrônicos

**Contexto:** Uma loja de eletrônicos precisa de um mega menu para desktop com múltiplos grupos, cada grupo com vários subitens, com ícones para facilitar a leitura.

1. **Backup:** ⋮ Mais → Backup / Restaurar.
2. Abra o cartão **Criar estrutura de menu**. Marque **Nível 1** e **Nível 2** para que a IA adicione ícones tanto aos grupos quanto aos subitens.
3. Digite a descrição:

   ```
   Mega menu para uma loja de eletrônicos, 4 grupos:
   - Celulares (iPhone, Samsung, Xiaomi, Acessórios para celular)
   - Laptops (Escritório, Gaming, MacBook, Acessórios para laptop)
   - Eletrodomésticos (Purificadores de água, Aspiradores robô, Fritadeiras elétricas)
   - Promoções
   ```

4. Clique em **Criar menu**. Acompanhe o **Registro de atividades** à direita. A IA constrói um mega menu de 4 grupos com subitens e ícones.
5. Adicione após criar: atribua uma **URL** a cada item, adicione uma imagem de banner ao grupo Promoções, depois **Salvar**.

**Resultado:** Um esqueleto de mega menu de vários níveis com ícones imediatamente — você só precisa anexar os links e fazer ajustes finos, em vez de criar dezenas de itens manualmente.
