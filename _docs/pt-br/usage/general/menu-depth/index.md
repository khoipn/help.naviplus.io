# Menu depth

O suporte de profundidade padrão para o menu Navi+ é 2 para o pacote Starter (gratuito) e aumentado para 3 para o pacote Business. Na maioria dos casos, uma profundidade de 2 é suficiente para necessidades ordinárias - E também fácil para os usuários, pois a grande profundidade do menu pode fazer com que os usuários se percam. No entanto, em alguns casos, por exemplo, você pode querer adicionar toda a sua estrutura de catálogo de produtos ao menu Navi+, nesse caso, uma profundidade de 3 níveis pode ser uma escolha perfeita, facilitando o acesso dos seus visitantes aos seus serviços e produtos.

Atualmente, com o pacote Business, você tem até 3 níveis de menu
