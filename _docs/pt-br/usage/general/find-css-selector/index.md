# Encontrar Seletor CSS

**Seletor CSS** é um conceito muito interessante em CSS — ele define qual parte do seu site pode ser **substituída** ou **ter um menu Navi+ inserido** (seja *antes* ou *depois* dela).

Aqui estão alguns **exemplos de Seletores CSS** que você pode usar:

```css
#header           /* A área do cabeçalho */
.navbar           /* A barra de navegação principal */
.footer           /* A seção do rodapé */
.main-content     /* O contêiner de conteúdo principal */
.product-gallery  /* A seção de imagens do produto */
div#menu-wrapper  /* Um div específico com ID “menu-wrapper” */
section.hero      /* A seção hero no topo da página */
```

Siga as instruções abaixo para **encontrar o seletor CSS correto** no seu site.
