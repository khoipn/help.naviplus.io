# Aplicar uma fonte personalizada

Navi+ oferece uma ampla seleção de Google Fonts, proporcionando uma maneira otimizada e conveniente de estilizar seu menu. Google Fonts são rápidos, altamente legíveis e amplamente suportados, tornando-os uma excelente escolha para a maioria dos designs. No entanto, se você gostaria de usar uma fonte personalizada que não está incluída na seleção padrão, você pode aplicá-la manualmente ao seu menu Navi+. Aqui está como:

Por exemplo, se você comprou a fonte **Honey Salt** de [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") e deseja usá-la em seu menu Navi+, basta adicionar o seguinte código CSS à seção **Folha de Estilos Interna / CSS**:

```
{
  font-family: "Honey Salt";
}
```
