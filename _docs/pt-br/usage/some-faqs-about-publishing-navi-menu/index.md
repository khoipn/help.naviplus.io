# Algumas perguntas frequentes sobre a publicação do menu Navi+

**Por que o Navi+ não está sendo exibido no meu site?**

Verifique se você completou todas as etapas. Para Tabbar/FAB, você precisa: 1) Incorporar o aplicativo Navi+ no seu tema, 2) Ativar o botão de publicação do menu.
