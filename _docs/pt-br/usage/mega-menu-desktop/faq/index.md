# FAQ

#### Pergunta 1: Por que o Navi+ não exibe automaticamente o Mega Menu de desktop em dispositivos móveis?

A experiência de navegação em desktop e mobile é fundamentalmente diferente — especialmente em como os usuários navegam e interagem com um site.

No desktop, os usuários têm mais espaço na tela e podem visualizar facilmente grandes e complexos mega menus exibidos sob o cabeçalho.

No entanto, em dispositivos móveis, a tela é menor e os usuários navegam principalmente com os polegares, então mostrar o mesmo mega menu tornaria a interface desordenada, difícil de tocar e mais lenta para carregar.

É por isso que o Navi+ não converte automaticamente um mega menu de desktop em uma versão móvel.

Em vez disso, você deve criar um menu móvel dedicado que ofereça uma melhor experiência ao usuário:

* Menus principais — coloque-os em uma TabBar na parte inferior da tela, onde são fáceis de alcançar com o polegar.
* Menus adicionais — mova-os para um Slide (Hamburger) menu (painel deslizante) ou um Mega Menu específico para mobile.

👉 Com o conveniente recurso de Copiar & Colar menu do Navi+, você pode duplicar seu mega menu de desktop e adaptá-lo para mobile em apenas alguns minutos — dando a você controle total e garantindo que ambas as versões estejam perfeitamente otimizadas para suas plataformas.
