# Exemplos de publicação

Abaixo estão alguns exemplos de como publicar um Slide Menu em seu site.

#### Método 1: Abra este menu a partir de outro item de menu Navi+ usando a sintaxe open:NaviMenu(SF-123456789)

Esta é uma maneira simples de publicar um Slide Menu ao usar vários menus Navi+. Por exemplo, um item de Tabbar pode ser usado para abrir o Slide Menu e expandir o espaço do menu.
