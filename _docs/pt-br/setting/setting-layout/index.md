# Setting Layout — Posição e Espaçamento

Controla posição, ordem de empilhamento e largura de submenu do menu inteiro.

## Padding e Margem do Menu

Margin Top/Right/Bottom/Left: espaço do menu até a borda da tela.
Padding Top/Right/Bottom/Left: espaço dentro do contêiner do menu.

Casos de uso: Empurrar Tabbar acima da barra de navegação iOS: Margin Bottom = 34. Espaço acima do Mega Menu: Margin Top = 8.

## Z-Index

Número maior = por cima. Menu coberto por cabeçalho fixo → aumentar. Menu cobrindo widget de chat → diminuir.
Cabeçalhos de tema: 1000–9999. Widgets de chat: 9999–99999.

## Largura do Submenu

Submenu Width (px): largura do painel de coluna única. Padrão 360px.

## Desktop

Mostrar submenu ao passar o mouse, Full-expand submenu width (px), Direção do submenu.
