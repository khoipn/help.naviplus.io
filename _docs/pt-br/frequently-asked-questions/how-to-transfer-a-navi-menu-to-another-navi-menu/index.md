# Como Transferir um Menu Navi+ para Outro Menu Navi+

Você já construiu um menu que realmente gostou... e então percebeu que precisava do mesmo em outro lugar? 😅

Boas notícias: você não precisa reconstruí-lo do zero.

### Quando você precisaria transferir um menu?

Isso é comumente necessário nas seguintes situações:

* Você completou um menu no mobile e quer usá-lo no desktop (ou vice-versa)
* Você construiu um menu para um site e quer reutilizá-lo em outro site, com alguns ajustes para se adequar melhor ao novo design

Atualmente, não há uma ferramenta automática que possa transferir menus entre sistemas Navi+. Portanto, o método mais eficaz por enquanto é o clássico, mas confiável: copiar e colar.

### Guia passo a passo

#### Passo 1: Prepare o menu de destino

Primeiro, configure a estrutura do menu de destino.

Crie a estrutura base com itens de menu de Nível 1 e Nível 2 para que esteja pronta para receber o conteúdo transferido.

#### Passo 2: Abra ambos os menus lado a lado

Abra ambos os menus no mesmo navegador (por exemplo, Chrome):

* Uma aba: Menu de origem (o menu que já tem conteúdo)
* Uma aba: Menu de destino (o menu para o qual você está transferindo)

Mantê-los abertos lado a lado tornará o processo mais rápido e ajudará a evitar erros.

#### Passo 3: Copie do menu de origem

Na aba do menu de origem, selecione os itens de menu que você deseja mover e clique no botão COPIAR.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2Fz8sYKzjcUT1ouCcroiPu%2F1.2.webp?alt=media&#x26;token=d69221b6-a952-4c27-833c-3df74d9d4044" alt=""><figcaption></figcaption></figure>

#### Passo 4: Cole no menu de destino

Mude para a aba do menu de destino, navegue até o local correto e cole os itens de menu copiados.

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2FLX3Be5eqfItKUMpXSCCj%2F1.3.webp?alt=media&#x26;token=74143e7e-efb7-485a-9eba-6a28d2ec9229" alt=""><figcaption></figcaption></figure>

🔁 Repita os Passos 3 e 4 até que toda a estrutura do menu tenha sido totalmente transferida. Dicas: Mova o menu em seções (por nível ou por ramo) para garantir que nada seja esquecido e a estrutura permaneça organizada.
