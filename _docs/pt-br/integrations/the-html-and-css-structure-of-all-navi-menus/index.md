# A estrutura HTML e CSS de todos os menus Navi+

Abaixo está a estrutura HTML e CSS para todos os menus Navi+, que permanecerá estável ao longo do ciclo de vida do produto. Embora possam existir variações menores para tipos de menu específicos, a estrutura básica abaixo não mudará. Você pode usar essa estrutura para escrever código CSS personalizado para design avançado de menus.
