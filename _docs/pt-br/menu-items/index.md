# Itens do Menu

Um **item de menu** é uma única linha na sua lista de menu. Os itens são os blocos de construção de cada menu do Navi+ — cada um pode ser um link de navegação, um cabeçalho de grupo, um espaçador em branco ou um divisor de aba.

## Referência rápida

**Onde edito um item?**  
Clique em qualquer item na árvore de menu → o painel Editar Item se abre à direita. Pressione **Atualizar** para aplicar as alterações, depois **Salvar** (`Ctrl+S` / `Cmd+S`) para publicar.

**Ordem dos campos em Editar Item:**  
Mídia (ícone / imagem) → Informações Básicas (texto, link, descrição, distintivo, divisor) → Layout → Interface (cores, espaçamento) → Avançado (animação, CSS) → Publicar (visibilidade)
