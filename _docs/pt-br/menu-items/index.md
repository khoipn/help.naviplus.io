# Itens de menu

Um **item de menu** é uma única linha em sua lista de menu. Os itens são os blocos de construção de cada menu Navi+ — cada um pode ser um link de navegação, um cabeçalho de grupo, um espaçador em branco ou um divisor de abas.

## Referência rápida

**Onde edito um item?**  
Clique em qualquer item na árvore do menu → o painel Editar item é aberto à direita. Pressione **Update** para aplicar as alterações e depois **Save** (`Ctrl+S` / `Cmd+S`) para publicar.

**Ordem de campos em Editar item:**  
Mídia (ícone / imagem) → Informações básicas (texto, link, descrição, badge, divisor) → Layout → Interface (cores, espaçamento) → Avançado (animação, CSS) → Publicar (visibilidade)
