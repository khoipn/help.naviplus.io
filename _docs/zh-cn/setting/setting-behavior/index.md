# 设置行为 — 动画与交互

设置在 **设置 → 动画** 和 **设置 → 展开效果** 控制菜单的速度、移动样式和交互反馈。

---

## 动画速度

适用于菜单中的所有运动（打开子菜单、项目过渡等）。

| 选项 | 持续时间 |
|---|---|
| 无 | 无动画 |
| 非常慢 | ~500ms |
| 慢 | ~350ms |
| **中等** *(默认)* | ~200ms |
| 快 | ~120ms |
| 非常快 | ~60ms |

---

## 动画样式

控制菜单的显示方式。仅适用于 **Slide menu** 和 **Tabbar** — Mega Menu 始终使用淡入效果。

| 选项 | 效果 |
|---|---|
| **滑动** *(默认)* | 从屏幕边缘滑入 |
| **淡入** | 渐渐淡入 |

---

## 展开效果

仅适用于 **Slide menu** — 控制子菜单的打开方式。

| 选项 | 效果 |
|---|---|
| **自上而下展开** | 子菜单在原地向下展开 |
| **从右侧滑入** | 子菜单作为新面板从右侧滑入 |

---

## 触摸反馈 *(Business / Elite)*

用户在移动设备上点击项目时的背景闪烁颜色。

| 字段 | 描述 |
|---|---|
| **触摸背景颜色** | 闪烁颜色（十六进制） |
| **触摸背景不透明度** | 闪烁不透明度（0–100） |

留空 = 无触摸效果。

---

## 箭头样式 *(Elite)*

自定义显示在有子项的父项上的展开箭头。

| 字段 | 描述 | 默认 |
|---|---|---|
| **颜色** | 箭头颜色（十六进制） | 继承文本颜色 |
| **不透明度** | 箭头不透明度（0–100） | 30 |
| **大小（px）** | 箭头大小 | 20px |
| **图标** | Remixicon 名称（例如 `ri-arrow-right-s-line`） | 系统默认 |
