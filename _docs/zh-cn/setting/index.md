# 设置

控制菜单的行为和显示方式 — 动画、展开行为、位置、可见性规则、幻灯片选项。

## 主题

| # | 主题 | |
|---|---|---|
| 1 | 动画、展开效果、触摸反馈 | [设置行为](/zh-cn/docs/setting/setting-behavior/) |
| 2 | 位置、间距、Z 索引、子菜单宽度 | [设置布局](/zh-cn/docs/setting/setting-layout/) |
| 3 | 按 URL、设备、Shopify 页面类型显示/隐藏 | [设置显示规则](/zh-cn/docs/setting/setting-display-rules/) |
| 4 | 幻灯片方向、宽度、关闭按钮 | [设置幻灯片](/zh-cn/docs/setting/setting-slide/) |

## 快速参考

- 行为 — 动画速度、展开样式、触摸反馈
- 布局 — 内边距/外边距、Z 索引、子菜单宽度
- 显示规则 — URL 关键词、设备、Shopify 页面类型
- 幻灯片 — 方向、宽度、关闭按钮（仅幻灯片菜单）
