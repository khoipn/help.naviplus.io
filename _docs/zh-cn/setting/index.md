# 设置

该**设置**部分控制您的菜单**行为**和**显示**。这些设置决定动画速度、展开行为、菜单定位、可见性规则（哪些页面显示菜单）和特定于幻灯片的选项。

---

## 主题

| # | 主题 | |
|---|---|---|
| 1 | 动画、展开效果和触摸反馈 | [设置行为](/docs/setting/setting-behavior/) |
| 2 | 位置、间距、z-index和子菜单宽度 | [设置布局](/docs/setting/setting-layout/) |
| 3 | 按URL、设备和Shopify页面类型显示/隐藏 | [设置显示规则](/docs/setting/setting-display-rules/) |
| 4 | 幻灯片方向、宽度和关闭按钮 | [设置幻灯片](/docs/setting/setting-slide/) |

---

## 快速参考

- **行为** — 动画速度、展开样式、触摸反馈
- **布局** — 内边距/外边距、z-index、子菜单宽度
- **显示规则** — 按URL关键字、设备、Shopify页面类型显示
- **幻灯片** — 方向、宽度、关闭按钮 *(仅限幻灯片菜单)*
