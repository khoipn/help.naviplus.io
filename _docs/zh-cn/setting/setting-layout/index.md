# 设置布局 — 位置与间距

控制整个菜单的位置、堆叠顺序和子菜单宽度。

## 菜单内边距和外边距

Margin Top/Right/Bottom/Left：菜单到屏幕边缘的距离。
Padding Top/Right/Bottom/Left：菜单容器内部的空间。

使用场景：将 Tabbar 推到 iOS 导航栏上方：Margin Bottom = 34。Mega Menu 上方留白：Margin Top = 8。

## Z 索引

数值越高 = 越靠上。菜单被固定标题覆盖 → 增大。菜单遮住聊天组件 → 减小。
主题标题：1000–9999。聊天组件：9999–99999。

## 子菜单宽度

Submenu Width（px）：单列面板宽度。默认 360px。

## 桌面端

悬停显示子菜单、Full-expand submenu width（px）、子菜单方向。

## 多站点（Elite）

逗号分隔的域名列表 例：brand.com,shop.brand.com
