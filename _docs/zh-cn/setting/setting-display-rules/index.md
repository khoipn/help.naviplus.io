# 设置显示规则 — 条件可见性

控制您的菜单根据设备类型、URL 或 Shopify 页面类型的显示位置和时间。

---

## 按设备显示

在菜单的 **发布** 标签中找到（不是每个项目）：

| 切换 | 效果 |
|---|---|
| **移动** | 在移动屏幕上显示/隐藏菜单 |
| **桌面** | 在桌面屏幕上显示/隐藏菜单 |

默认：仅启用移动。如果您希望菜单在更大屏幕上显示，必须手动启用桌面。

> 要按设备显示/隐藏单个项目 → 请参见 [项目可见性](/docs/menu-items/item-visibility/)。

---

## 按 URL 关键字显示

**设置 → 按关键字显示** 中的两个字段让您控制菜单在哪些 URL 上显示。

### 仅在 URL 包含关键字时显示

**字段：** `发布 — 包含关键字`

```
collections, products/sale
```

菜单仅在其 URL 包含 `collections` **或** `products/sale` 的页面上显示。

**示例：** 一个 "商店导航" 菜单用于产品页面 → 仅在 `collections/` 和 `products/` 上显示。

---

### 当 URL 包含关键字时隐藏

**字段：** `发布 — 不包含关键字`

```
checkout, account/login
```

菜单在结账和登录页面上隐藏；在其他地方可见。

**示例：** 在结账期间隐藏 Tabbar 以避免分散客户注意力。

---

## 按 Shopify 页面类型显示

**发布** 标签还包括 6 个 Shopify 页面组（仅限 Shopify 商店）：

| 组 | 页面 |
|---|---|
| **首页** | 首页 |
| **产品详情** | 产品详情页面 |
| **产品类别** | 集合/类别页面 |
| **页面** | 静态页面（关于、联系等） |
| **博客** | 博客列表和文章页面 |
| **其他** | 不在上述 5 个组中的页面 |

取消选中一个组 → 菜单将不会出现在该页面类型上。

---

## 规则如何叠加

所有条件按以下顺序一起评估：

```
1. 设备是否启用？              →  如果否：隐藏
2. 页面类型是否选中？           →  如果否：隐藏
3. URL 是否匹配 "不显示"？    →  如果是：隐藏
4. URL 是否匹配 "仅显示"？     →  如果否：隐藏
5. 显示菜单
```
