# 设置滑动 — 滑动菜单选项

这些设置仅适用于**滑动菜单**（上下文滑动/汉堡菜单）。

---

## 滑动方向

**设置 → 滑动方向** — 菜单从哪里滑入，以及它是如何固定的。

| 选项 | 描述 |
|---|---|
| 从左侧 | 从左边缘滑入（最常见） |
| 从右侧 | 从右边缘滑入 |
| 从顶部 | 从顶部滑下 |
| 从底部 | 从底部滑上 |
| 固定左侧（桌面） | 左侧始终可见的侧边栏 — 无需触发 |
| 固定右侧（桌面） | 右侧始终可见的侧边栏 |

**在移动设备上固定 / 在桌面上固定:**
- **在移动设备上固定** → 菜单在移动设备上始终可见，无需打开。
- **在桌面上固定** → 菜单在桌面上始终可见（持久侧边栏）。

---

## 滑动宽度

| 字段 | 描述 | 默认 |
|---|---|---|
| **移动宽度** | 移动设备上的宽度（px或%） | 自动 |
| **桌面最大宽度（px）** | 桌面上的最大宽度 | 自动 |
| **桌面子宽度（px）** | 桌面上二级子菜单面板的宽度 | 自动 |

**示例：** 从左侧滑动，移动屏幕的80%，桌面最大400px:
- 移动宽度 = `80%`
- 桌面最大宽度 = `400`

---

## 关闭图标

| 字段 | 描述 | 默认 |
|---|---|---|
| **隐藏关闭图标** | 隐藏×关闭按钮 | 可见 |
| **关闭图标对齐** | 关闭按钮的位置：左 / 右 | 右 |
| **关闭图标顶部（px）** | 菜单顶部到关闭按钮的距离 | 12px |

> 如果您隐藏关闭按钮，用户必须点击菜单外的覆盖区域才能关闭它。当菜单覆盖大部分屏幕时，请保持关闭按钮可见。

---

## 桌面子菜单方向

**设置 → 桌面 → 子菜单方向**

当滑动菜单作为固定侧边栏显示在桌面上时，子菜单可以向4个方向打开：

| 值 | 子菜单打开方向 |
|---|---|
| 1 | 右（默认） |
| 2 | 左 |
| 3 | 下 |
| 4 | 上 |
