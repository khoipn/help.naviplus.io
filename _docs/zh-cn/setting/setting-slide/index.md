# 设置幻灯片 — 幻灯片菜单选项

仅适用于幻灯片菜单（Context Slide / 汉堡菜单）。

## 幻灯片方向

From Left（最常见）/ From Right / From Top / From Bottom / Fixed Left desktop / Fixed Right desktop。
Fix on Mobile = 移动端始终可见。Fix on Desktop = 固定侧边栏。

## 幻灯片宽度

Mobile Width（px 或 %）、Desktop Max Width（px）、Desktop Sub Width（px）。均默认 Auto。
示例：Mobile Width = 80%，Desktop Max Width = 400。

## 关闭图标

Hide Close Icon（默认显示）、Close Icon Align Left/Right（默认 Right）、Close Icon Top px（默认 12px）。
如果隐藏：用户必须点击遮罩层关闭。

## 桌面端子菜单方向

1=Right（默认）、2=Left、3=Down、4=Up
