# 用您的AI应用自动设计

使用您已经熟悉的 AI 工具重新设计菜单 — ChatGPT、Claude 或 Gemini。Navi+ 准备好数据和提示词；您只需把它交给 AI 运行，再把结果带回来。此功能 **免费**。

打开方式：**AI** 按钮 → **用您的 AI 应用自动设计** 卡片。

---

## 开始前 — 备份您的菜单

> **⋮ 更多 → 备份 / 恢复** — 在进行更改前，将当前菜单保存为文件或保存到云端。请不要跳过这一步。

---

<div class="install-step" markdown="1">
### 1 — 训练您的 AI

您需要一个基于网页的 AI 工具。推荐的大上下文模型：**Claude Opus 4.7**、**ChatGPT GPT-5** 或 **Gemini Pro 1.5**。

点击 **打开训练文档**，复制全部内容，并粘贴到您的 AI 对话中。这会教会 AI 了解 Navi+ 菜单的结构方式。
</div>

<div class="install-step" markdown="1">
### 2 — 将您当前的菜单（JSON）发送给 AI

当 AI 通过第 1 步理解了结构后，将您当前的菜单发送给它：

- **将当前菜单复制到剪贴板** 并粘贴到对话中，或
- **下载 JSON 文件** 并将其附加到对话中。
</div>

<div class="install-step" markdown="1">
### 3 — 给出设计指令

用自然语言输入您的需求，例如：

- 给菜单再添加 2 个项目
- 将主色改为蓝色
- 将产品项目归到一个带标题的部分下
- 添加 2 个项目：Blog 和 Careers。
- 将主色改为蓝色 `#1a73e8`。
- 精简为 4 个主项目，其余的移到一个 More 项目下。
- 给每个项目添加一个合适的图标，并将 Promotions 项目加粗。
- 将所有项目名称翻译成英文。

您对设计拥有完全的掌控权。AI 会以 JSON 形式返回一个新菜单。
</div>

<div class="install-step" markdown="1">
### 4 — 粘贴结果并应用

复制 AI 返回的 JSON，然后通过以下两种方式之一将其应用到 Navi+：

- **选项 1：** 将 JSON 粘贴到输入框中并点击 **应用**。
- **选项 2：** 点击 **上传 JSON 文件** 并选择您的 `.json` 文件。

重复第 3–4 步，直到菜单看起来满意为止，然后点击 **保存**。
</div>

> 提示：AI 只会更改 JSON 中的结构和内容。您配置过但未提及的字段会被保留 — 但应用后仍请检查菜单。

---

## 案例研究 — 用 Claude 整理标签栏

**背景：** 一家商店有一个杂乱的 8 项标签栏，希望将其精简为 5 个项目并添加图标，同时保留所有现有链接。

1. **备份：** ⋮ 更多 → 备份 / 恢复 → 保存一个副本。
2. **第 1 步 — 训练 AI：** 打开 Claude（Opus 4.7），在 Navi+ 中点击 **打开训练文档**，复制全部内容并粘贴到 Claude 中。
3. **第 2 步 — 发送菜单：** 点击 **将当前菜单复制到剪贴板**，并将 JSON 粘贴到 Claude 中。
4. **第 3 步 — 下达指令：** 告诉 Claude：

   ```
   将这 8 个项目合并为 5 个主项目：Home、Products、Promotions、
   Blog、Account。把剩下的项目移到合适的子分组中。
   给每个一级项目添加一个图标。保持所有现有链接不变。
   ```

5. **第 4 步 — 应用：** Claude 返回新的 JSON。复制它，粘贴到 Navi+ 的 **应用** 框中 → 点击 **应用**。
6. 检查：5 个项目、完整图标、原始链接完好无损。需要更多调整？重复第 3–4 步。满意了？点击 **保存**。

**结果：** 一个干净的 5 项标签栏，带图标，所有链接都被保留 — 无需手动编辑每个项目。
