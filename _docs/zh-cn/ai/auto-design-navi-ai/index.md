# Navi+ AI自动设计

让 Navi+ 处理所有 AI 工作 — 您只需在编辑器中描述您想要的菜单。无需外部工具，无需复制和粘贴 JSON。*(Business / Elite)*

打开方式：**AI** 按钮 → **Navi+ AI 自动设计** 卡片。

---

## 它与"用您的 AI 应用自动设计"有何不同？

| | 您的 AI 应用 | Navi+ AI |
|---|---|---|
| AI 工具 | 您打开 ChatGPT / Claude / Gemini | Navi+ 为您调用 AI |
| 复制/粘贴 JSON | 是，手动 | 不需要 |
| 步骤 | 4 步 | 1 步（对话） |
| 计划 | 免费 | Business / Elite |

---

## 开始前 — 备份您的菜单

> **⋮ 更多 → 备份 / 恢复** — 在应用 AI 结果前备份您当前的菜单。

---

## 如何使用

1. 打开 **Navi+ AI 自动设计** 卡片。
2. 在对话框中输入描述，例如"为一家时尚商店创建一个 5 项标签栏，黑白配色。"
3. （可选）使用 **上传** 按钮附加一张参考图片，让 AI 遵循其布局/颜色。
4. 点击 **发送**。AI 返回一个设计并将其应用到编辑器以供预览。
5. 继续对话以完善（"更改 Contact 图标"、"删除最后一个项目"）……或点击 **新对话** 重新开始。
6. 满意了？点击 **保存**。

> 您的对话历史会被保存 — 点击 **历史** 可重新打开之前的设计会话。

## 计划要求

此功能属于 **Business / Elite** 计划。在较低计划上，当您打开此卡片时会看到升级提示。

## 示例提示词

- 为一家化妆品商店创建一个 5 项标签栏，柔和的粉色调。
- 给每个项目添加一个图标，并将最后一个项目变成一个醒目的 Contact 按钮。
- 设计一个像我刚发送的图片那样的大菜单。*(使用上传按钮附加一张图片)*
- 删除 Blog 项目，将 Promotions 和 Sale 合并为一个。

---

## 案例研究 — 用 3 轮对话构建一个化妆品标签栏

**背景：** 一家还没有菜单的新化妆品商店，希望有一个与其柔和粉色品牌相匹配的标签栏。

1. 为安全起见先 **备份**（⋮ 更多 → 备份 / 恢复）。
2. 打开 **Navi+ AI 自动设计** 卡片。第 1 轮：

   ```
   为一家化妆品商店创建一个 5 项标签栏：Home、Products、
   Offers、Beauty Guide、Account。柔和的粉色调。
   ```

   AI 立即构建标签栏并在预览中显示。
3. 第 2 轮 — 使用 **上传** 按钮附加一张示例 UI 图片，然后输入：

   ```
   让圆角和图标与这张图片中的风格相匹配。
   ```

4. 第 3 轮 — 一个小调整：

   ```
   将 Account 项目变成一个带电话图标的醒目 Contact 按钮。
   ```

5. 满意了，点击 **保存**。下次需要更改时，点击 **历史** 重新打开此会话。

**结果：** 一个与品牌相匹配的标签栏，在 3 轮对话中完成，无需离开 Navi+。
