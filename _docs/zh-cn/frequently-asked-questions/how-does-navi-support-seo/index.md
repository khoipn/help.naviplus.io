# Navi+如何支持SEO？

链接是优化SEO与搜索引擎的关键因素之一。使用链接进行SEO优化适用于反向链接和内部链接。

1. **反向链接** (External Links)\
   反向链接是指其他网站指向您网站的链接。这些是SEO中的一个重要因素。
   * **质量胜于数量**：来自信誉良好的网站的链接比来自不明确或低质量来源的许多链接更有价值。
   * **自然链接**：自然的、未购买或未交换的链接对SEO更有利。
   * **锚文本**：您用作链接的文本应包含与您链接的页面内容相关的关键词。
2. **内部链接**\
   同一网站上页面之间的内部链接有助于引导用户并提高页面排名。
   * **明智地使用内部链接**：确保您网站上的每个重要页面都从其他页面链接。
   * **相关锚文本**：使用相关关键词作为内部链接的锚文本以提高可搜索性。

Navi+从Elite计划开始自动支持基于链接的SEO。在该级别，Navi+菜单中的每个链接都经过优化，包含完整的URL（确保搜索引擎正确索引）、锚文本和相关标签。Navi+还通过添加标题和alt标签来支持图像SEO，以优化图像搜索。

将您的整个产品目录添加到Navi+菜单对搜索引擎非常有利，帮助它们理解页面列表、网站地图和与关键词相关的内容。Navi+的速度是SEO的另一个主要优势。所有库和资源（图像、JS和CSS）都托管在顶级CDN系统上，响应时间低于100毫秒且缓存稳定，这提升了您的SEO评分。
