# 多语言 (Shopify Markets)

Navi+ 以两种方式支持多语言：**行内翻译语法**和直接内置于您的菜单中的 **语言切换器**。

## 1. 行内翻译语法

使用以下语法直接在菜单项的标签中添加翻译：

```
Default-content <language-code-1: Content-1><language-code-2: Content-2>
```

**示例** — 一个以英语、法语和越南语显示的标签栏：

```
Home <fr: Maison><vi: Trang chủ>
```

当菜单渲染时，Navi+ 会检查当前页面语言并显示匹配的翻译。如果没有找到匹配项，则回退到默认。

此语法适用于 **标签**、**描述** 和 **链接到** 字段。

## 2. 语言切换器

您可以在任何 Navi+ 菜单中构建语言切换器。在菜单项上使用 `open:ChangeLanguage()` 操作：

```
open:ChangeLanguage(fr)
```

当访客点击该项时，网站会切换到法语。您可以从像 [Flaticon](https://www.flaticon.com) 这样的网站添加国旗图标，以使切换器更具视觉效果。
