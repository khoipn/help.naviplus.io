# 使用多语言功能。

语法：

```
默认内容 <language-code-1: Content-1><language-code-2: Content-2>
```

示例：

```
首页 <fr: Maison><zh: 家>
```

这是Navi+支持的所有语言代码的默认完整列表。

有时，您可以使用SEO系统将这两个字符的语言代码转换为不同的格式，例如，将EN更改为en-EN或english。在这种情况下，您仍然可以正常轻松地使用语言代码。
