# 菜单深度

Navi+菜单的默认深度支持为Starter套餐（免费）时为2，Business套餐时增加到3。在大多数情况下，深度为2就足够满足普通需求 - 而且对于用户来说也很简单，因为菜单的深度过大可能会让用户迷失方向。然而，在某些情况下，例如，您可能希望将整个产品目录结构添加到Navi+菜单中，在这种情况下，3级深度可能是一个完美的选择，使您的访客更容易访问您的服务和产品。

目前，使用Business套餐，您最多可以拥有3个菜单级别
