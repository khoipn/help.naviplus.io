# 菜单深度

Navi+ 菜单的默认深度支持为 Starter 套餐（免费）时为 2，Business 套餐时增加到 3。在大多数情况下，深度为 2 足以满足普通需求 - 而且对于用户来说，菜单的深度过大可能会让用户迷失方向。然而，在某些情况下，例如，您可能希望将整个产品目录结构添加到 Navi+ 菜单中，在这种情况下，3 层深度可能是一个完美的选择，使您的访客更容易访问您的服务和产品。

目前，使用 Business 套餐，您最多可以拥有 3 个菜单级别
