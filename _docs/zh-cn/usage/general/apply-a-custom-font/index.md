# 应用自定义字体

Navi+ 提供了广泛的 Google Fonts 选择，为您的菜单提供了一种优化和便捷的样式方式。Google Fonts 快速、易读且广泛支持，使其成为大多数设计的优秀选择。然而，如果您想使用不包含在默认选择中的自定义字体，您可以手动将其应用于您的 Navi+ 菜单。以下是操作方法：

例如，如果您从 [MJ Type](https://mjtype.com/product/honey-salt/ "MJ Type") 购买了 **Honey Salt** 字体，并希望在您的 Navi+ 菜单中使用它，只需将以下 CSS 代码添加到 **内部样式表 / CSS** 部分：

```
{
  font-family: "Honey Salt";
}
```
