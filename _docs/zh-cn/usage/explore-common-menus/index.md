# 探索常见菜单

Navi+ 为您的网站提供全面的菜单解决方案——远超传统下拉菜单。

根据您的设备类型和所需的用户体验，您可以选择几种灵活且优化的菜单样式：

#### 1. Tab Bar (Bottom Navigation)

也称为底部导航栏、底部导航或标签导航——是一种现代导航样式，被全球品牌如 Shein、ASOS、Zara、Etsy、H&amp;M、Nike 和 Uniqlo 用于提升移动用户体验。它适用于各种规模的网站，并在移动和桌面上都有效。

出现在移动屏幕的底部，帮助用户：

* 快速访问您网站的 4-5 个主要区域
* 用拇指轻松导航
* 享受一种在移动设备上感觉自然的应用程序体验

平台：**桌面和移动（极受欢迎）**

#### 2. Mega Menu (Desktop) - 极受欢迎

为桌面网站设计的 Mega Menu 非常适合您想要：

* 同时显示多个产品类别或内容部分
* 将链接组织成列、组或视觉布局
* 提供专业且直观的浏览体验

平台：**仅限桌面**

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2FwXn5EF8H17b2QVSEYSQp%2FSlide-0.webp?alt=media&#x26;token=5ad18e94-82ba-43fa-aede-943741b49221" alt=""><figcaption></figcaption></figure>

***

#### 3. Mobile Mega Menu - 不常见

为移动屏幕优化的 Mega Menu 版本——紧凑、适合拇指且易于导航。

您可以自定义类别、图标和视觉效果的显示方式，以确保最佳的移动用户体验。移动 Mega Menu 和桌面 Mega Menu 之间最大的区别在于，在移动设备上，您最多可以有 4 个菜单项，因此您需要决定哪些是最重要的并应优先考虑。

平台：**仅限移动**

***

#### 4. Slide Menu (Hamburger Menu)

从屏幕的左侧或右侧滑出的面板，通常用于次要或实用内容，例如：

* 账户信息、政策、支持页面
* 不常访问的产品类别
* 额外链接，如“关于我们”、“联系我们”或“博客”

平台：**桌面（不常见）和移动（极受欢迎）**

<figure><img src="https://4226540503-files.gitbook.io/~/files/v0/b/gitbook-x-prod.appspot.com/o/spaces%2F4y2gLTmg4ToH5vkorG8s%2Fuploads%2FLCmdTapRLNGzq0dIAnMB%2FSlide-0-2.webp?alt=media&#x26;token=67d280d0-58b1-4ce1-b4d3-ddaae8a3307c" alt=""><figcaption></figcaption></figure>

***

#### 5. FAB - Floating Menu / Quick Access

一个固定在屏幕角落（通常是右下角或底部中间）的浮动按钮或菜单。

理想用于：

* 快速联系按钮（聊天、热线、Zalo、Messenger）
* 关键操作（CTA），如“立即购买”或“开始免费试用”
* 在不占用太多空间的情况下增加转化率

平台：**桌面（受欢迎）和移动（极受欢迎）**

***

#### 6. Grid Menu

Grid Menu 以网格布局显示导航项，允许用户一次查看多个类别——每个类别作为带有图标或图像的方形瓷砖。

这种样式特别适合：

* 移动或平板电脑的主页和子 Mega Menu
* 包含 6-12 个类别的产品集合
* 创建干净、应用程序般的视觉体验

平台：**仅限移动（受欢迎）**
