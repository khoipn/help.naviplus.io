# 常见问题

#### 问题 1：为什么Navi+在移动设备上不自动显示桌面Mega菜单？

桌面和移动设备上的浏览体验根本不同——尤其是在用户如何导航和与网站互动方面。

在桌面上，用户有更多的屏幕空间，可以轻松查看在标题下显示的大型复杂Mega菜单。

然而，在移动设备上，屏幕较小，用户主要用拇指进行导航，因此显示相同的Mega菜单会使界面显得杂乱，难以点击，并且加载速度较慢。

这就是为什么Navi+不自动将桌面Mega菜单转换为移动版本。

相反，您应该创建一个专门的移动菜单，以提供更好的用户体验：

* 主菜单——将它们放在屏幕底部的TabBar中，方便用拇指轻松触及。
* 附加菜单——将它们移入Slide（汉堡）菜单（滑出面板）或移动专用的Mega菜单。

👉 使用Navi+方便的复制和粘贴菜单功能，您可以在几分钟内复制桌面Mega菜单并将其调整为移动版——让您完全掌控，并确保两个版本都完美优化以适应其平台。
