# 设计

该 **设计** 部分控制您菜单的整体外观：颜色、排版、间距、背景和图标。此处的设置适用于 **整个菜单**，使用三层继承系统。

## 快速参考

```
全局默认值（硬编码）
    ↓
第 1 级  — 适用于所有顶级菜单项
    ↓  （第 2-3 级如果为空则继承）
第 2-3 级 — 适用于子菜单 / 下拉菜单
    ↓  （覆盖所有内容）
逐项 — 编辑项目中的单独设置
```

**规则：** 空 = 从上面继承。设置一个值 = 覆盖。
