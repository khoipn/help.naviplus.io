# 设计级别 1 — 主菜单项

在 **设计 → 级别 1** 中的设置适用于菜单中的所有顶级项。这是基础层 — 级别 2-3 继承这些值，逐项设置会覆盖它们。

---

## 文本样式

| 字段 | 描述 | 默认 |
|---|---|---|
| **文本颜色** | 标签文本颜色 | 主题继承 |
| **字体大小 (px)** | 文本大小 | 主题继承 |
| **字体粗细** | 厚度：100–900 或 继承 | 继承 |
| **文本转换** | 无 / 大写 / 全部大写 / 全部小写 | 无 |
| **字体系列** | Google 字体或 "默认 (主题)" | 默认 |
| **描述颜色** | 次要描述行的颜色 | 文本颜色的 60% 不透明度 |
| **描述大小 (px)** | 描述行的字体大小 | 字体大小的 70% |

---

## 图标 / 图片样式

| 字段 | 描述 | 默认 |
|---|---|---|
| **图标颜色** | 矢量图标的颜色 | 继承文本颜色 |
| **图标大小 (px)** | 图标尺寸 | 22px |
| **文本-图标间距 (px)** | 图标与文本之间的间隙 | 自动 |
| **图片半径 (px)** | 缩略图的角落圆度 | 0 |

---

## 间距

| 字段 | 描述 | 格式 |
|---|---|---|
| **项目内边距 (px)** | 每个项目的内部间距 | `上 右 下 左` 或 `移动  桌面` |
| **项目外边距 (px)** | 每个项目的外部间距 | `上 右 下 左` 或 `移动  桌面` |

---

## 背景 & 框架

| 字段 | 描述 | 默认 |
|---|---|---|
| **背景颜色** | 项目区域的背景颜色 | 透明 |
| **背景图片** | 背景图片 (URL 或画廊) | 无 |
| **不透明度** | 背景不透明度 (0–100) | 100 |
| **隐藏背景** | 隐藏背景以显示设备背景 | 关闭 |
| **高度 (px)** | 每个项目的行高 | 50px |
| **边框半径 (px)** | 角落圆度 | 0 |
| **投影** | 项目下方的投影 | 关闭 |
| **最小高度 (px)** | 最小项目高度 | 无限制 |

---

## 显示

| 字段 | 描述 |
|---|---|
| **显示布局** | 图标 + 文本排列：自动 / 自上而下 / 自左而右 / 自下而上 / 自右而左 / 仅图标 / 仅文本 / 空 |
| **对齐** | 内容对齐：自动 / 左 / 中 / 右 |
| **隐藏文本** | 隐藏所有文本标签 — 仅图标 |
| **显示展开箭头** | 在有子项的父项上显示展开箭头 |

---

## 徽章

| 字段 | 描述 |
|---|---|
| **徽章颜色** | 项目上通知徽章/点的背景颜色 |

---

## 颜色模板

Navi+ 包含预设颜色调色板：**浅色、深色、海洋、沙子、玫瑰、森林**。

选择模板会自动填充常见颜色值。应用后，您可以调整各个字段。
