# 最新更新

### 2026年4月

多语言菜单现在可以自动运行 — Navi+ 检测访客语言并立即切换菜单标签，无需额外设置。Tab Slide 是新的布局选项，可将菜单项分组到水平标签中而非平铺列表，适合在滑动菜单中组织"男士 / 女士 / 促销"等分区。颜色选择器经过重新设计，提供预设调色板，可在几秒内将品牌色应用到整个菜单。所有高级设置 — 动画、触摸反馈、图标、CSS 类 — 现已整合到一个面板中。WordPress 支持得到改善，与 Elementor、Divi 和 WPBakery 的冲突减少。

---

### 2026年3月

分区标题现在可以同时链接到页面并展开子项 — 点击展开，点击箭头导航到页面。Grid Menu 在小屏幕上的布局问题已修复，在各种手机尺寸上显示一致。分隔线和触摸高亮颜色可以自由自定义。

---

### 2026年2月

Z-index 冲突现在自动解决 — 当购物车抽屉、搜索浮层或 Cookie 横幅出现时，Navi+ 自动退到其后面，不会遮挡内容。Slide Horizontal 使二级面板从右侧滑入，如原生 iOS 应用一样自然，适合深层产品分类树。菜单配置现在可以通过弹窗以 JSON 格式导出和导入，方便备份或在店铺间迁移。菜单中的图片支持懒加载，初始页面加载更快。可以直接在任何菜单标签中添加表情符号。

---

### 2026年1月

新的 Slide Horizontal 菜单类型专为移动端深层导航树而设计 — 每一级从右侧滑入，返回按钮和标题自动更新。Item 模板让您构建一次菜单结构并一键在任意菜单中复用。Import/Export 现在支持 AI：粘贴链接列表或站点地图，Navi+ 自动生成完整菜单树；也可以导出为 JSON，在 AI 工具中编辑后再导入。新的 Debug 模式可以直接从控制台在您的实时网站上叠加显示 z-index 值和菜单边界。CDN 基础设施升级至 Cloudflare R2 边缘网络，并新增了第二个备用 CDN。

---

### 2025年12月

Navi+ 现在可在所有平台上运行 — WordPress、WooCommerce、Wix、Webflow、Squarespace 以及任何支持 `<script>` 标签的网站。与 Shopify 相同的控制台、相同的菜单类型、相同的单行嵌入代码。控制台界面现已支持多语言。

---

### 2025年11月

修复了部分设备上页面顶部和底部自动显示/隐藏时菜单闪烁的问题。新增安全边界阈值，菜单显示和隐藏更加流畅自然。

---

### 2025年10月

Navi+ 全部资源迁移至 Cloudflare 全球边缘网络 — 菜单平均加载时间降至约 52ms。Elite 套餐用户还获得 BunnyCDN 备用 CDN 支持，确保即使某个 CDN 出现故障菜单也能正常加载。SLA 升级至 99.99%。
