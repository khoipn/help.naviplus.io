# 更新日志 / 发布说明

### 2026年4月

- **多语言菜单** 现在自动工作 — Navi+ 检测访问者的语言并即时切换菜单标签，无需额外设置。
- **Tab Slide** — 新的布局选项，将菜单项分组为水平标签，而不是平面列表（例如，滑动菜单中的 Men / Women / Sale 标签）。
- **颜色模板** — 重新设计的颜色选择器，带有预设调色板。几秒钟内将您的品牌颜色应用于整个菜单。
- **统一设置面板** — 所有高级选项（动画、触摸反馈、图标、可重用的 CSS 类）现在集中在一个地方。
- **WordPress** — 更快的嵌入，与 Elementor、Divi 和 WPBakery 的冲突更少。

---

### 2026年3月

- **节标题** 现在可以同时链接到页面并展开子项 — 点击展开，点击箭头导航。
- **网格菜单** 在小屏幕上的布局边缘情况已修复，以确保在所有手机尺寸上的一致渲染。
- **自定义分隔符颜色** — 菜单中的分隔线和触摸高亮颜色现在可以设置为任何颜色。

---

### 2026年2月

- **Z-index 自动修复** — Navi+ 现在检测到您的商店面板（购物车抽屉、搜索覆盖、Cookie 横幅）何时出现，并自动在其后面调整。
- **Slide Horizontal** — 新的滑动方向，二级面板从右侧滑入，像原生 iOS 应用一样。
- **导出 / 导入弹出窗口** — 备份、移动或以 JSON 格式在几秒钟内复制整个菜单配置。
- **懒加载图像** — 菜单中的图像仅在可见时加载，提高初始页面加载速度。
- **Emoji 支持** — 直接将 Emoji 添加到任何菜单项标签中。

---

### 2026年1月

- **Slide Horizontal 菜单类型** — 为移动设备上的深层导航树而构建。每个级别从右侧滑入；后退按钮和标题自动更新。
- **项目模板** — 一次保存菜单结构，并在任何菜单中一键重用。
- **AI 导入 / 导出** — 粘贴链接列表或网站地图，Navi+ 生成完整的菜单树。或导出为 JSON，在 AI 工具中编辑，再导入回来。
- **调试模式** — 直接从仪表板在您的实时网站上覆盖 z-index 值、菜单边界和 CSS 选择器高亮。
- **CDN 升级** — 基础设施升级到 Cloudflare R2 边缘网络，并添加了第二个故障转移 CDN。

---

### 2025年12月

- **所有平台** — Navi+ 现在可以在 WordPress、WooCommerce、Wix、Webflow、Squarespace 和任何接受 `<script>` 标签的网站上运行。与 Shopify 相同的仪表板，相同的菜单类型。
- **仪表板本地化** — Navi+ 仪表板 UI 现在提供多种语言。

---

### 2025年11月

- **自动隐藏闪烁修复** — 修复了一个错误，在某些设备上，自动隐藏/自动显示行为在页面顶部和底部边缘会闪烁。

---

### 2025年10月

- **Cloudflare CDN** — 所有 Navi+ 资产迁移到 Cloudflare 的全球边缘网络。平均菜单加载时间降至 ~52ms。
- **Elite 故障转移** — Elite 计划用户获得第二个 BunnyCDN 备用，如果一个 CDN 出现故障，菜单保持在线。SLA 升级至 99.99%。
