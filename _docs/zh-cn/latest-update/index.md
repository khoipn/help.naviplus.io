# 最新更新

**2025 - 12月9日**

推出在所有平台上运行的Navi+版本，超越Shopify：WordPress、WooCommerce、Wix等，访问[dash.naviplus.app](https://dash.naviplus.app/ "dash.naviplus.app")

***

**2025 - 11月7日**\
修复自动隐藏、自动显示仍然显示、在到达设备边缘时隐藏菜单的问题，通过创建安全边缘。

**原因：** 在移动设备（如iphone + android）上处理网站的UI/UX使其美观，但在android上创建了一个导致错误的间隙。

***

**2025 - 10月20日**\
由于BunnyCDN上的缓存清除故障，将所有CDN服务从BunnyCDN迁移到Cloudflare。
