# 最新更新

### 2026年4月

- **多语言菜单** 自动运行 — Navi+ 检测访客语言并立即切换菜单标签，无需额外设置。
- **Tab Slide** — 新布局选项，将菜单项分组到横向标签而非平铺列表（例如滑动菜单中的"男装 / 女装 / 特卖"标签）。
- **颜色模板** — 重新设计的颜色选择器，提供预设调色板。几秒内将品牌色应用到整个菜单。
- **统一设置面板** — 动画、触摸反馈、图标、CSS 类等所有高级选项集中在一处。
- **WordPress** — 嵌入速度更快，与 Elementor、Divi、WPBakery 的冲突减少。

---

### 2026年3月

- **分区标题** 可同时链接到页面并展开子项 — 点击展开，点击箭头导航到页面。
- **Grid Menu** — 修复小屏幕上的布局问题，在各种手机尺寸上显示一致。
- **自定义分隔线颜色** — 分隔线和触摸高亮颜色可自由设置。

---

### 2026年2月

- **Z-index 自动解决** — 购物车抽屉、搜索浮层或 Cookie 横幅出现时，Navi+ 自动退到其后面。
- **Slide Horizontal** — 二级面板从右侧滑入，如原生 iOS 应用。
- **Export / Import 弹窗** — 以 JSON 格式几秒内完成菜单配置的备份、迁移或复制。
- **图片懒加载** — 菜单中的图片仅在进入视野时加载，初始页面加载更快。
- **表情符号支持** — 可直接在任何菜单项标签中添加表情符号。

---

### 2026年1月

- **Slide Horizontal 菜单类型** — 专为移动端深层导航树设计。每级从右侧滑入，返回按钮和标题自动更新。
- **Item 模板** — 保存一次菜单结构，一键在任意菜单中复用。
- **AI Import / Export** — 粘贴链接列表或站点地图，Navi+ 自动生成菜单树。也可导出 JSON，用 AI 编辑后再导入。
- **Debug 模式** — 从控制台直接在实时网站上叠加显示 z-index、菜单边界和 CSS 选择器。
- **CDN 升级** — 基础设施升级至 Cloudflare R2 边缘网络，新增第二个备用 CDN。

---

### 2025年12月

- **全平台支持** — Navi+ 现在可在 WordPress、WooCommerce、Wix、Webflow、Squarespace 以及任何支持 `<script>` 标签的网站上运行。与 Shopify 相同的控制台和菜单类型。
- **控制台多语言** — Navi+ 控制台界面现已支持多语言。

---

### 2025年11月

- **修复闪烁问题** — 修复了部分设备上页面顶部和底部 auto-hide/auto-show 闪烁的 bug。

---

### 2025年10月

- **Cloudflare CDN** — 全部资源迁移至 Cloudflare 全球边缘网络。菜单平均加载时间降至约 52ms。
- **Elite 故障转移** — Elite 套餐新增 BunnyCDN 备用 CDN。SLA 升级至 99.99%。
