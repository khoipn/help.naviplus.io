# 菜单项

**菜单项**是菜单列表中的一行。菜单项是每个 Navi+ 菜单的构建块——每个菜单项可以是导航链接、分组标题、空白间隔或标签分隔符。

## 快速参考

**在哪里编辑菜单项？**  
点击菜单树中的任意菜单项 → 右侧打开编辑菜单项面板。按**更新**应用更改，然后按**保存**（`Ctrl+S` / `Cmd+S`）发布。

**编辑菜单项的字段顺序：**  
媒体（图标 / 图片）→ 基本信息（文本、链接、描述、徽标、分隔线）→ 布局 → 界面（颜色、间距）→ 高级（动画、CSS）→ 发布（显示设置）
