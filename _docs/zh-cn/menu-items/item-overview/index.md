# 项目概述

## 菜单项是什么？

菜单列表中的每一行都是一个 **项目**。项目是 Navi+ 中的最小单位——它可以是导航链接、组标题、空白间隔或标签分隔符。

多个项目堆叠在一起形成一个菜单。然后该菜单在您的网站上呈现给访客。

---

## 项目种类

当您添加新项目时，您选择其 **种类**。种类决定了项目的外观和功能。

| 种类 | 描述 | 使用时机 |
|---|---|---|
| **图标 / 文本** | 带有图标和标签的标准项目 | 常规导航链接 |
| **大图像 / 文本** | 带有全幅图像和覆盖文本的大块 | 类别横幅、特色产品图像 |
| **组标题** | 部分标题——不是链接 | 视觉上将下面的项目分组 |
| **空白空间** | 透明间隙 | 间距和布局对齐 |
| **标签标记** | 标记新标签的开始 | 将菜单分成水平标签 |
| **自定义 HTML** | 嵌入任意 HTML | 自定义小部件、特殊横幅 |

## 字段结构

单击任何项目以打开编辑项目面板。它被组织成以下几个部分：

```
┌─────────────────────────────────────────┐
│  媒体                                  │
│    图标  |  图像                       │
├─────────────────────────────────────────┤
│  基本信息                             │
│    文本 · 链接到 · HTML                │
│    描述 · 分隔符 · 徽章              │
├─────────────────────────────────────────┤
│  布局                                 │
│    显示布局 · 宽度 · 位置            │
│    高度固定                           │
├─────────────────────────────────────────┤
│  界面 (UI)                             │
│    类名 · 文本样式                   │
│    图标/图像样式 · 背景              │
│    间距 · 对齐                        │
├─────────────────────────────────────────┤
│  高级                                  │
│    动画 · CSS · 属性                  │
├─────────────────────────────────────────┤
│  发布                                  │
│    是否已发布 · 登录条件            │
└─────────────────────────────────────────┘
```

---

## 编辑工作流程

1. 在菜单树中单击一个项目 → 编辑项目面板打开。
2. 更改您需要的字段。
3. 按 **更新** 以应用对该项目的更改。
4. 按 **保存** (`Ctrl+S` / `Cmd+S`) 以保存整个菜单并刷新模拟器。

> 如果您按了更新但尚未保存，则更改不会写入您的实时菜单。按住 Shift 的同时按保存以保存并立即重新加载模拟器。
