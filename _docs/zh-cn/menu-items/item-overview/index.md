# 菜单项概述

## 什么是菜单项？
菜单列表中的每一行都是一个菜单项。菜单项是 Navi+ 中最小的单元——它可以是导航链接、分组标题、空白间隔或标签分隔符。

## 菜单项类型
| 类型 | 描述 | 使用场景 |
|---|---|---|
| **图标 / 文本** | 带有图标和标签的标准菜单项 | 普通导航链接 |
| **大图片 / 文本** | 带全宽图片的大磁贴 | 分类横幅、推荐商品 |
| **分组标题** | 区块标题——不是链接 | 在视觉上对下方菜单项分组 |
| **空白间隔** | 透明间距 | 间距和布局对齐 |
| **标签标记** | 标记新标签的开始 | 将菜单分成横向标签 |
| **自定义HTML** | 嵌入任意 HTML | 自定义小部件、特殊横幅 |

## 字段结构
点击任意菜单项打开编辑菜单项面板：

```
媒体：图标 | 图片
基本信息：文本・链接目标・HTML・描述・分隔线・徽标
布局：显示布局・宽度・位置・高度固定
界面：类名・文本样式・图标/图片样式・背景・间距・对齐
高级：动画・CSS・属性
发布：是否发布・登录条件
```

## 编辑工作流程
1. 点击菜单项 → 编辑菜单项面板打开。
2. 修改所需字段。
3. 按**更新**应用更改。
4. 按**保存**（`Ctrl+S` / `Cmd+S`）发布。
