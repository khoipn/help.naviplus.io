# 创建和管理项目

## 添加新项目

1. 在菜单树中，点击 **+ 添加项目**（或列表底部的加号按钮）。
2. 选择您想要的项目类型（图标/文本、大图像、组标题、空白空间、标签标记…）。
3. 新项目将以默认内容出现在列表的末尾。
4. 点击该项目以打开编辑项目面板并填写详细信息。
5. 按 **更新** → **保存**。

> 新项目总是添加在底部。添加后将它们拖动到您想要的位置。

---

## 重新排序项目

### 拖放

- 将鼠标悬停在项目左侧的拖动手柄（≡）上。
- 向 **上/下** 拖动以在同一层级内更改顺序。
- 向 **右** 拖动以缩进——该项目成为上面项目的子项。
- 向 **左** 拖动以减少缩进——该项目向上移动一个层级。

### 通过键盘快捷键（当选中项目时）

| 键 | 操作 |
|---|---|
| `↑` | 向上移动（Mac: `Cmd+↑` / Win: `Alt+↑`） |
| `↓` | 向下移动（Mac: `Cmd+↓` / Win: `Alt+↓`） |
| `→` | 缩进——项目成为上面项目的子项 |
| `←` | 减少缩进——项目返回到父级层级 |

> 快捷键仅在光标悬停在菜单树上并且项目被选中（蓝色轮廓）时有效。

---

## 编辑项目

1. 点击项目 → 编辑项目面板打开。
2. 更改您需要的字段。
3. 按 **更新** 以保存对该项目的更改。

> 按 **取消** 以在不保存的情况下关闭面板。

---

## 复制项目

1. 将鼠标悬停在项目上 → 快速操作菜单出现。
2. 点击 **复制**（或按 `C`）。
3. 复制项将立即出现在原始项下方，所有内容和设置均被保留。
4. 根据需要编辑复制项。

---

## 删除项目

1. 将鼠标悬停在项目上 → 快速操作菜单出现。
2. 点击 **删除**（或在Mac上按 `Del` / `Backspace`）。
3. 出现确认对话框——点击 **确认** 以永久删除。

> **注意：** 删除父项也会删除所有子项。确认后无法撤消。

---

## 撤消 / 重做

| 快捷键 | 操作 |
|---|---|
| `Ctrl+Z` / `Cmd+Z` | 撤消最后一个操作 |
| `Ctrl+Y` / `Cmd+Shift+Z` | 重做 |

撤消历史仅存在于当前会话中——重新加载页面会清除它。

---

## 保存菜单

在添加、重新排序或删除项目后：

- 点击工具栏中的 **保存**，或按 `Ctrl+S` / `Cmd+S`。
- 按住 `Shift` 的同时按保存以保存并立即重新加载模拟器。

在您保存之前，客户无法看到更改。
