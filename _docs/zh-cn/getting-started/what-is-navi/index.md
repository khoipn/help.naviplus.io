# 什么是 Navi+？（工作原理）

Navi+ 是一款**无代码导航构建工具** — 无需修改主题或编写任何代码，即可为任何网站创建精美的高转化菜单。

如果网站访客难以找到所需内容，可能是您的菜单拖了后腿。Navi+ 帮您在几分钟内解决这个问题。

## 五种菜单类型

根据您的使用场景选择合适的菜单：

| 菜单类型 | 最适合 |
| --- | --- |
| **Tab Bar** | 移动优先导航，感觉像原生应用 |
| **Mega Menu** | 带图片、列和链接的丰富下拉菜单 — 桌面端和移动端分别有独立版本 |
| **Slide Menu** | 支持多层级的汉堡菜单 |
| **Grid Menu** | 视觉化分类浏览 — 非常适合电商 |
| **FAB** | 快速访问浮动按钮，随时触手可及 |

您可以在同一个网站上同时使用多种菜单类型。

## 工作原理

Navi+ 是一个**小部件** — 它不会替换或修改您现有的主题。它向您的页面注入一个轻量级脚本，在您现有布局上方渲染菜单。

您从 **Navi+ 控制台**构建和管理一切。更改立即生效 — 无需编辑代码、修改模板或重新发布主题。

流程如下：

1. 在控制台创建菜单
2. Navi+ 生成一个小脚本
3. 该脚本嵌入您的网站（通过 Shopify 的 App Block，或其他平台的代码片段）
4. 每次页面加载时，菜单在您现有布局上方渲染

## 兼容平台

**Shopify** · **WordPress** · **WooCommerce** · **Wix** · **Webflow** · **Squarespace** · 以及任何接受 `<script>` 标签的网站。

## Navi+ 不是什么

- **不是主题** — 它与您现有主题并行运行，而非替代
- **不是需要手动安装的代码** — 一个 App Block（Shopify）或一个嵌入代码片段就够了
- **不是可能破坏您网站的插件** — 如果 Navi+ 被禁用，您原来的主题菜单会自动恢复

## 专为速度设计

Navi+ 资源通过 Cloudflare CDN 分发，平均加载时间约 52ms — 通常比您平台本身的脚本更快。
