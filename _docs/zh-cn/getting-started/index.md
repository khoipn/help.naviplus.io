# 入门指南

欢迎来到 **Navi+** — 我们很高兴您在这里！🎉

如果您网站的访客在寻找他们想要的内容时遇到困难，您的菜单可能让他们失望。Navi+ 帮助您解决这个问题 — 无需编写一行代码。

## 什么是 Navi+？

Navi+ 是一个 **无代码导航构建器**，使您可以轻松为任何网站创建美观、高转化率的菜单。无论您经营的是在线商店还是商业网站，您都可以在 **几分钟内** 设置专业的导航。

从五种强大的菜单类型中选择，以满足您的需求：

| 菜单类型           | 最适合                                         |
| ------------------- | ------------------------------------------------ |
| **Tab Bar**         | 移动优先导航，感觉像本地应用                   |
| **Mega Menu**       | 带有图像、列和链接的丰富下拉菜单               |
| **Slide Menu**      | 平滑的汉堡菜单，支持多级                       |
| **Grid Menu**       | 视觉类别浏览 — 非常适合电子商务               |
| **Floating Button** | 快速访问的操作始终触手可及                     |
