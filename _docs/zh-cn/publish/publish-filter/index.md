# 发布过滤器 — 设备和 URL 规则

在启用发布切换后，您可以使用两个过滤层来缩小菜单出现的时间：**设备**（移动/桌面 + 滚动行为）和 **URL 关键字**（根据 URL 内容显示/隐藏）。

---

## 按设备过滤

### 按平台启用/禁用

发布模态顶部的两个切换：

| 设置 | 默认 | 备注 |
|---|---|---|
| **在移动设备上显示菜单** | 开 | 在移动设备上显示菜单 |
| **在桌面上显示菜单** | 关 | 在桌面设备上显示菜单 |

某些菜单类型禁用其中一个切换 — 例如，桌面 Mega 菜单禁用移动切换。

---

### 滚动行为（仅限粘性菜单）

为移动和桌面单独配置。

#### 移动滚动

| 设置 | 行为 |
|---|---|
| **向下滚动时自动隐藏** | 当用户向下滚动时隐藏菜单 **向下** |
| **向下滚动后自动显示** | 仅在用户向下滚动后显示菜单（在页面加载时不显示） |

#### 桌面滚动

| 设置 | 行为 |
|---|---|
| **向下滚动时自动隐藏** | 在桌面上向下滚动时隐藏菜单 |
| **向下滚动后自动显示** | 仅在桌面上向下滚动后显示 |

> 自动隐藏和自动显示可以一起启用：菜单在向下滚动时隐藏，在向上滚动时重新出现 — 这是底部 Tabbar 的常见模式。

---

### 屏幕位置（仅限粘性 / Tabbar）

从带有视觉插图的单选按钮选择一个位置。

**移动** — 6 个预设位置（底部中心、底部左侧、底部右侧、顶部变体...）

**桌面** — 12 个预设位置（顶部栏、底部栏、左/右侧边栏、角落位置...）

---

### 页面组（仅限 Shopify，粘性菜单）

选择菜单出现的 Shopify 页面类型。默认：所有。

| 组 | URL 模式 |
|---|---|
| 首页 | `/` |
| 产品详情 | `/products/*` |
| 产品类别 | `/collections/*` |
| 页面 | `/pages/*` |
| 博客 | `/blogs/*` |
| 其他 | 所有剩余页面 |

> 全球市场（WordPress、Wix 等）没有此功能。请改用 URL 关键字过滤。

---

## 按 URL 关键字过滤

**"按关键字显示菜单"** 卡片 — 适用于除上下文滑动外的所有菜单类型。

### 两种过滤类型

| 设置 | 行为 |
|---|---|
| **包含关键字** | 仅当 URL **包含** 至少一个关键字时菜单 **显示** |
| **不包含关键字** | 当 URL **包含** 任何关键字时菜单 **不显示** |

多个关键字用逗号分隔 `,`。

### 示例

**仅在滑雪板产品页面上显示：**
```
包含：滑雪板
```
→ 菜单仅在类似 `yourdomain.com/products/the-collection-snowboard-hydrogen` 的 URL 上显示

**在多地点页面上隐藏：**
```
不包含：多地点
```
→ 菜单在 `yourdomain.com/products/the-multi-location-snowboard` 上 **不** 显示

**组合：**
```
包含：集合
不包含：促销，清仓
```
→ 仅在集合页面上显示，但在 "促销" 和 "清仓" 集合上隐藏。

### 备注

- 过滤使用 **完整 URL 字符串匹配**，而不是正则表达式。
- 关键字是 **不区分大小写** 的。
- 留空 = 无过滤（菜单在所有页面上显示）。
- 两个过滤条件是 AND 关系 — 必须满足两个条件。
