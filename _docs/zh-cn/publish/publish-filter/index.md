# Publish Filter — 设备和 URL 规则

启用发布开关后，您可以使用两个过滤层来精确控制菜单的显示条件：**设备**（移动端/桌面端 + 滚动行为）和 **URL 关键词**（根据 URL 内容显示/隐藏）。

---

## 按设备过滤

### 按平台启用/禁用

Publish 弹窗顶部的两个开关：

| 设置 | 默认 | 说明 |
|---|---|---|
| **在移动端显示菜单** | 开启 | 在移动设备上显示菜单 |
| **在桌面端显示菜单** | 关闭 | 在桌面设备上显示菜单 |

某些菜单类型会禁用其中一个开关 — 例如，Desktop Mega Menu 会禁用移动端开关。

---

### 滚动行为（仅限 Sticky 菜单）

移动端和桌面端分别配置。

#### 移动端滚动

| 设置 | 行为 |
|---|---|
| **向下滚动时自动隐藏** | 用户**向下**滚动时隐藏菜单 |
| **滚动后自动显示** | 仅在用户向下滚动后显示菜单（页面加载时不显示） |

#### 桌面端滚动

| 设置 | 行为 |
|---|---|
| **向下滚动时自动隐藏** | 在桌面端向下滚动时隐藏菜单 |
| **滚动后自动显示** | 在桌面端向下滚动后才显示 |

> 自动隐藏和自动显示可以同时启用：向下滚动时菜单隐藏，向上滚动时重新出现 — 底部标签栏的常见模式。

---

### 屏幕位置（仅限 Sticky / Tabbar）

从带有视觉图示的单选按钮选择器中选择位置。

**移动端** — 6 个预设位置（中央底部、左下、右下、顶部变体...）

**桌面端** — 12 个预设位置（顶部栏、底部栏、左/右侧边栏、角落位置...）

---

### 页面组（仅限 Shopify，Sticky 菜单）

选择菜单在哪些 Shopify 页面类型上显示。默认：全部。

| 组别 | URL 模式 |
|---|---|
| 首页 | `/` |
| 商品详情 | `/products/*` |
| 商品分类 | `/collections/*` |
| 页面 | `/pages/*` |
| 博客 | `/blogs/*` |
| 其他 | 所有其余页面 |

> 全局市场（WordPress、Wix 等）没有此功能。改用 URL 关键词过滤器。

---

## 按 URL 关键词过滤

**"按关键词显示菜单"**卡片 — 适用于除 Context Slide 之外的所有菜单类型。

### 两种过滤类型

| 设置 | 行为 |
|---|---|
| **包含关键词** | 仅当 URL **包含**至少一个关键词时**显示**菜单 |
| **不包含关键词** | 当 URL **包含**任何关键词时**不显示**菜单 |

多个关键词用逗号 `,` 分隔。

### 示例

**仅在滑雪板商品页面显示：**
```
包含: snowboard
```
→ 菜单仅在 `yourdomain.com/products/the-collection-snowboard-hydrogen` 等 URL 上显示

**在多地点页面隐藏：**
```
不包含: multi-location
```
→ 菜单**不在** `yourdomain.com/products/the-multi-location-snowboard` 上显示

**组合使用：**
```
包含: collections
不包含: sale, outlet
```
→ 仅在分类页面显示，但在"sale"和"outlet"分类中隐藏。

### 说明

- 过滤器使用**完整 URL 字符串匹配**，而非正则表达式。
- 关键词**不区分大小写**。
- 留空 = 无过滤（菜单在所有页面显示）。
- 两个过滤器为 AND 关系 — 两个条件都必须满足。
