# 发布（Publish）

**Publish** 部分控制您的菜单在实时网站上的显示方式：切换菜单开/关、选择嵌入方式、配置设备可见性以及按 URL 过滤。这是设计之后的最后一步 — 如果不保存 Publish 设置，菜单将不会显示。

## 快速参考

```
菜单类型              主要方法
────────────────────────────────────────────────────
Sticky / TABBAR     App Embeds (Shopify) / <head> (全局) + 启用开关
Sticky / FAB        App Embeds (Shopify) / <head> (全局) + 启用开关
Slide (CONTEXT)     App Embeds / <head> + 启用开关 + 配置触发器
Section (Mega/Grid) Insert/Replace (CSS Selector) 或 App Block (Shopify)
```

**规则：** 开关已启用 = 菜单处于上线状态。开关已禁用 = 从网站完全隐藏，无需删除嵌入代码。
