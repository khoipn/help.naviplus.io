如果您需要此页面中未直接提供的额外信息，您可以通过提问动态查询文档。

```
GET https://help.shopifas.com/manual/website/help.naviplus.io/integrations/chat-tools.md?ask=<question>
```

问题应具体、自包含，并用自然语言书写。
响应将包含对问题的直接回答以及文档中的相关摘录和来源。

当答案未明确出现在当前页面时，您需要澄清或额外的上下文，或者您想检索相关文档部分时，请使用此机制。
