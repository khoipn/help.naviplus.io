## 快速参考

- 使用**此菜单的自定义CSS**为每个菜单设置特定样式
- 使用**Reusable Classes (CSS)**设置菜单间共享样式
- CSS针对三个级别：主栏 → 子菜单 → 嵌套子菜单
- JavaScript库（如TourGuideJS）可通过菜单项属性嵌入
