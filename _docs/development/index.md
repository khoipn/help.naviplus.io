## Quick Reference

- Use **Custom CSS for this menu** for per-menu styles
- Use **Reusable Classes (CSS)** for shared styles across menus
- CSS targets three levels: main bar → submenu → nested submenu
- JavaScript libraries (e.g. TourGuideJS) can be embedded via menu item attributes
