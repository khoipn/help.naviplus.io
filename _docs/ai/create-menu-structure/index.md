# Create menu structure

Describe the menu you need and let AI build a ready-made skeleton — get a starting point fast instead of adding each item by hand. This feature is **free**.

Open: the **AI** button → **Create menu structure** card.

---

## Before you start — back up your menu

> **⋮ More → Backup / Restore** — back up your current menu before creating a new structure, since the result overwrites the existing menu.

---

## How to use

1. Open the **Create menu structure** card.
2. (Optional) Choose which levels AI should **auto-add icons** to:

   | Option | Applies to |
   |---|---|
   | Level 1 | Main items (level 1) |
   | Level 2 | Level-2 sub-items |
   | Level 3 | Level-3 sub-items |

3. Type your description in the **Describe your request** box, e.g. "Menu for a cosmetics store: Home, Products (Lipstick, Moisturizer, Mask), Promotions, Contact."
4. (Optional) Click **Use template** to insert a sample prompt for your menu type, then edit it.
5. Click **Create menu**. Progress shows in the **Activity log** on the right.
6. The AI builds the structure and applies it to the editor. Review, refine, then click **Save**.

> Click **Save** (the disk icon) to store your prompt for reuse next time.

---

## After creating

The AI only builds the **structure skeleton** (items, hierarchy, icons). You should review and add:

- Links (URLs) for each item
- Images and badges if needed
- Colors and layout via the **Design** and **Setting** panels

---

## Example descriptions

- Men's fashion store: Home, Tops (T-shirts, Shirts, Jackets), Pants, Accessories, Sale, Contact.
- 5-item tab bar for a food delivery app: Home, Search, Cart, Orders, Account.
- Slide menu for a blog: Latest posts, Categories (Tech, Travel, Food), About, Contact.
- Mega menu for an electronics store with 4 groups: Phones, Laptops, Accessories, Promotions — 4–5 sub-items each.

## Case study — build a mega menu for an electronics store

**Context:** An electronics store needs a multi-group desktop mega menu, each group with several sub-items, with icons for easy scanning.

1. **Back up:** ⋮ More → Backup / Restore.
2. Open the **Create menu structure** card. Tick **Level 1** and **Level 2** so AI adds icons to both groups and sub-items.
3. Type the description:

   ```
   Mega menu for an electronics store, 4 groups:
   - Phones (iPhone, Samsung, Xiaomi, Phone accessories)
   - Laptops (Office, Gaming, MacBook, Laptop accessories)
   - Home appliances (Water purifiers, Robot vacuums, Air fryers)
   - Promotions
   ```

4. Click **Create menu**. Watch the **Activity log** on the right. The AI builds a 4-group mega menu with sub-items and icons.
5. Add after creating: assign a **URL** to each item, add a banner image to the Promotions group, then **Save**.

**Result:** A multi-level mega menu skeleton with icons right away — you only need to attach links and fine-tune, instead of creating dozens of items by hand.
